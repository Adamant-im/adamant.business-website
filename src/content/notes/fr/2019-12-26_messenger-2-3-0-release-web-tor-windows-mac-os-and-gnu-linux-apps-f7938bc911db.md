ADAMANT Messenger v2.3.0 est désormais disponible avec des versions pour le Web, Tor, Windows, macOS et GNU/Linux. Cette version introduit le support du Bitcoin ainsi que plusieurs améliorations dans la gestion des statuts de transaction et des corrections de bogues.

Le traitement des transactions a été amélioré sur plusieurs chaînes. Le nombre de confirmations affiché sur les écrans de détail des transactions se met désormais à jour dynamiquement, et un nouveau statut Enregistré pour les transactions permet une gestion plus efficace des mises à jour de statut. Les statuts des transactions Ethereum, qui étaient auparavant incorrects, ont été corrigés. En outre, le traitement des liens vers les discussions sur msg.adamant.im a été corrigé.

La base de code actuelle se trouve dans la branche master du [dépôt GitHub ADAMANT Messenger](https://github.com/Adamant-im/adamant-im). Les développeurs souhaitant contribuer au projet open source ADAMANT peuvent y trouver la dernière version source.
