Cette version met à jour les dépendances du projet et réécrit le tableau de bord en utilisant Svelte. L'ensemble de la base de code a été repensée et optimisée, et plusieurs bogues ont été corrigés.

### Changements cassants

Le support des versions de Node.js antérieures à 18.12.1, la version LTS actuelle, a été abandonné. Le pool utilise désormais lowdb pour sa base de données. Reportez-vous à la section de migration dans le fichier README pour plus de détails.
