Cette version réécrit et met à jour la manière dont les données sont stockées sur l'appareil de l'utilisateur. Les utilisateurs souhaitant éviter la déconnexion automatique doivent désormais définir un mot de passe supplémentaire, utilisé pour chiffrer les données locales. Toutes les données stockées sur l'appareil sont chiffrées à l'aide d'un chiffrement local, et Argon2 est utilisé pour le hachage des mots de passe. Par conséquent, le stockage de données sur les appareils personnels pour un accès plus rapide est désormais sécurisé.

Les données sécurisées sont également transférées de LocalStorage à IndexedDb, ce qui offre davantage d'espace pour les messages.
