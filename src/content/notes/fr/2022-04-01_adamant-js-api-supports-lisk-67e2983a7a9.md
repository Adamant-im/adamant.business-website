La bibliothèque ADAMANT JavaScript API prend désormais en charge les clés cryptographiques Lisk, accompagnée d'un remaniement complet du codebase. La version v1.5.0 remplace l'ancien client HTTP par Axios, améliorant ainsi la gestion des requêtes et la maintenabilité de l'ensemble de la bibliothèque. Les directives de contribution et le style de code ont également été mis à jour pour faciliter la collaboration.

La documentation complète de l'API est disponible sur le [wiki ADAMANT API](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification), et les détails de la version sont disponibles sur la [page de publication v1.5.0](https://github.com/Adamant-im/adamant-api-jsclient/releases/tag/v1.5.0).
