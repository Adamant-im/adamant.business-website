Cette version se concentre sur l'amélioration des tests et la maintenance des dépendances pour la base de code ADAMANT. La suite de tests du nœud a été corrigée, et de nouveaux tests API ont été ajoutés afin d'élargir la couverture. Un nouvel endpoint `POST /api/transactions` est introduit dans le cadre de cette version.

La dépendance sodium a été mise à jour pour résoudre des problèmes de compatibilité rencontrés avec les versions récentes de Node.js, garantissant un fonctionnement plus fluide dans les environnements d'exécution actuels.
