Cette version introduit plusieurs nouvelles fonctionnalités API et des améliorations d'infrastructure pour le logiciel de nœud ADAMANT.

De nouveaux points d'accès `/api/node` ont été ajoutés, élargissant les informations disponibles pour les clients interagissant avec le réseau. Un nouveau point d'accès `/api/chatrooms` est également inclus. En outre, un serveur WebSocket a été introduit, permettant aux clients légers de s'abonner et de recevoir des notifications pour les nouvelles transactions.

Les champs `version` et `minVersion` ont été retirés du fichier de configuration.

### Changements cassants

Les champs `version` et `minVersion` ne sont plus lus à partir du fichier de configuration. Les déploiements qui définissent ces valeurs via le fichier de configuration devront mettre à jour leur configuration en conséquence.
