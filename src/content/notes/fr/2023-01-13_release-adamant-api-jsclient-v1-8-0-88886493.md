Cette version du client JS API ADAMANT introduit un nouveau paramètre `checkNodeTimeout`, permettant aux développeurs de configurer la durée d'expiration utilisée lors de la vérification de la disponibilité des nœuds. Cette ajout offre un meilleur contrôle sur la manière dont le client vérifie la connectivité aux nœuds ADAMANT, aidant ainsi les applications à gérer de manière plus fluide les nœuds lents ou non réactifs.
