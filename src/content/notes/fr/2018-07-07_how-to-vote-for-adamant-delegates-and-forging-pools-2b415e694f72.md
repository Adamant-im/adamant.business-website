ADAMANT utilise la preuve d'enjeu déléguée (dPoS) comme protocole de consensus de sa blockchain. Les délégués classés parmi les 101 premières places actives forgent des blocs et perçoivent les frais de transaction. Un pool de forging est lui-même un délégué, donc le processus de vote est identique que vous souteniez un délégué individuel ou un pool. Vous pouvez voter pour n'importe quel délégué, y compris pour vous-même.

### Prérequis

Tout compte détenant au moins 50 ADM peut voter. Les frais de vote s'élèvent à 50 ADM, quel que soit le nombre de délégués pour lesquels vous votez — jusqu'à 30 à la fois — et sont déduits de votre compte et forjés par les délégués actuels. Vous pouvez voter pour soutenir (upvote) ou retirer un vote précédent (downvote).

### Voter via le portefeuille web

Connectez-vous sur [msg.adamant.im](https://msg.adamant.im) et accédez à **Paramètres → Voter pour les délégués**. Cette page affiche la liste de tous les délégués ADAMANT enregistrés, avec leur classement et vos votes actuels.

![Comment voter pour les délégués ADAMANT et les pools de forging](/images/engineering-notes/medium/2b415e694f72/002-1-zrnvu6rsipsvzlhrc0w5aa-png.webp)

La colonne **Nom** indique le nom du délégué enregistré et **Classement** sa position ; seuls les 101 premiers délégués forgent des blocs. En cliquant sur le nom d'un délégué, des informations supplémentaires s'affichent pour vous aider à décider de le soutenir.

Utilisez les icônes pouce vers le haut et pouce vers le bas pour voter ou retirer votre vote, puis cliquez sur **Revoir le vote** pour confirmer vos sélections et sur **Voter** pour appliquer les modifications.

![Comment voter pour les délégués ADAMANT et les pools de forging](/images/engineering-notes/medium/2b415e694f72/003-1-topby8sxxwodlqzftbiwjg-png.webp)

### Voter via la console

Le vote peut également être effectué via la [Console ADAMANT](https://github.com/Adamant-im/adamant-console/wiki/Available-Commands#vote). Cette option est destinée aux utilisateurs avancés et n'est pas recommandée pour les utilisateurs réguliers de portefeuille.

Pour plus d'informations, consultez la documentation ADAMANT sur le [forging](https://medium.com/adamant-im/earning-money-on-adm-forging-4c7b6eb15516) et sur la procédure pour [devenir délégué](https://medium.com/adamant-im/how-to-become-an-adamant-delegate-745f01d032f).
