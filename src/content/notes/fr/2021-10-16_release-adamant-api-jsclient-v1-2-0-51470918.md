Cette version du client JS API ADAMANT introduit la prise en charge de la génération de clés BTC. La mise à jour étend les capacités cryptographiques du client pour inclure la création de paires de clés Bitcoin, en complément des fonctionnalités de clés ADAMANT existantes.

Aucune modification incompatible n'est introduite dans cette version.
