Cette version d'ADAMANT pour iOS introduit les réactions aux messages, qui fonctionnent également avec les transferts, ainsi qu'un menu contextuel amélioré pour les messages. Les utilisateurs peuvent désormais afficher directement dans la discussion l'avatar d'un correspondant, et l'application fournit des informations détaillées pour les transactions infructueuses.

Plusieurs bogues liés au design ont été corrigés, et l'amplitude du balayage a été réduite pour une interaction plus fluide. Les performances générales de l'application sont désormais stables.
