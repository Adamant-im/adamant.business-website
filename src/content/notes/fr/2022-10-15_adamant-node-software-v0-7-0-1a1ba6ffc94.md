ADAMANT Node v0.7.0 se concentre sur le nettoyage et la maintenance des dépendances. Cette version met à jour les dépendances du projet, déplace le code hérité dans un répertoire dédié `/legacy` et supprime la documentation obsolète. Ces modifications n'affectent pas le consensus du réseau, donc les propriétaires de nœuds n'ont pas besoin de faire une mise à jour.

Les détails de la version sont disponibles sur la [page de publication v0.7.0 sur GitHub](https://github.com/Adamant-im/adamant/releases/tag/v0.7.0).
