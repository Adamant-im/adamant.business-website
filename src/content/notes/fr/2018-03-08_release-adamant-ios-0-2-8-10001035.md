Cette version d'ADAMANT iOS introduit plusieurs améliorations de l'authentification utilisateur et de l'interface. Les utilisateurs peuvent désormais rester connectés entre les sessions, l'autorisation par code PIN offrant une couche de sécurité supplémentaire. L'authentification biométrique est prise en charge via Touch ID et Face ID.

Les barres de navigation ont été mises à jour pour utiliser des titres volumineux au style iOS 11, offrant une interface plus moderne et lisible. Des nettoyages généraux du code ont également été effectués pour améliorer la maintenabilité.

### Changements cassants

Aucun changement cassant n’a été introduit dans cette version.
