La bibliothèque API JavaScript ADAMANT v1.4.0 introduit deux nouvelles méthodes pour la gestion des délégués : `delegateNew()` pour enregistrer un nouveau délégué et `voteForDelegate()` pour voter en faveur de délégués existants. Cette version inclut également des mises à jour de dépendances courantes. Reportez-vous à la [spécification de l'API](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification) pour plus de détails sur l'utilisation.
