Cette version du client JS ADAMANT API introduit le support de la blockchain Lisk en complément des fonctionnalités existantes d'ADAMANT. La couche HTTP a été migrée vers un client basé sur Axios, améliorant ainsi la gestion des requêtes et la maintenabilité sur l'ensemble des réseaux supportés.

Ces modifications s'accompagnent d'une refonte plus large visant à nettoyer la structure interne et à réduire les duplications de code. Le projet intègre également des mises à jour de ses directives de style et de sa documentation pour les contributeurs, afin d'aider les nouveaux utilisateurs à démarrer plus facilement.
