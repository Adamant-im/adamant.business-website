Cette version introduit des applications de bureau autonomes pour macOS, Windows et Linux, offrant aux utilisateurs des versions natives installables sur tous les principaux systèmes d'exploitation. L'efficacité du réseau a été améliorée en réduisant le nombre de requêtes effectuées lors de l'utilisation de connexions socket. De plus, les dépendances du projet ont été mises à jour vers leurs dernières versions compatibles.
