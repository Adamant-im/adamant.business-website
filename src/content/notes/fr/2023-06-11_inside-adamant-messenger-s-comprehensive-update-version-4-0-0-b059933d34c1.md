ADAMANT Messenger version 4.0.0 introduit une mise à jour complète de la pile technologique visant à améliorer les performances globales et à réduire la consommation de mémoire. Cette version constitue principalement une mise à niveau fondamentale permettant de faciliter le développement futur de nouvelles fonctionnalités.

Les utilisateurs bénéficieront d'un échange de messages plus rapide et plus fluide grâce à une performance optimisée et à divers correctifs de bogues améliorant la stabilité. ADAMANT Messenger continue de fonctionner comme une application web progressive (PWA), assurant une compatibilité sur les environnements Web, Tor, Android, Windows, Linux et macOS.

Des informations techniques supplémentaires sont disponibles dans les notes officielles de publication sur le dépôt GitHub d'ADAMANT.
