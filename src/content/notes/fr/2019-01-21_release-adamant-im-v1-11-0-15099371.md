Cette version d'ADAMANT introduit la fonctionnalité de portefeuille pour le token DOGE et la possibilité d'effectuer des transferts via l'application web. Les utilisateurs peuvent désormais gérer et transférer des DOGE directement depuis la plateforme.

De plus, l'envoi de messages pour les devises de type ETH a été optimisé afin d'améliorer la rapidité. Un indicateur de chargement (spinner) a été ajouté pour fournir un retour visuel pendant le traitement des transactions, ainsi que plusieurs autres améliorations de l'expérience utilisateur.
