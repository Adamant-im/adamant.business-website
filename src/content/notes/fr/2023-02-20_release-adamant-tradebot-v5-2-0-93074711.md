Cette version met l'accent sur les améliorations d'entretien et de stabilité pour ADAMANT Tradebot. Les dépendances ont été mises à jour vers leurs dernières versions, et un travail de refactorisation interne a été effectué pour améliorer la qualité du code. Plusieurs corrections de bogues sont incluses dans cette version.

Le connecteur d'API P2PB2B a été réécrit pour une meilleure fiabilité et maintenabilité.
