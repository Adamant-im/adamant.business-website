## Modifications

Cette version réduit les frais d'envoi de messages de 0,005 à 0,001 AMD, réduisant ainsi le coût de la communication sur le réseau. L'application vérifie désormais les messages entrants plus fréquemment, améliorant la réactivité. Un bouton menant à la page de l'ICO a été ajouté, et les utilisateurs reçoivent désormais des messages ICO au lieu des anciens messages Pre-ICO. Les favicons ont été mis à jour, ainsi que quelques modifications internes mineures.

## Corrections

Un problème provoquant la disparition de messages a été résolu. Un correctif temporaire a été appliqué pour le Local Storage, qui devrait être entièrement supprimé dans les prochaines versions. Le problème de rendu des liens (#16), pouvant potentiellement être utilisé pour tromper les utilisateurs, a été corrigé.

## Traductions

De nouvelles traductions ont été ajoutées pour l'arabe, l'allemand, le portugais (contribué par @DataDrug via la PR #17), le roumain, le suédois (contribué par @pdahlberg via la PR #29) et le vietnamien.
