Cette version apporte plusieurs améliorations au client ADAMANT. L'écran de connexion inclut désormais une animation de chargement, et le carnet d'adresses est stocké dans le Key-Value Store (KVS) comme demandé dans l'issue #28. Les utilisateurs peuvent désormais voter pour des délégués et afficher une liste de délégués avec les informations de forging. Les constantes linguistiques reviennent automatiquement à l'anglais lorsque une chaîne localisée est manquante. Le lien ICO a été supprimé.

Le nombre de confirmations des transactions Ethereum est désormais précis, et le comportement de rejet des transactions ETH a été corrigé. L'affichage des frais de transaction ADM a été corrigé, et les fichiers linguistiques ont été mis à jour.
