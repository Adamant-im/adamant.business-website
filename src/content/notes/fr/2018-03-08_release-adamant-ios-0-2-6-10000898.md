Cette mise à jour corrige un problème de thread dans l'application ADAMANT iOS. Le lecteur de code QR était parfois appelé sur un thread d'arrière-plan, ce qui pouvait entraîner un comportement inattendu lors des opérations de numérisation. Grâce à cette correction, le lecteur de code QR est désormais systématiquement appelé sur le thread approprié, améliorant ainsi la stabilité lorsque les utilisateurs scannent des adresses de portefeuille ou d'autres codes QR dans l'application.
