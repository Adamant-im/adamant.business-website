### API plus rapide dans la dernière version Android

Suite aux annonces précédentes concernant les améliorations de vitesse de l'API pour le nœud ADAMANT, l'équipe de développement Android a publié une mise à jour incluant une nouvelle API de discussions.

Une comparaison entre l'ancienne et la nouvelle API montre que les opérations de discussion et la procédure de connexion s'effectuent désormais nettement plus rapidement. D'autres optimisations sont prévues.

Le dernier fichier APK est disponible sur la [page des publications GitHub pour Android](https://github.com/Adamant-im/adamant-android/releases/). Les utilisateurs existants doivent s'assurer qu'ils exécutent la dernière version ; si l'application ne se met pas à jour automatiquement, désinstallez l'ancienne version et installez manuellement le nouveau fichier APK.

Les commentaires et rapports de bogues peuvent être soumis via le dépôt GitHub. L'équipe prépare également une publication sur Google Play.
