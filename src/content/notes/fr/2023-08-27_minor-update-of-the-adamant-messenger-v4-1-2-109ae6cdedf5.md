ADAMANT Messenger v4.1.2 est une mise à jour mineure axée sur la correction de bogues et l'amélioration générale de la stabilité. Cette mise à jour n'ajoute pas de nouvelles fonctionnalités, mais corrige des problèmes affectant la fiabilité globale du messager sur toutes les plateformes prises en charge.

L'application est une Progressive Web Application disponible pour le Web, Tor, Android, Windows, Linux et macOS. Le code source est publié sur GitHub dans le dépôt du projet ADAMANT, permettant aux utilisateurs de compiler l'application eux-mêmes afin de vérifier indépendamment ses propriétés de confidentialité et de sécurité.

Les notes de version et les instructions de compilation sont disponibles dans l'[étiquette de publication v4.1.2](https://github.com/Adamant-im/adamant-im/releases/tag/v4.1.2) sur GitHub.
