Cette version du ADAMANT Tradebot introduit le support de l'échange Bit-Z, élargissant ainsi la gamme de plateformes disponibles pour le trading automatisé. Les utilisateurs peuvent désormais définir la précision de sortie pour les paires de trading grâce aux nouvelles options de configuration coin1Decimals et coin2Decimals. La base de données des ordres a été mise à jour pour inclure le champ exchange, améliorant la tenue des registres sur les plateformes prises en charge. Une nouvelle commande /params affiche les paramètres de trading actuels, facilitant la vérification de la configuration active. Cette version inclut également plusieurs corrections de bogues.
