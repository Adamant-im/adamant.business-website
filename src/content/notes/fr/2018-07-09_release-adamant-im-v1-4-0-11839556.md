Cette version ajoute un fichier LICENSE au dépôt. Le stockage de la liste de contacts suit désormais les recommandations de sécurité décrites dans AIP-3, avec compatibilité descendante pour les contacts stockés dans les versions précédentes.

Plusieurs correctifs sont inclus dans cette version. Un bogue affectant la synchronisation de plusieurs modifications de la liste de contacts entre différents appareils a été résolu. Les étapes inutilisées du cycle de vie des composants ont été supprimées, ainsi que d'autres composants inutilisés. Un problème provoquant un GIF de chargement infini lors d'un chargement incorrect a également été corrigé.
