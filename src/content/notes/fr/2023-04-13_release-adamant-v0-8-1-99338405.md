Cette version ajoute la journalisation pour le script de migration de la base de données. Le nœud ADAMANT enregistre désormais des messages avant et après l'exécution du script de migration :

```
[inf] 2023-12-22 04:00:25 | Found 5 pending migrations. Start executing, this may take a while…
[inf] 2023-12-22 04:17:13 | Migrations have been successfully completed.
```

La valeur par défaut de la propriété `cors.origin` a été modifiée : elle passe de l'origine de la requête reflétée (`true`) à `*`. Consultez la documentation pour en savoir plus sur la configuration CORS du nœud ADAMANT.

```json5
{
  "cors": {
    "origin": "*" // default
  }
}
```

Le fichier `config.json` a été ajouté à `.gitignore`. Les utilisateurs doivent désormais créer le fichier `config.json` manuellement ou copier `config.default.json` pour démarrer le nœud ADAMANT.

```
cp config.default.json config.json
```

### Changements cassants

La valeur par défaut de l'en-tête CORS `Access-Control-Allow-Origin` a changé : elle ne reflète plus l'origine de la requête, mais utilise désormais `*`. Les utilisateurs qui dépendent du comportement précédent (origine reflétée) doivent mettre à jour leur fichier `config.json` en conséquence. De plus, `config.json` est désormais ignoré par git, ce qui implique que les utilisateurs doivent créer ou copier ce fichier manuellement avant de démarrer le nœud.
