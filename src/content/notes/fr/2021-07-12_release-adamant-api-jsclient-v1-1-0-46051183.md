Cette version du client JavaScript ADAMANT API introduit la prise en charge de la génération de clés DASH. Les développeurs utilisant la bibliothèque peuvent désormais générer des clés DASH en complément des fonctionnalités existantes de génération de clés de cryptomonnaie.
