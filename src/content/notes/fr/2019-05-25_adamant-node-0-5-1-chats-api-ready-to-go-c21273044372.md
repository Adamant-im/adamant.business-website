ADAMANT Node version 0.5.1 est désormais disponible sur GitHub, achevant le travail sur l'API Chats initialement annoncé en janvier. La fonction principale est le point de terminaison `/api/chatrooms`, qui permet de récupérer les contacts et les discussions beaucoup plus rapidement que l'ancienne API. Les développeurs doivent se référer à la spécification [AIP-14](https://aips.adamant.im/AIPS/aip-14) pour les détails d'implémentation.

Cette mise à jour n'est pas obligatoire pour tous les propriétaires de nœuds. Cependant, les nœuds qui doivent se connecter à des applications de messagerie doivent passer à la dernière version pour profiter de la nouvelle API. L'équipe ADAMANT Android a déjà démontré des améliorations mesurables de vitesse en utilisant le nouveau point de terminaison, et des gains similaires sont attendus sur les autres applications ADAMANT à mesure qu'elles l'adopteront.
