Cette version introduit la possibilité d'ouvrir les détails des transactions directement dans l'explorateur de blockchain. Les nouveaux chats commencent désormais avec un écran vide, alignant le client Android sur la version iOS. De plus, des refactorisations mineures ont été effectuées afin d'améliorer la qualité du code.
