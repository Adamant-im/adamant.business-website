ADAMANT InfoServices a été mis à jour en v3.5.0, offrant une API combinée pour les cours des monnaies fiduciaires et des cryptomonnaies. Le service agrège des données provenant de plusieurs sources en amont, notamment MOEX, Currency-Api, ExchangeRates.host, CoinMarketCap, CryptoCompare et CoinGecko, puis croise les résultats pour détecter les écarts et améliorer la fiabilité.

Étant exécuté sur votre propre serveur, le service conserve les données historiques des cours sous votre contrôle et reste libre d'utilisation. Le projet est open source et hébergé sous l'organisation ADAMANT sur GitHub.

La documentation de l'API est disponible dans le [wiki InfoServices API](https://github.com/Adamant-im/adamant-currencyinfo-services/wiki/InfoServices-API-documentation).
