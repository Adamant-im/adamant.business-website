Cette version corrige plusieurs bogues affectant le mode interactif. Les sous-commandes inconnues n'arrêtent plus le mode interactif, les arguments de type chaîne sont désormais correctement séparés par des espaces, et les flèches haut et bas naviguent correctement dans l'historique des commandes.

De plus, cette version ajoute la prise en charge des commentaires JSON dans les fichiers de configuration, ce qui facilite l'annotation des configurations sans provoquer d'erreurs d'analyse.
