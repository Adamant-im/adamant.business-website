Cette version introduit plusieurs améliorations fonctionnelles dans l'application ADAMANT Messenger. L'application Android est désormais construite avec Capacitor.js, le niveau d'API passant de `19` à `22`. Un type de transaction non pris en charge a été ajouté au module de chat, et un nouveau hook `useIsMobile` a été introduit. Le token KLY a été ajouté au portefeuille, tandis que le token LSK a été retiré.

Plusieurs corrections de bogues sont incluses dans cette version. L'ordre de navigation des commandes dans le chat a été corrigé, et la couleur de la bordure du champ de saisie de message a été ajustée. Des corrections mineures de l'internationalisation ont été appliquées. Le comportement de défilement lors du retour en arrière a été corrigé. L'expression régulière pour les noms de nœuds a été corrigée. Les workflows GitHub ont été migrés vers Node v20, et un problème selon lequel `Object.groupBy` n'était pas une fonction dans les anciens navigateurs a été résolu.

La dette technique a également été traitée. La commande `wallets:generate` dans adamant-wallets a été améliorée, et les dépendances Electron ont été mises à jour.
