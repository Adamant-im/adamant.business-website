Cette version d'ADAMANT pour iOS corrige un plantage survenant lors de l'utilisation de Face ID sur iPhone X. Les nouveaux transferts sont désormais automatiquement marqués comme non lus afin d'aider les utilisateurs à suivre l'activité récente. De plus, la récupération en arrière-plan et les notifications push ont été corrigées afin de garantir une livraison fiable des messages. L'application prend désormais également en charge les notifications intégrées pour les messages entrants et les transferts.
