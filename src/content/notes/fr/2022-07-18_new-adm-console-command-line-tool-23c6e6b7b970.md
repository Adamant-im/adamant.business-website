La console ADAMANT, un outil en ligne de commande pour interagir avec la blockchain ADAMANT, a été entièrement repensée dans la version 2.0.0. Conçue pour compléter les API JS et Node existantes, la console permet aux utilisateurs de créer des comptes, de transférer des jetons, d'envoyer des messages et d'enregistrer des délégués.

L'outil fonctionne à la fois comme une interface en ligne de commande autonome et comme une bibliothèque NodeJS, ce qui le rend adapté à l'intégration dans des scripts ou applications externes. Les développeurs peuvent utiliser la console pour construire des applications décentralisées complètes, telles que des messageries ou des systèmes d'authentification à deux facteurs basés sur la blockchain, en tirant parti de ses nouvelles capacités de déchiffrement de messages.

La version 2.0.0 représente une refonte complète de la base de code, qui utilise désormais le package `adamant-api`. Elle introduit un mode interactif, une aide intégrée pour les commandes, des dépendances mises à jour, ainsi que divers correctifs de bogues.
