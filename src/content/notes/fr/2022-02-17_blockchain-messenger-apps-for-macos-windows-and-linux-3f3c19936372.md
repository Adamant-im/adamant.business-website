ADAMANT Messenger v3.1.0 introduit des applications autonomes pour macOS, Windows et Linux. Cette version réduit le nombre de requêtes réseau lors de l'utilisation de connexions socket et met à jour les dépendances du projet.

L'application peut être compilée à partir du code source via le [dépôt ADAMANT](https://github.com/Adamant-im/adamant-im). Les détails de la publication sont disponibles sur la [page de la version v3.1.0](https://github.com/Adamant-im/adamant-im/releases/tag/v3.1.0).
