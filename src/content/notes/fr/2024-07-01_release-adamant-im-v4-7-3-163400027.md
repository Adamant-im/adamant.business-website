Cette version corrige un bug dans le module de chat où les transferts envoyés dans le chat sans commentaire étaient incorrectement affichés comme « Transaction non prise en charge ». Le problème, référencé sous le numéro #642, a été résolu afin que ces transferts s'affichent désormais correctement dans l'interface de chat.
