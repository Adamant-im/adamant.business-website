Cette version d'ADAMANT Messenger introduit la prise en charge d'AIP-8, AIP-9 et AIP-15. Exchangebot et Betbot officiels ont été ajoutés au carnet d'adresses pour un accès plus facile.

Les adresses cryptos disposent désormais d'un bouton Partager permettant aux utilisateurs de copier rapidement une adresse ou de générer un code QR. Tous les champs de saisie d'adresse peuvent désormais être remplis à l'aide d'un lecteur de code QR, simplifiant ainsi le processus de transfert. Les champs de saisie sur les appareils de bureau affichent une indication concernant l'utilisation des émojis. En outre, les statuts des transferts sont revérifiés lors de la connexion via LoginByPassword.
