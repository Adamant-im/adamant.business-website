Cette version corrige le démarrage d'une nouvelle discussion via le lecteur QR et résout plusieurs problèmes de liste de discussions. Vous pouvez désormais appuyer sur les URL dans les discussions pour les ouvrir dans Safari. N'oubliez pas de ne pas suivre les liens que vous recevez de comptes inconnus ou non fiables, sinon votre adresse IP pourrait être compromise.

La liste des discussions affiche désormais le préfixe "Vous : " pour les messages envoyés. Les anciennes ressources ont été nettoyées, et le projet a été mis à jour vers Xcode 9.3 et Swift 4.1.
