Cette version d'ADAMANT iOS introduit le support de Dash, élargissant ainsi la gamme de cryptomonnaies disponibles dans l'application. Les nouvelles installations commenceront désormais avec une configuration propre, en supprimant tous les paramètres précédemment stockés, afin d'assurer une expérience de configuration initiale fraîche.

Plusieurs corrections de bogues ont été appliquées dans cette version pour résoudre les problèmes signalés par les utilisateurs et améliorer la stabilité générale de l'application.
