Cette version d'ADAMANT Messenger introduit la possibilité d'afficher les transferts directement dans les discussions, ce qui facilite le suivi de l'historique des transactions en parallèle du contexte de la conversation.

Plusieurs corrections ont été appliquées dans cette version. Les fichiers manifeste ont été mis à jour, et le service worker met désormais en cache les polices web afin d'améliorer la fiabilité hors ligne et les performances de chargement. Diverses mises à jour de style ont également été incluses pour affiner l'interface utilisateur.

Des mises à jour de traduction sont disponibles pour le coréen, l'espagnol et le français, améliorant ainsi la couverture de localisation pour ces communautés linguistiques.
