Cette version introduit des changements de style afin de rendre les nouveaux champs plus utilisables sur les appareils mobiles. La connexion à l'aide d'une phrase secrète stockée sous forme de code QR est désormais prise en charge, grâce à @chovy. Le flux de développement a été amélioré : `npm run dev` prend désormais en charge le rechargement à chaud.

Plusieurs corrections et améliorations sont incluses. Le stockage des données dans le KVS est désormais déclenché uniquement lorsqu'un nombre suffisant de jetons ADM est disponible, éliminant ainsi les fenêtres intrusives. Un bogue a été corrigé, et de nouvelles constantes linguistiques ont été ajoutées, accompagnées de mises à jour générales des langues.

Les langues ukrainien, philippin, suédois, hindi, turc, vietnamien, roumain et arabe ont été temporairement désactivées jusqu'à ce que quelqu'un mette à jour les fichiers linguistiques correspondants.
