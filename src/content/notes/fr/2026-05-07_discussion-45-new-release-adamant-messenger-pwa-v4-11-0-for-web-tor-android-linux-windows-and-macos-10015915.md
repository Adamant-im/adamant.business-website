ADAMANT a publié la version 4.11.0 de son Messenger PWA, désormais disponible sur les plateformes Web, Tor, Android, Linux, Windows et macOS. Cette version met l'accent sur l'amélioration de la fiabilité, de la finition et de la maturité, plutôt que sur l'ajout de fonctionnalités majeures.

La dernière version est taguée v4.11.1 sur GitHub, indiquant un correctif rapide publié peu après la sortie initiale de la version 4.11.0. Les utilisateurs exécutant le PWA doivent mettre à jour vers le dernier tag pour bénéficier des correctifs publiés après la sortie.

ADAMANT Messenger est une application de messagerie open-source basée sur la blockchain. La version PWA (Progressive Web App) permet aux utilisateurs d'exécuter le messager dans un navigateur ou de l'installer comme application native sur les plateformes desktop et mobiles prises en charge, y compris Tor pour les utilisateurs nécessitant un accès par routage onion.
