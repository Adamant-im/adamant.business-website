Cette version intègre l'affichage des transactions de transfert directement dans la vue de discussion, permettant aux utilisateurs de voir les transferts en ligne dans leurs conversations. Appuyer sur une bulle de transfert révèle des détails supplémentaires sur la transaction.

En arrière-plan, les modèles CoreData ont été restructurés afin que `ChatTransaction` serve désormais de modèle abstrait pour toutes les transactions dans la discussion. Le `TransfersProvider` a été mis à jour en conséquence pour fonctionner avec la couche de données révisée.

La localisation a été mise à jour pour les langues anglaise et russe.
