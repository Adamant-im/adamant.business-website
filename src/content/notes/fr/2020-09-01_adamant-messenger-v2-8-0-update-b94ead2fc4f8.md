ADAMANT Messenger v2.8.0 réduit les frais de transaction pour les transferts Ethereum et ERC-20, poursuivant ainsi les efforts du projet pour rendre son portefeuille cryptographique intégré plus économique au quotidien. La mise à jour améliore également la boîte de dialogue d'achat et de vente de jetons pour une expérience de trading plus claire.

Côté correction de bogues, cette version résout un problème avec le presse-papiers qui empêchait les utilisateurs iOS d'utiliser les boutons Copier dans le presse-papiers — une correction apparemment mineure mais importante pour l'ergonomie du portefeuille mobile. De plus, un lien vers l'exchange Atomars a été ajouté aux options intégrées pour l'échange de jetons.

Les notes complètes de la version sont disponibles dans la [publication ADAMANT Messenger v2.8.0](https://github.com/Adamant-im/adamant-im/releases/tag/v2.8.0) sur GitHub.
