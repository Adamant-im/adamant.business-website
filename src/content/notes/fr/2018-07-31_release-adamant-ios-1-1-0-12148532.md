Cette version d'ADAMANT iOS introduit la possibilité de renommer un partenaire directement depuis le carnet d'adresses, offrant aux utilisateurs un meilleur contrôle sur les libellés de leurs contacts. Le bouton « Vers Chat » est désormais masqué lorsque les détails d'une transaction sont affichés depuis un chat, réduisant ainsi l'encombrement inutile de l'interface.

Une nouvelle icône « Jetons gratuits » a été ajoutée à l'interface. Les formateurs de solde ont été corrigés pour garantir un affichage correct des valeurs dans l'ensemble de l'application. Le composant UIColor.adamant a été refactorisé afin d'améliorer la maintenabilité du code. Par ailleurs, un problème d'affichage de la ligne biométrique sur les anciens appareils dépourvus de Touch ID a été résolu.
