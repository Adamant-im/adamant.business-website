Cette version corrige les notifications push et modifie le comportement de déconnexion afin que les utilisateurs restent connectés lorsqu'un onglet est fermé, plutôt que d'être déconnectés automatiquement. Le mode sombre est désormais l'apparence par défaut de l'application.

Plusieurs améliorations ont été apportées à la gestion des liens : les liens vers la documentation sur les mots de passe utilisateur s'ouvrent désormais dans une nouvelle fenêtre avec `noopener`, et les liens contenus dans les messages de discussion s'ouvrent également dans de nouvelles fenêtres avec `noopener`, pour une sécurité accrue.

Les détails des transactions affichent désormais les commentaires et indiquent l'adresse propre à l'utilisateur avec le libellé « Moi ». Dans la liste des transactions, les commentaires sont également affichés, et les noms accompagnant les adresses ADM le sont aussi pour les transactions impliquant d'autres cryptomonnaies. En outre, les utilisateurs peuvent accéder directement à une discussion à partir d'une entrée dans la liste des transactions pour d'autres cryptomonnaies.
