Cette version introduit une fonctionnalité de vérification de santé de la liste des nœuds et réécrit les composants internes du client ADM API pour une meilleure fiabilité.

Plusieurs corrections sont incluses dans cette version. La barre inférieure ne disparaît plus pendant l’utilisation, et le bug des messages vides signalé dans l’issue #139 a été résolu. Les liens Markdown s’affichent désormais correctement, comme décrit dans l’issue #141. Les problèmes liés au vote ont été corrigés, et le script de build Travis a été réparé.
