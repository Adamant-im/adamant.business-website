Cette version introduit la prise en charge du Bit-Z Token (BZ), élargissant ainsi la gamme d'actifs pris en charge. Des connexions WebSocket ont été ajoutées afin d'améliorer la gestion des données en temps réel et la communication avec les API des exchanges. Les dépendances ont été mises à jour vers leurs dernières versions, et plusieurs correctifs ont été appliqués pour résoudre des problèmes identifiés dans les versions précédentes.
