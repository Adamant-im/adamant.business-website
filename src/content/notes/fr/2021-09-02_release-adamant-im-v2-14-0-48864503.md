Cette version introduit plusieurs améliorations au traitement des transactions et aux fonctionnalités de messagerie. Les utilisateurs peuvent désormais copier l'ID de transaction, l'expéditeur et les champs du destinataire directement depuis les détails de la transaction, et le bouton de mise à jour dans cette vue a été animé pour un meilleur retour visuel. Les paramètres d'envoi de Dash ont été ajustés avec des seuils corrigés pour les montants maximum et minimum, et un correctif résout les problèmes lors de l'envoi de petites quantités de Lisk. Les messages d'erreur affichés lors de l'envoi de transactions ont été reformattés pour une meilleure clarté.

La base de code a été mise à jour pour prendre en compte le hardfork London d'ETH. Le composant snackbar a été réglé pour un comportement amélioré, et le lien vers le site Tor a été remplacé par une nouvelle destination. Les dépendances sous-jacentes ont été mises à jour vers leurs dernières versions compatibles.
