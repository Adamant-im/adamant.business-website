Cette version introduit la génération de codes QR pour les phrases secrètes, permettant une connexion rapide en scannant un code imprimé que vous pouvez conserver dans un endroit physique sécurisé. Les utilisateurs peuvent également générer et partager des codes QR contenant leur adresse ADAMANT, et scanner directement le code d'un ami depuis la fenêtre Nouvelle discussion pour démarrer une conversation rapidement.

L'icône de l'application a été améliorée, et le style des notifications a été changé pour le mode clair. En outre, cette version inclut de nombreuses corrections internes et nettoyages.
