Cette version apporte une refonte complète de ADAMANT Console, désormais basée sur adamant-api. La console peut désormais déchiffrer les messages et inclut une aide intégrée pour les commandes disponibles. Un mode interactif a été introduit pour simplifier l'utilisation.

Le package peut désormais être inclus comme bibliothèque JavaScript, élargissant ainsi les possibilités d'intégration au-delà de l'interface en ligne de commande. Les dépendances ont été mises à jour et plusieurs bogues ont été corrigés.

### Changements cassants

La refonte complète implique que la structure interne et les API ont changé de manière significative. Les scripts ou intégrations existants qui reposent sur l'ancienne architecture devront être mis à jour pour fonctionner avec la nouvelle base basée sur adamant-api.
