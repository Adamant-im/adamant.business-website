L'équipe ADAMANT a publié l'application officielle ADAMANT Messenger pour iOS, désormais disponible au téléchargement sur l'Apple App Store. Pour installer l'application, recherchez "ADAMANT Messenger" dans l'App Store sur votre iPhone.

L'application iOS a fait l'objet d'une refonte complète, introduisant de nouvelles fonctionnalités auparavant indisponibles dans les versions précédentes. La version initiale est identifiée comme la version 1.0.0 sur le dépôt GitHub ADAMANT.
