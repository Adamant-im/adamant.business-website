Cette version migre le processus de construction du projet vers Webpack 4 et inclut un refactoring interne du codebase. Le bouton retour dans le chat revient désormais systématiquement à la liste des discussions au lieu d'appeler history.back. Des messages de bienvenue ont été ajoutés, et les utilisateurs peuvent désormais se connecter via un code QR.

Plusieurs corrections sont incluses dans cette version. La lecture par caméra a été corrigée, et les adresses ETH sont désormais stockées dans KVS lorsque le solde est positif. Les problèmes liés aux icônes ont également été résolus.
