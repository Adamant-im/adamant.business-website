Cette version s'articule autour d'une modernisation majeure de la base de code d'ADAMANT Messenger. L'application a été mise à jour vers Vue 3 et Vuetify 3, et Stylus a été supprimé au profit d'une réécriture complète en SASS. Le système de construction est passé de Webpack à Vite, et l'exécuteur de tests a été changé de Jest à Vitest. Le support de TypeScript a été ajouté, ainsi que la configuration de Prettier et d'EditorConfig. Le gestionnaire de paquets est passé de Yarn à NPM. Les versions majeures d'axios, socket.io-client, @mdi/font et d'autres dépendances ont été mises à jour, et tous les paquets ont été synchronisés avec leurs dernières versions mineures et correctives. Les tests échoués et les instantanés obsolètes ont été corrigés, les configurations Browserslist et ESLint ont été déplacées du fichier package.json vers des fichiers séparés, et plusieurs composants ont été réécrits pour utiliser l'API de composition.

Côté fonctionnalités, le bot Bounty Wallet a été supprimé des discussions (#482), bien que les utilisateurs ayant précédemment échangé avec le bot puissent toujours accéder à leur historique de discussion. Des informations supplémentaires ont été ajoutées au WebAppManifest pour les installations PWA (#481). Dans la fenêtre BuyTokensDialog, Coindeal a été remplacé par Azbit (#476). Des workflows GitHub Actions pour les builds Electron et PWA ont été ajoutés (#458, #461).

Plusieurs bogues ont été corrigés dans cette version. Une erreur survenant lors du chargement des délégués sur un nouveau compte dans le tableau DelegatesTable a été corrigée (#475). Un événement de défilement non déclenché sur certains appareils dans la vue Transactions a été corrigé (#474). La discussion conserve désormais la position de défilement lors d'une navigation arrière (#470), et la vue des discussions défile automatiquement vers le bas lorsqu'un nouveau message arrive (#455). De nombreuses corrections mineures de style ont également été appliquées (#454, #465, #466, #467, #468, #469, #471, #473, #479, #480).
