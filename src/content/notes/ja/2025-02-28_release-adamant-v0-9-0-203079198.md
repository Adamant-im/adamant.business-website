このリリースでは、ADAMANTノードソフトウェア向けのAPI改善と修正がいくつか導入されています。`/api/delegates/get`エンドポイントは、以前からサポートされていた`username`および`publicKey`パラメータに加え、`address`パラメータも受け付けるようになりました。使用法は簡単です：

```bash
/api/delegates/get?address=U1234567890
```

WebSocketのサポートが強化され、従来の1種類のトランザクションタイプに加えて、複数のトランザクションタイプを購読できるようになりました。メッセージトランザクションについては、新しい`assetChatTypes`イベントにより、クライアントが特定のメッセージタイプを購読できるようになっています。

`/api/node/status`エンドポイントは、`/api/loader/status`および`/api/loader/status/sync`のデータを統合することで追加のプロパティを返すようになり、完全なブロックチェーンステータス情報を取得するために必要なリクエスト数が削減されました：

```json5
{
  // ...
  "loader": {
    "loaded": true,
    "now": 45345521,
    "syncing": false,
    "consensus": 100,
    "blocks": 0,
    "blocksCount": 0
  }
}
```

いくつかのバグが修正されました。`+alpha`や`-dev.0`のようなプリリリースまたはビルドメタデータを含むバージョン名を持つピアとの接続問題が解決され、完全なSemVer 2.0準拠が実現されました。一時的な接続エラー後にピアが不必要に削除される問題も修正され、設定ファイルのバリデーションも正常に動作するようになりました。

`/delegates/get`エンドポイントは単一トランザクション検索向けに最適化されました。依存関係が更新され、ログ出力の軽微な改善が行われ、テストに関する新しいコントリビューターガイドが追加されました。
