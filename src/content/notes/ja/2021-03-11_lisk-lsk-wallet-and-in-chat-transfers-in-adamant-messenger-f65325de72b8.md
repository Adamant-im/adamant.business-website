ADAMANT Messenger v2.10.0 では、LSK トークンのサポートを開始しました。これにより、ユーザーは Web、Tor、Android、iOS 各プラットフォームのチャット内で LSK を保管および送金できるようになります。これにより、既にサポートされている ADM、Bitcoin、Ethereum、Doge、Dash、BNB、KSC、RES、USDS に LSK が追加されました。

ADAMANT ウォレットは完全にオープンソースであり、ユーザーはソースコードからビルドすることが可能で、クローズドソースの代替品に依存する必要はありません。秘密鍵はユーザーの端末から決して離れず、単一のパスフレーズで統合されたすべての暗号通貨ウォレットにアクセスできます。また、秘密鍵をエクスポートしてバックアップすることも可能です。ADAMANT の特徴的な機能として、メッセンジャーのチャット内から直接暗号通貨を送金できる点が挙げられます。

ADAMANT ブロックチェーン自体は Lisk のコードをベースに構築されており、特定のブロック生成時間、手数料、および分散型メッセージングのためのチャットルーム API が追加されています。今後の計画には、Lisk SDK の新バージョン、新しいアドレス形式、および ADAMANT エコシステム内での各種 Lisk サイドチェーンのサポートが含まれています。また、Lisk プラットフォーム上に Wrapped ADM トークンを作成し、サービスおよび分散型取引の選択肢を拡大することも検討されています。

ADM はユーティリティトークンとして機能し、その価値は匿名かつ安全な通信のインフラコストを賄うためのメッセージおよびデータ転送手数料によって支えられています。ADAMANT は、委任型プルーフオブステーク（dPoS）コンセンサスモデルを採用しており、これは「Fair dPoS」と呼ばれる改良が加えられています。このモデルでは、各ウォレットの投票権重をそのウォレットが投票した委任者数で割ることで、大口ウォレットの影響力を低減し、フォージングへの参入障壁を下げています。さらに、委任者がブロックを生成し損ねた場合、その生産性が低下し、結果として投票権重にも悪影響を与えます。

Lisk は、JavaScript/TypeScript SDK と Node.js コアを使用するブロックチェーンアプリケーションプラットフォームです。PoS/dPoS コンセンサス方式を採用しています。Ethereum と比較して、JavaScript の人気によりより広い開発者層が対象となり、初心者でも導入が容易で、トランザクション手数料が低く、エネルギー効率の高いコンセンサスを実現しています。現在、Lisk SDK はベータ版であり、セキュリティ監査の結果も公開されています。
