このADAMANT iOSのリリースでは、アプリがバックグラウンドで実行されている際に生体認証が要求されてしまうバグを修正しました。この修正により、生体認証のプロンプトは適切なフォアグラウンドでの操作時のみにトリガーされるようになり、ユーザーが予期せず認証ダイアログで中断されることが防止されます。
