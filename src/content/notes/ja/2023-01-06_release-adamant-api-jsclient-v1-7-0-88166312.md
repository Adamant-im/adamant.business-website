このリリースでは、`api.setStartupCallback()` メソッドが導入され、開発者がクライアントの起動プロセス中に実行されるコールバックを登録できるようになりました。さらに、APIコンストラクタは現在、第3引数としてコールバックを受け取ることができ、インスタンス化時に起動ロジックを直接提供する便利な方法を提供します。
