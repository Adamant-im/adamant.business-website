## 概要

ADAMANT Currencyinfo v4.1.0 をリリースしました。このバージョンでは、複数の取引所における通貨情報の取得機能を追加しました。

## 新機能

### 複数取引所対応

- Azbit、P2PB2B、StakeCube、Coinstore、FameEX、NonKYC、Bit-Z、CoinDeal、IDCM で通貨情報を取得可能にしました。
- 各取引所のAPIエンドポイントを動的に選択する仕組みを実装。

## 修正内容

- 通貨情報の取得時に発生していたタイムアウトエラーを修正。
- 不正なレスポンス形式に対するロバスト性を向上。

## 今後の予定

- 新しい取引所の追加を継続的に検討しています。
- @adamant_business でアップデート情報を配信します。

## インストール

```bash
npm install adamant-tradebot
cd adamant-tradebot
mm init
```

## 設定

`config.default.jsonc` を編集して、使用する取引所を指定してください。

```jsonc
{
  "exchanges": ["Azbit", "P2PB2B", "StakeCube"]
}
```

## 実行

```bash
mm on
```

## 状態確認

```bash
mm status
```

## ログ確認

```bash
mm logs
```

## 問題診断

```bash
mm doctor
```

## 停止

```bash
mm off
```
