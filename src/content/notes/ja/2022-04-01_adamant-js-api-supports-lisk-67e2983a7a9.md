ADAMANT JavaScript APIライブラリは、コードベースの包括的なリファクタリングに加えて、Liskの暗号鍵をサポートするようになりました。v1.5.0リリースでは、従来のHTTPクライアントをAxiosに置き換えることで、ライブラリ全体でのリクエスト処理と保守性を向上させました。また、コントリビューションガイドラインとコードスタイルも更新され、共同開発がよりスムーズになります。

完全なAPIドキュメントは[ADAMANT API wiki](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification)で確認でき、リリースの詳細は[v1.5.0リリースページ](https://github.com/Adamant-im/adamant-api-jsclient/releases/tag/v1.5.0)に記載されています。
