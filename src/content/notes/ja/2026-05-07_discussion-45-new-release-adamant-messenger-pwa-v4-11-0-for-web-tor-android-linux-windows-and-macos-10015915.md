ADAMANTは、Messenger PWAのバージョン4.11.0をリリースしました。これはWeb、Tor、Android、Linux、Windows、macOSで利用可能です。今回のリリースでは、大きな新機能の追加よりも、信頼性、完成度、成熟度の向上に重点を置いています。

最新のビルドはGitHub上でv4.11.1としてタグ付けされており、初期のv4.11.0リリース後に迅速に適用されたパッチであることを示しています。PWAを利用しているユーザーは、リリース後の修正を適用するために最新のタグへ更新する必要があります。

ADAMANT Messengerは、オープンソースのブロックチェーンベースのメッセージングアプリケーションです。PWA（Progressive Web App）ビルドにより、ユーザーはブラウザでメッセンジャーを実行したり、サポートされているデスクトップおよびモバイルプラットフォームにネイティブアプリ風にインストールしたりできます。オンイオンルーティングアクセスを必要とするユーザー向けにTorでの利用も可能です。
