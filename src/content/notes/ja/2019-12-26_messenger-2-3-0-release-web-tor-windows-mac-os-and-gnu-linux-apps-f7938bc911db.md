ADAMANT Messenger v2.3.0が、Web、Tor、Windows、macOS、GNU/Linux向けのビルドで利用可能になりました。このリリースでは、ビットコインのサポートに加え、トランザクションのステータス処理に関するいくつかの改善とバグ修正が行われました。

複数のブロックチェーンにわたるトランザクション処理が洗練されました。トランザクション詳細画面の確認回数は、動的に更新されるようになりました。また、新しいトランザクションステータスである「Registered」が追加され、ステータスの更新をより効果的に処理できるようになっています。以前は不正確だったイーサリアムのトランザクションステータスも修正されました。さらに、msg.adamant.imのチャットへのリンク処理も正常に動作するように修正されています。

現在のコードベースは、[ADAMANT Messenger GitHubリポジトリ](https://github.com/Adamant-im/adamant-im)のmasterブランチに存在しています。オープンソースのADAMANTプロジェクトへの貢献に関心のある開発者は、ここから最新のソースコードを入手できます。
