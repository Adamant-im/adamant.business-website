このリリースでは、Binance Coin（BNB）、Binance USD（BUSD）、Dai（DAI）、Ethereum Name Service（ENS）、Holo（HOT）、Injective（INJ）、Chainlink（LINK）、Decentraland（MANA）、Polygon（MATIC）、PAX Gold（PAXG）、Quant（QNT）、Ren（REN）、SKALE（SKL）、Status（SNT）、Synthetix Network（SNX）、TrueUSD（TUSD）、Uniswap（UNI）、USD Coin（USDC）、PAX Dollar（USDP）、Stably USD（USDS）、Tether（USDT）、WOO Network（WOO）、Onyxcoin（XCN）など、多数のERC20トークンのサポートを導入します。ユーザーはウォレット内で表示するコインを管理できるようになり、ETHおよびERC20の送金は手数料の削減とスムーズな操作のために最適化されています。

このアップデートでは、チャット内でのトランザクション処理の改善、トランザクション処理の高速化、メッセージ用のよりクリーンで高速なUIも実装されています。ADMアドレスは、マークダウンのハイパーリンクとして表示されるようになりました。macOSではプッシュ通知のサポートとCPU使用率の最適化により、サポートが強化されています。さらに、ヘルスチェックによるより安定した接続、バックグラウンドから復帰時のチャットの更新の改善、およびさまざまな軽微なバグ修正とともに、全体的なパフォーマンスが向上しています。

### 壊れる変更

アプリケーションは、動作するためにiOS 15以降が必要になりました。
