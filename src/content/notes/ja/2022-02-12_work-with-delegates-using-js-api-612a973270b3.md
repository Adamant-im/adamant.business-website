ADAMANT JavaScript APIライブラリv1.4.0では、デリゲート管理のための2つの新しいメソッドが導入されました：`delegateNew()`は新しいデリゲートを登録し、`voteForDelegate()`は既存のデリゲートに投票するためのものです。このリリースには、依存関係の定期的な更新も含まれています。使用方法の詳細は、[API仕様](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification)を参照してください。
