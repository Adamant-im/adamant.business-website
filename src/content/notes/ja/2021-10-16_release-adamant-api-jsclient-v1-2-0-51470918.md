このリリースのADAMANT API JS Clientでは、BTCキーの生成をサポートしています。このアップデートにより、既存のADAMANTキー機能に加えて、Bitcoinキーペアの生成機能がクライアントの暗号化機能に追加されました。

このバージョンでは、破壊的変更は導入されていません。
