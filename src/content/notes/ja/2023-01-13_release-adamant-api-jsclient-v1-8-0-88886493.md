このリリースの ADAMANT API JS クライアントには、新しい `checkNodeTimeout` パラメータが導入され、ノードの可用性を確認する際のタイムアウト期間を開発者が設定できるようになります。この追加により、クライアントが ADAMANT ノードへの接続を確認する方法をより細かく制御できるようになり、アプリケーションが遅いまたは応答のないノードをより適切に処理するのを支援します。
