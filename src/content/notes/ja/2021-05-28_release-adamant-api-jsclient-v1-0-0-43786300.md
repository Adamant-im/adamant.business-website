これはADAMANT API JavaScriptクライアントの最初の安定版リリースです。このライブラリは、チェーン上のデータ読み取りのためのGETリクエストや、トークン・メッセージ送信のためのメソッドなど、ADAMANTブロックチェーンへ高信頼性でアクセスする機能を提供します。

クライアントは、公開鍵のキャッシュ、メッセージの暗号化・復号、トランザクションの作成と署名、ADMキーペアの取り扱いといった暗号化操作を処理します。また、ADMアカウントに紐付けられたEthereumの暗号化アドレスおよび鍵の生成、ADAMANTエポック時間の取り扱いもサポートしています。

その他の機能として、WebSocket接続のサポートや、警告・エラー・情報メッセージ用の組み込みログ機能も備えています。
