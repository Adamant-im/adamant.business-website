このADAMANT API JS Clientのリリースでは、既存のADAMANT機能に加えてLiskブロックチェーンをサポートします。HTTP層はAxiosベースのクライアントに移行され、サポートされるネットワーク全体でのリクエスト処理と保守性が向上しました。

これらの変更に伴い、内部構造の整理や重複コードの削減を目的とした大規模なリファクタリングも実施されています。また、新しいコントリビューターがより簡単に開発を開始できるよう、スタイルガイドラインやコントリビューション用ドキュメントも更新されました。
