ADAMANT Nodeバージョン0.5.1がGitHubで利用可能になりました。これにより、1月に最初に発表されたチャットAPIの開発が完了しました。注目機能は`/api/chatrooms`エンドポイントで、以前のAPIよりもはるかに高速に連絡先やチャットを取得できるようになります。実装の詳細については、[AIP-14仕様](https://aips.adamant.im/AIPS/aip-14)を参照してください。

このアップデートは、すべてのノード所有者にとって必須ではありません。ただし、メッセンジャーアプリと接続する必要があるノードは、新しいAPIの利点を活用するために最新バージョンにアップグレードする必要があります。ADAMANT Androidチームはすでに新しいエンドポイントを使用して計測可能な速度向上を実現しており、他のADAMANTアプリでも同様の改善が採用により期待されています。
