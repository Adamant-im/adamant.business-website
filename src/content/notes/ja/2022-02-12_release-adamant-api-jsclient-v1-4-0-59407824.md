このリリースでは、ADAMANT API JS Clientに `delegateNew()` および `voteForDelegate()` の2つの新メソッドが追加されました。これらの追加により、クライアントのデリゲート関連機能が拡張されています。さらに、プロジェクトの依存パッケージが最新バージョンに更新されています。
