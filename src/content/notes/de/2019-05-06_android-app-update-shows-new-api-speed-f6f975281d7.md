### Schnellere API im neuesten Android-Release

Nach früheren Ankündigungen zur Verbesserung der API-Geschwindigkeit für den ADAMANT-Knoten hat das Android-Entwicklungsteam ein Update veröffentlicht, das eine neue Chats-API enthält.

Ein Vergleich zwischen der alten und der aktualisierten API zeigt, dass sowohl Chat-Operationen als auch der Anmeldevorgang deutlich schneller abgeschlossen werden als zuvor. Weitere Optimierungen sind geplant.

Die neueste APK ist auf der [GitHub-Releases-Seite für Android](https://github.com/Adamant-im/adamant-android/releases/) verfügbar. Bestehende Nutzer sollten sicherstellen, dass sie die neueste Version verwenden; falls die App sich nicht automatisch aktualisiert, sollte die vorherige Version deinstalliert und die neue APK manuell installiert werden.

Feedback und Fehlerberichte können über das GitHub-Repository eingereicht werden. Das Team bereitet außerdem eine Veröffentlichung im Google Play Store vor.
