Diese Version führt die Unterstützung für Resfinex Token (RES) ein und aktualisiert den Namen sowie das Logo von Stably Dollar (USDS). Die Höhe des Senden-Token-Menüs wurde für eine bessere Bedienbarkeit angepasst, und der Dialog zum Kaufen und Verkaufen von Token wurde aktualisiert.

Eine Client-Node-Socket-Verbindung ist nun verfügbar, ebenso ein neuer Status „Im Blockchain“ für Nachrichten. Die Validierung von BTC-Adressen wurde hinzugefügt, und eine Korrektur behebt die Validierung beim Einfügen von Adressen aus der Zwischenablage. Eine weitere Korrektur behebt ein Problem mit negativen Beträgen beim Senden.

### Breaking changes

In dieser Version werden keine inkompatiblen Änderungen eingeführt.
