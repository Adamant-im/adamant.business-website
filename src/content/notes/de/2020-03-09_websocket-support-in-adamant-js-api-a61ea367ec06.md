ADAMANT ist ein Open-Source-Projekt mit einer öffentlichen Blockchain-API, einem Konsolenwerkzeug und einem JavaScript-API-Client. Der JavaScript-API-Client wurde auf Version 0.5.0 aktualisiert, wobei die wichtigste Neuerung die WebSocket-Unterstützung ist.

Mit WebSocket-Verbindungen erhalten Anwendungen sofortige Benachrichtigungen über neue Transaktionen und Nachrichten. Dadurch wird die Latenz im Vergleich zum Abfragen der REST-API reduziert, sodass Bots und Dienste schneller auf Nutzeranfragen reagieren können.

Die Veröffentlichung ist auf GitHub verfügbar, und die Dokumentation zur JavaScript-API wird im Projekt-Wiki gepflegt.
