ADAMANT Node v0.7.0 konzentriert sich auf Aufräumarbeiten und die Wartung von Abhängigkeiten. Die Version aktualisiert die Projektabhängigkeiten, verschiebt Legacy-Code in ein dediziertes `/legacy`-Verzeichnis und entfernt veraltete Dokumentation. Diese Änderungen haben keine Auswirkungen auf den Netzwerk-Consensus, daher müssen Node-Betreiber das Update nicht durchführen.

Die Details zum Release sind auf der [v0.7.0 GitHub Release-Seite](https://github.com/Adamant-im/adamant/releases/tag/v0.7.0) verfügbar.
