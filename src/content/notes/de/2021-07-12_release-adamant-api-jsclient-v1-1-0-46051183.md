Diese Version des ADAMANT API JavaScript-Clients unterstützt nun die Erzeugung von DASH-Schlüsseln. Entwickler, die die Bibliothek nutzen, können jetzt neben der bestehenden Funktionalität zur Erzeugung von Kryptowährungsschlüsseln auch DASH-Schlüssel generieren.
