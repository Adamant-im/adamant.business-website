Dieses Patch-Release behebt ein Problem, das die Übertragung von Token innerhalb von Chatnachrichten betraf. Nutzer, die Probleme beim Senden von Tokenübertragungen über die Chat-Oberfläche hatten, sollten nun feststellen, dass die Funktionalität wie erwartet funktioniert.
