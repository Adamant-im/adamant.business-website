Diese Version führt einen neuen `logLevel`-Parameter für den ADAMANT API JavaScript-Client ein. Der Parameter akzeptiert die folgenden Werte in aufsteigender Ausführlichkeit: `none` < `error` < `warn` < `info` < `log`.
