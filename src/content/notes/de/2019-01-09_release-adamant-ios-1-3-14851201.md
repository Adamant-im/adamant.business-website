ADAMANT iOS Version 1.3 führt die Unterstützung für das Lisk-Wallet sowie Benutzeravatare ein, um ein persönlicheres Messaging-Erlebnis zu ermöglichen. Nutzer können jetzt ADM-Überweisungen mit Kommentaren versenden und profitieren von der vollen Unterstützung für iPad-Geräte, was die Kompatibilität mit weiteren Endgeräten verbessert.

Diese Version beinhaltet außerdem die Darstellung von Markdown für eine anspruchsvollere Nachrichtenformatierung. Das Senden von Nachrichten wurde hinsichtlich Zuverlässigkeit verbessert, und QR-Codes wurden für eine bessere Scanleistung optimiert. Ergänzt wird das Update durch allgemeine Stabilitätsverbesserungen und Fehlerbehebungen.
