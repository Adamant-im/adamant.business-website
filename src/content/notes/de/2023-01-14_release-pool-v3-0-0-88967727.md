Dieses Release aktualisiert die Projektabhängigkeiten und schreibt das Dashboard mit Svelte neu. Die gesamte Codebasis wurde überarbeitet und optimiert, und mehrere Fehler wurden behoben.

### Breaking changes

Die Unterstützung für Node.js-Versionen unterhalb von 18.12.1, der aktuellen LTS-Version, wurde eingestellt. Der Pool verwendet nun lowdb als Datenbank. Weitere Einzelheiten finden Sie im Migrationsabschnitt der README-Datei.
