Der ADAMANT Exchanger, der Kryptowährungstauschgeschäfte direkt innerhalb von ADAMANT-Chats ermöglicht, wurde auf Version 2.2.2 aktualisiert. Der Exchanger unterstützt nun Bitcoin (BTC), Stably Dollar (USDS, ERC20) und Binance Coin (BNB, ERC20), zusätzlich zu den bereits unterstützten Währungen Ethereum (ETH), Dash (DASH), ADAMANT (ADM) und dem Resfinex-Token (RES, ERC20). BZ Coin wurde von der Liste der unterstützten Assets entfernt.

Um einen Tausch zu starten, überweisen Nutzer Kryptowährung an den ADAMANT Exchange Bot innerhalb einer Chat-Sitzung. Die Eingabe von **/help** an den Bot liefert aktuelle Tauschparameter und verfügbare Befehle.

![ADAMANT Exchanger unterstützt jetzt Bitcoin (BTC), Stably Dollar (USDS) und Binance Coin (BNB)](/images/engineering-notes/medium/5a307a4a4c40/002-0-xkzcwwyruweqjc1q.webp)

Der Exchanger ist Open Source im Rahmen des ADAMANT-Projekts, und Betreiber können ihre eigene Instanz betreiben. Die vollständige Version ist auf GitHub im Repository `adamant-im/adamant-exchangebot` unter dem Tag `v2.2.2` verfügbar.
