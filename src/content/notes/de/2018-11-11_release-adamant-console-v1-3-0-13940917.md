Diese Version führt zwei neue Befehle ein: `account new` und `get transactions`. Außerdem wird die Option `--passPhrase` unterstützt, die es Benutzern ermöglicht, eine Passphrase direkt beim Ausführen eines Befehls anzugeben.

Mehrere Fehlerbehebungen sind in dieser Version enthalten. Veraltete `new Buffer()`-Aufrufe wurden ersetzt, und ein Fehler, der zu einer fehlerhaften Konfigurationsladung führte, wurde behoben. Ein vorübergehendes Abhängigkeitsupdate wurde angewendet, solange auf die Zusammenführung von dthree/vorpal#322 gewartet wird.
