Diese Wartungsversion von ADAMANT Messenger führt Verbesserungen im visuellen Stil über die gesamte Client-Oberfläche ein und verbessert so die allgemeine Konsistenz und Optik.

Das Update fügt außerdem Benachrichtigungsmarkierungen für neue Nachrichten hinzu, wodurch Benutzer ungelesene Unterhaltungen auf einen Blick leichter erkennen können.

Diese Version enthält keine inkompatiblen Änderungen.
