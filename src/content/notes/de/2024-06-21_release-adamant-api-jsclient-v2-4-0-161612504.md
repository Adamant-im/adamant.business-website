Diese Version behebt die Validierung von öffentlichen Schlüsseln und Delegatennamen für die `api.voteForDelegate()`-Methode. Außerdem wird nun überprüft, ob eine Adresse beim Aufruf von `api.voteForDelegate()` tatsächlich ein Delegat ist. Ebenfalls hinzugefügt wurden detailliertere Fehlermeldungen bei Validierungsfehlern für dieselbe Methode.
