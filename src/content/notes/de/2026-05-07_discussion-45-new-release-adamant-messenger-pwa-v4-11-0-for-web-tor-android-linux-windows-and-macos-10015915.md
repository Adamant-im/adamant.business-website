ADAMANT hat Version 4.11.0 seiner Messenger-PWA veröffentlicht, die nun für Web, Tor, Android, Linux, Windows und macOS verfügbar ist. Das Update steht im Zeichen von Verbesserungen bei Zuverlässigkeit, Feinschliff und Reife, nicht von großen neuen Funktionen.

Der neueste Build ist auf GitHub als v4.11.1 gekennzeichnet, was auf einen schnellen Nachfolgepatch nach der ursprünglichen Veröffentlichung von 4.11.0 hinweist. Nutzer der PWA sollten auf das neueste Tag aktualisieren, um eventuelle Nachbesserungen nach der Veröffentlichung zu erhalten.

ADAMANT Messenger ist eine quelloffene, blockchainbasierte Messaging-Anwendung. Die PWA-Version (Progressive Web App) ermöglicht es Nutzern, den Messenger im Browser auszuführen oder ihn als native-ähnliche Anwendung auf unterstützten Desktop- und Mobilplattformen zu installieren, einschließlich Tor für Nutzer, die Onion-Routing-Zugang benötigen.
