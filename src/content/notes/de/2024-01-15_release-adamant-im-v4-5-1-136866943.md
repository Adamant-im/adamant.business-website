Diese Version ermöglicht das Speichern von Nachrichtenentwürfen und Antworten im Chat, fügt die Unterstützung für Chinesisch (vereinfacht) hinzu und integriert das STORJ-Token in die Wallet.

Mehrere Fehler wurden in dieser Version behoben. Der linke Chevron im Wallet ist nun anklickbar, und seine Farben wurden für eine bessere Sichtbarkeit angepasst. Ein Problem mit der Pluralform von Adressen im Russischen wurde ebenfalls behoben.

Auf technischer Seite wurden Abhängigkeiten wie Vue, Vuetify und Electron auf neuere Versionen aktualisiert. Das README wurde um Statusanzeigen (shields) ergänzt, und die Standardsprache kann nun über eine Umgebungsvariable konfiguriert werden.

### Breaking changes

Es gibt keine Breaking Changes in diesem Release.
