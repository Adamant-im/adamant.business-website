ADAMANT Console v3.1.0 aktualisiert die Konsole für ADAMANT Node v0.10.0 und erneuert die CLI, JSON-RPC, JavaScript-Wrapper, Dokumentation und Validierungstoolkette. Diese Version fügt Unterstützung für ADAMANT Node v0.10.0 Antworten und Abfragen über `adamant-api` v3 hinzu. Außerdem werden Aktualisierungen eingeführt für den Knotenstatus, Chaträume/-nachrichten, Chattransaktionen, die Transaktions-Option `returnUnconfirmed`, Delegatensuche und Direktüberweisungs-Abfragen. Metadaten und Abhängigkeiten der Pakete wurden aktualisiert, ebenso wie eine neue VitePress-Dokumentationsseite, automatisch generierte TypeDoc-API-Referenz und die Bereitstellung auf GitHub Pages bei einer Veröffentlichung. Weitere Verbesserungen umfassen CLI-Hilfe-Beispiele, erweiterte JSON-RPC-Abdeckung, öffentliche API JSDoc, Syntaxhervorhebung für formatierte JSON-Ausgaben und erhöhte Testabdeckung für API-Wrapper, CLI-Hilfe-Verhalten, Konfigurations-/Client-Metadaten, Eingabeverlauf und Protokollierung.
