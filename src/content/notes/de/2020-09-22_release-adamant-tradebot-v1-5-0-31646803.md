Diese Version erweitert die Börsenunterstützung, indem Orderbücher für Bit-Z und CoinDeal hinzugefügt werden, wodurch Nutzer eine umfassendere Sicht auf die Marktliquidität über zusätzliche Handelsplattformen erhalten. Ein neuer Befehl zum Abrufen von Einzahlungsadressen wurde ebenfalls eingeführt, wodurch der Prozess der Kontofinanzierung direkt über die Bot-Oberfläche vereinfacht wird.
