## Änderungen

Diese Version senkt die Gebühr für das Senden von Nachrichten von 0,005 auf 0,001 AMD und verringert so die Kommunikationskosten im Netzwerk. Die Anwendung prüft nun häufiger eingehende Nachrichten, wodurch die Reaktionsgeschwindigkeit verbessert wird. Ein Button, der zur ICO-Seite verlinkt, wurde hinzugefügt, und Nutzer erhalten nun ICO-Nachrichten anstelle der früheren Pre-ICO-Nachrichten. Die Favicons wurden aktualisiert, ebenso wurden einige kleinere interne Änderungen vorgenommen.

## Fehlerbehebungen

Ein Problem, bei dem Nachrichten verschwanden, wurde behoben. Ein vorübergehender Fix für den Local Storage wurde angewendet, der in zukünftigen Versionen vollständig entfernt werden soll. Das Problem mit der Link-Anzeige (#16), das potenziell zur Täuschung von Nutzern missbraucht werden konnte, wurde behoben.

## Übersetzungen

Neue Übersetzungen wurden für Arabisch, Deutsch, Portugiesisch (beigesteuert von @DataDrug via PR #17), Rumänisch, Schwedisch (beigesteuert von @pdahlberg via PR #29) und Vietnamesisch hinzugefügt.
