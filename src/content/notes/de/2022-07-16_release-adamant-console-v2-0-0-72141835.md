Diese Version bietet eine vollständige Neugestaltung des ADAMANT Consoles, das nun auf adamant-api basiert. Der Console wurde die Fähigkeit hinzugefügt, Nachrichten zu entschlüsseln, und verfügt über eine integrierte Hilfe für verfügbare Befehle. Zur Vereinfachung der Nutzung wurde ein interaktiver Modus eingeführt.

Das Paket kann nun als JavaScript-Bibliothek eingebunden werden, wodurch sich die Integrationsmöglichkeiten über die Befehlszeile hinaus erweitern. Die Abhängigkeiten wurden aktualisiert, und mehrere Fehlerbehebungen sind enthalten.

### Breaking changes

Die vollständige Neugestaltung bedeutet, dass sich die interne Struktur und die APIs erheblich geändert haben. Bestehende Skripte oder Integrationen, die auf der vorherigen Architektur basieren, müssen aktualisiert werden, um mit der neuen, auf adamant-api basierenden Grundlage zu funktionieren.
