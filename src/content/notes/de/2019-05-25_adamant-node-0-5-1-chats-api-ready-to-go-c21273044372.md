ADAMANT Node Version 0.5.1 ist jetzt auf GitHub verfügbar und vervollständigt die im Januar erstmals angekündigte Arbeit an der Chats API. Das Hauptmerkmal ist der `/api/chatrooms`-Endpunkt, der das Abrufen von Kontakten und Chats deutlich schneller ermöglicht als die vorherige API. Entwickler sollten zur Implementierung die [AIP-14-Spezifikation](https://aips.adamant.im/AIPS/aip-14) konsultieren.

Dieses Update ist nicht für alle Knotenbetreiber verpflichtend. Knoten, die eine Verbindung mit Messenger-Apps herstellen müssen, sollten jedoch auf die neueste Version aktualisieren, um die Vorteile der neuen API nutzen zu können. Das ADAMANT Android-Team hat bereits messbare Geschwindigkeitsverbesserungen mit dem neuen Endpunkt demonstriert, und ähnliche Verbesserungen werden erwartet, sobald andere ADAMANT-Apps diesen übernehmen.
