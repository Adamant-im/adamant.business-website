Diese Version des ADAMANT API JS Clients führt einen neuen `checkNodeTimeout`-Parameter ein, der es Entwicklern ermöglicht, die Timeout-Dauer bei der Prüfung der Verfügbarkeit von Knoten zu konfigurieren. Diese Erweiterung bietet eine genauere Kontrolle darüber, wie der Client die Verbindung zu ADAMANT-Knoten überprüft, und hilft Anwendungen, langsamer reagierende oder nicht antwortende Knoten besser zu behandeln.
