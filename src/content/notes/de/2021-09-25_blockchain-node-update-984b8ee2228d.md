ADAMANT Node v0.6.5 ist jetzt verfügbar. Diese Version konzentriert sich auf Codequalität und Tooling statt auf Protokolländerungen. Ein Upgrade ist daher optional und hat keine Auswirkungen auf den Netzwerkkonsens.

Die standardmäßigen Broadcast-Einstellungen wurden geändert und das Installations-Bash-Skript aktualisiert. Entwicklungspakete wurden aktualisiert, der Code wurde gelintet und die Test-Suite behoben.

Zwei bemerkenswerte Neuerungen sind der Befehlszeilenparameter `-genesis` und ein dedizierter `portWS` 36665 für das Testnet. Dadurch lassen sich Genesis- und Testnet-Konfigurationen einfacher ohne manuelle Anpassungen ausführen.

Installationsanweisungen für Ubuntu, macOS und Windows sind in den [ADAMANT Node v0.6.5 Release Notes](https://github.com/Adamant-im/adamant/releases/tag/v0.6.5) verfügbar.
