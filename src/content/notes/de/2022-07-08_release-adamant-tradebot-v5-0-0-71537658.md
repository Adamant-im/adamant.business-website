Diese Version konzentriert sich auf interne Umstrukturierungen und Aktualisierungen von Abhängigkeiten für den ADAMANT Tradebot. Im Rahmen der Wartungsarbeiten wurde eine `config.default.json`-Datei hinzugefügt, um eine klarere Basiskonfiguration für Betreiber bereitzustellen.

Eine neue Marktmacheroption, `mm_orderBookMaxOrderPercent`, wurde eingeführt, um eine feinere Kontrolle über die Beteiligung am Orderbuch zu ermöglichen.

### Breaking changes
In dieser Version werden keine inkompatiblen Änderungen eingeführt.
