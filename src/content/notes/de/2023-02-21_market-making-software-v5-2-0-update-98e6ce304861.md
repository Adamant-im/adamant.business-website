Der ADAMANT Trade & Market-making Bot ist eine kostenlose Open-Source-Software zum Ausführen von Trades auf Kryptobörsen, zur Generierung von Handelsvolumen, zum Halten der Spread- und Liquiditätslage, zum Setzen von Preisspannen und zum Aufbau dynamischer Orderbücher. Die Version 5.2.0 ist ein Wartungsrelease, das auf Stabilität und Zuverlässigkeit der Connectoren abzielt.

Die bedeutendste Änderung in diesem Release ist eine komplette Neuschreibung des P2PB2B-API-Connectors, wodurch die Kommunikation mit der Börse verbessert und connectorbezogene Fehler reduziert werden sollten. Das Release beinhaltet außerdem allgemeine Refactorings, Aktualisierungen von Abhängigkeiten und Fehlerbehebungen im gesamten Codebase.

Der Quellcode ist auf [GitHub](https://github.com/Adamant-im/adamant-tradebot) verfügbar.
