Diese Version von ADAMANT iOS bringt mehrere Verbesserungen bei der Benutzerauthentifizierung und der Benutzeroberfläche. Benutzer können nun über Sitzungen hinweg angemeldet bleiben, wobei die Autorisierung über einen PIN-Code eine zusätzliche Sicherheitsebene bietet. Die biometrische Authentifizierung wird sowohl über Touch ID als auch Face ID unterstützt.

Die Navigationsleisten wurden aktualisiert, um große Titel im Stil von iOS 11 zu verwenden, was eine modernere und besser lesbare Oberfläche ermöglicht. Zudem wurden allgemeine Codebereinigungen durchgeführt, um die Wartbarkeit zu verbessern.

### Breaking changes

In dieser Version wurden keine inkompatiblen Änderungen eingeführt.
