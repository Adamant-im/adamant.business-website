Diese Version des ADAMANT API JS Clients unterstützt nun die Generierung von BTC-Keys. Die Aktualisierung erweitert die kryptografischen Funktionen des Clients um die Erstellung von Bitcoin-Schlüsselpaaren, zusätzlich zur bestehenden ADAMANT-Schlüsselfunktionalität.

In dieser Version wurden keine Breaking Changes eingeführt.
