Diese Version fügt Logging für das Datenbank-Migrationsskript hinzu. Der ADAMANT-Knoten protokolliert nun Nachrichten vor und nach der Ausführung des Migrationsskripts:

```
[inf] 2023-12-22 04:00:25 | Found 5 pending migrations. Start executing, this may take a while…
[inf] 2023-12-22 04:17:13 | Migrations have been successfully completed.
```

Der Standardwert für die `cors.origin`-Eigenschaft wurde von der Spiegelung des Anfrageherkunftsursprungs (`true`) auf `*` geändert. Weitere Informationen zur CORS-Konfiguration für den ADAMANT-Knoten finden Sie in der Dokumentation.

```json5
{
  "cors": {
    "origin": "*" // default
  }
}
```

Die Datei `config.json` wurde zu `.gitignore` hinzugefügt. Benutzer müssen die Datei `config.json` nun selbst erstellen oder `config.default.json` kopieren, um den ADAMANT-Knoten zu starten.

```
cp config.default.json config.json
```

### Breaking changes

Der Standard-CORS-Header-Wert für `Access-Control-Allow-Origin` hat sich von der Spiegelung des Anfrageherkunftsursprungs auf `*` geändert. Benutzer, die auf das vorherige Verhalten mit reflektiertem Ursprung angewiesen sind, sollten ihre `config.json` entsprechend aktualisieren. Zusätzlich wird `config.json` nun von git ignoriert, sodass Benutzer diese Datei manuell erstellen oder kopieren müssen, bevor sie den Knoten starten.
