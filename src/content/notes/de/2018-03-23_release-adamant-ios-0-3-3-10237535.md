Diese Version von ADAMANT iOS behebt einen Fehler, bei dem die biometrische Authentifizierung angefordert wurde, während die App im Hintergrund lief. Die Korrektur stellt sicher, dass biometrische Aufforderungen nur während geeigneter Interaktionen im Vordergrund ausgelöst werden, wodurch unerwartete Authentifizierungsdialoge verhindert werden, die Benutzer unterbrechen könnten.
