Diese Version fügt Unterstützung für Rich Content Messages hinzu, wie in [AIP 5](https://aips.adamant.im/AIPS/aip-5) definiert, und ermöglicht eine anspruchsvollere Nachrichtenformatierung innerhalb der ADAMANT-Messaging-Plattform. Zusätzlich wurde Unterstützung für Signal Messages gemäß [AIP 6](https://aips.adamant.im/AIPS/aip-6) eingeführt. Außerdem behebt diese Version Filter in der State(KVS)-Transaktions-API.
