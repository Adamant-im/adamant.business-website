## Verbesserungen

- **Benachrichtigungen:** Verbesserte Stabilität und Zuverlässigkeit von Push-Benachrichtigungen.
- **Benutzeroberfläche:** Optische Anpassungen in der Chat-Ansicht für bessere Lesbarkeit.
- **Leistung:** Reduzierte Speichernutzung bei längerer Nutzung.

## Fehlerbehebungen

- Behoben: Problem mit der Synchronisation von Nachrichten nach dem Wiedereinschalten der App.
- Behoben: Gelegentlicher Absturz beim Öffnen von Medienanlagen in älteren Chats.
- Behoben: Falsche Anzeige des Verbindungsstatus im Offline-Modus.

## Hinweise zur Installation

Die Aktualisierung ist über den App Store verfügbar. Stellen Sie sicher, dass Sie über eine stabile Internetverbindung verfügen, um alle Funktionen ordnungsgemäß nutzen zu können.

Weitere Informationen finden Sie im [ADAMANT Messenger](https://adammessenger.org) Projekt auf GitHub.

Fehler können im offiziellen Repository gemeldet werden: [github.com/adamant-im/adamant-ios](https://github.com/adamant-im/adamant-ios).

Folgen Sie [@adamant_business](https://t.me/adamant_business) auf Telegram für Updates.
