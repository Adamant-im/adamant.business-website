Diese Version konzentriert sich auf eine umfassende Modernisierung des ADAMANT Messenger-Codebases. Die Anwendung wurde auf Vue 3 und Vuetify 3 aktualisiert, und Stylus wurde zugunsten einer kompletten Neuschreibung in SASS entfernt. Das Build-System wurde von Webpack zu Vite migriert, und der Test-Runner wurde von Jest zu Vitest gewechselt. TypeScript-Unterstützung wurde hinzugefügt, ebenso wie Prettier- und EditorConfig-Konfigurationen. Der Paketmanager wurde von Yarn auf NPM gewechselt. Hauptversionen von axios, socket.io-client, @mdi/font und anderen Abhängigkeiten wurden aktualisiert, und alle Pakete wurden auf die neuesten Neben- und Patch-Versionen gebracht. Fehlgeschlagene Tests und veraltete Snapshots wurden behoben, Browserslist- und ESLint-Konfigurationen wurden aus package.json in separate Dateien verschoben, und mehrere Komponenten wurden umgeschrieben, um die Composition API zu verwenden.

Auf der Funktionsebene wurde der Bounty Wallet Bot aus den Chats entfernt (#482), obwohl Benutzer, die zuvor mit dem Bot kommuniziert haben, weiterhin auf ihren Chatverlauf zugreifen können. Dem WebAppManifest wurden zusätzliche Informationen für PWA-Installationen hinzugefügt (#481). Im BuyTokensDialog wurde Coindeal durch Azbit ersetzt (#476). GitHub Actions Workflows für Electron- und PWA-Builds wurden hinzugefügt (#458, #461).

Mehrere Fehler wurden in dieser Version behoben. Ein Fehler, der beim Laden der Delegierten in einem neuen Konto in der DelegatesTable auftrat, wurde behoben (#475). Ein Scroll-Event, der auf einigen Geräten in der Transaktionsansicht nicht ausgelöst wurde, wurde korrigiert (#474). Der Chat behält nun die Scroll-Position bei Zurück-Navigation (#470), und die Chat-Ansicht scrollt beim Eintreffen einer neuen Nachricht automatisch nach unten (#455). Zudem wurden zahlreiche kleinere Stilkorrekturen vorgenommen (#454, #465, #466, #467, #468, #469, #471, #473, #479, #480).
