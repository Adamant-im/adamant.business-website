Diese Version von ADAMANT iOS behebt ein Problem bei der Knotenverbindung, das verhinderte, dass die App eine Verbindung herstellen konnte. Ein Absturz beim Start wurde ebenfalls behoben. Der Dash-Explorer wurde aktualisiert, und mehrere weitere kleinere Fehlerbehebungen sind enthalten.

Keine inkompatiblen Änderungen wurden in dieser Version eingeführt.
