ADAMANT ist ein dezentraler Messenger, der keine Telefonnummer oder E-Mail erfordert. Er fragt nicht auf Ihre Kontakte zu und neue Chats werden über anonyme Adressen gestartet. Der Messenger ist verfügbar für Web, Tor, iOS, Android, Windows, macOS und GNU/Linux.

### Konto erstellen

Nach Auswahl einer Anwendungssprache wählen Sie *Neu erstellen* und speichern den generierten Passsatz. Dieser Passsatz ist die einzige Anmeldeinformation, die zum Einloggen auf jedem Gerät benötigt wird, muss daher geheim gehalten werden und kann bei Verlust nicht wiederhergestellt werden. Ein Benutzername ist nicht erforderlich, da der Passsatz eindeutig ist. Neue Konten können willkommene ADM-Token erhalten, indem sie im Reiter *Konto* auf *Kostenlose ADM-Token* tippen und ein Captcha lösen.

### Einstellungen

![So starten Sie mit einem Blockchain Messenger](/images/engineering-notes/medium/54d1eb9704e6/002-0-cywyk7bs8vqaiuw8.webp)

Für schnellere Anmeldungen auf einem einzelnen Gerät können Sie *Angemeldet bleiben* aktivieren und ein lokales Passwort festlegen. Der Passsatz ist weiterhin erforderlich, wenn Sie sich auf anderen Geräten anmelden. Push-Benachrichtigungen können aktiviert werden, um über neue Nachrichten informiert zu werden.

### Kontakte und Chats starten

Ihre ADAMANT-Adresse, die im Reiter *Konto* angezeigt wird, sieht wie `U8281862340475070014` aus und ist eindeutig. Sie können sie als reinen Text, als Link wie `https://msg.adamant.im?address=U8281862340475070014` oder als QR-Code teilen. Das Öffnen des Links in einem Browser führt den Empfänger direkt nach dem Einloggen oder Erstellen eines Kontos zu einem Chat mit Ihnen.

![So starten Sie mit einem Blockchain Messenger](/images/engineering-notes/medium/54d1eb9704e6/003-0-jb6t2kkgaqns5wsu.webp)

![So starten Sie mit einem Blockchain Messenger](/images/engineering-notes/medium/54d1eb9704e6/004-0-lgzwassv0cfbikod.webp)

Um einen Chat zu starten, öffnen Sie den Reiter *Chats*, tippen Sie auf *Neuen Chat starten* und fügen Sie die Adresse Ihres Gesprächspartners ein oder scannen dessen QR-Code. Sie können jedem Kontakt einen benutzerdefinierten Namen zuweisen. Emojis werden über die native Tastatur auf Mobilgeräten, Win + "." unter Windows oder Ctrl + Cmd + Leertaste unter macOS unterstützt.

### Kryptowallets und Überweisungen

![So starten Sie mit einem Blockchain Messenger](/images/engineering-notes/medium/54d1eb9704e6/005-0-gmegpvtzjfa13ucf.webp)

ADAMANT beinhaltet integrierte Kryptowallets. Wenn Sie beispielsweise Ethereum im Reiter *Konto* auswählen, wird Ihre ETH-Einzahlungsadresse angezeigt, die von jedem externen Wallet aufgeladen werden kann. Sobald eine Überweisung bestätigt ist, aktualisieren sich automatisch der Kontostand und der Transaktionsverlauf.

![So starten Sie mit einem Blockchain Messenger](/images/engineering-notes/medium/54d1eb9704e6/006-0-qwhweawd2kux1bdl.webp)

Sie können Kryptowährungen direkt an Adressen über den Reiter *Konto* oder direkt an Chatpartner senden. In einer Unterhaltung drücken Sie "+" und wählen *ETH senden*, geben dann den Betrag und einen optionalen Kommentar ein. Die Überweisung erscheint im Chat und der Kontostand des Empfängers wird aktualisiert.

### Bots

![So starten Sie mit einem Blockchain Messenger](/images/engineering-notes/medium/54d1eb9704e6/007-0-y2farpvkrbe0m-4h.webp)

ADAMANT unterstützt Bots, darunter einen Kryptowährungstauscher und einen Wettbot, die bereits im Wallet vorinstalliert sind. Um den Tauscher zu nutzen, senden Sie ihm */help*, überweisen anschließend die Kryptowährung, die Sie tauschen möchten, und geben in der Kommentarspalte die gewünschte Ausgangswährung an.
