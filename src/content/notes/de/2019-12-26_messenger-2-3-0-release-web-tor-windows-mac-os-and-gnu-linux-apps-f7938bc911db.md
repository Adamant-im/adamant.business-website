ADAMANT Messenger v2.3.0 ist jetzt mit Builds für Web, Tor, Windows, macOS und GNU/Linux verfügbar. Die Veröffentlichung führt Bitcoin-Unterstützung ein und bringt mehrere Verbesserungen bei der Handhabung von Transaktionsstatus sowie Fehlerbehebungen.

Die Transaktionsverarbeitung wurde für mehrere Chains optimiert. Die Anzeige der Bestätigungsanzahl auf den Transaktionsdetails aktualisiert sich nun dynamisch, und ein neuer Status „Registered“ für Transaktionen ermöglicht eine effektivere Verarbeitung von Statusaktualisierungen. Fehlerhafte Ethereum-Transaktionsstatus wurden behoben. Zudem wurde die Verarbeitung von Links zu msg.adamant.im-Chats korrigiert.

Der aktuelle Code befindet sich im master-Branch des [ADAMANT Messenger GitHub-Repositorys](https://github.com/Adamant-im/adamant-im). Entwickler, die am Open-Source-Projekt ADAMANT mitwirken möchten, finden dort den neuesten Quellcode.
