ADAMANT InfoServices wurde auf v3.5.0 aktualisiert und bietet eine kombinierte API für Fiat- und Kryptowährungskurse. Der Dienst aggregiert Daten aus mehreren Quellen, darunter MOEX, Currency-Api, ExchangeRates.host, CoinMarketCap, CryptoCompare und CoinGecko, und vergleicht die Ergebnisse, um Abweichungen zu erkennen und die Zuverlässigkeit zu verbessern.

Da der Dienst auf Ihrem eigenen Server läuft, bleiben historische Kursdaten unter Ihrer Kontrolle und die Nutzung ist für alle Zwecke kostenlos. Das Projekt ist quelloffen und wird unter der ADAMANT-Organisation auf GitHub gehostet.

Die API-Dokumentation ist im [InfoServices API Wiki](https://github.com/Adamant-im/adamant-currencyinfo-services/wiki/InfoServices-API-documentation) verfügbar.
