Dies ist ein Hotfix-Release, das mehrere Probleme in der ADAMANT Messenger-Anwendung behebt.

Das Release behebt ein Problem mit der Abstimmungsfunktion. Außerdem wird ein Problem mit der falschen Reihenfolge von Anfragen behoben, das einigen Nutzern die Erhaltung kostenloser Token verwehrt hatte. Zusätzlich wurde ein Problem mit neuen Chats behoben.
