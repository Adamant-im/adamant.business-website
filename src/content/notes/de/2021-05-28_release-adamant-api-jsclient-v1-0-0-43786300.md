Dies ist die erste stabile Version des ADAMANT API JavaScript-Clients. Die Bibliothek bietet zuverlässigen Zugriff auf die ADAMANT-Blockchain, einschließlich GET-Anfragen zum Lesen von On-Chain-Daten sowie Methoden zum Senden von Tokens und Nachrichten.

Der Client übernimmt kryptografische Operationen wie das Zwischenspeichern öffentlicher Schlüssel, das Verschlüsseln und Entschlüsseln von Nachrichten, das Erstellen und Signieren von Transaktionen sowie die Arbeit mit ADM-Schlüsselpaaren. Außerdem unterstützt er die Generierung einer Ethereum-Kryptoadresse und zugehöriger Schlüssel, die an ein ADM-Konto gebunden sind, sowie die Verwendung von ADAMANT-Epochenzeit.

Zu den weiteren Funktionen gehören die Unterstützung von WebSocket-Verbindungen und ein integriertes Logging für Warnungen, Fehler und Informationsmeldungen.
