Diese Version führt die Unterstützung für eine breite Palette von ERC20-Token ein, darunter Binance Coin (BNB), Binance USD (BUSD), Dai (DAI), Ethereum Name Service (ENS), Holo (HOT), Injective (INJ), Chainlink (LINK), Decentraland (MANA), Polygon (MATIC), PAX Gold (PAXG), Quant (QNT), Ren (REN), SKALE (SKL), Status (SNT), Synthetix Network (SNX), TrueUSD (TUSD), Uniswap (UNI), USD Coin (USDC), PAX Dollar (USDP), Stably USD (USDS), Tether (USDT), WOO Network (WOO) und Onyxcoin (XCN). Benutzer können jetzt sichtbare Coins in einer Wallet verwalten, und ETH- sowie ERC20-Überweisungen wurden für niedrigere Gebühren und einen reibungsloseren Betrieb optimiert.

Das Update bringt außerdem eine verbesserte Verarbeitung von Transaktionen im Chat, schnellere Transaktionsabwicklung und eine übersichtlichere, schnellere Benutzeroberfläche für Nachrichten. ADM-Adressen werden nun als Markdown-Hyperlinks dargestellt. Die macOS-Unterstützung wurde durch Push-Benachrichtigungen und Optimierungen der CPU-Auslastung verbessert. Zudem verfügt die App über eine stabilere Verbindung mit Health-Checks, eine verbesserte Chat-Aktualisierung beim Zurückkehren aus dem Hintergrund sowie allgemeine Leistungsverbesserungen zusammen mit verschiedenen kleineren Fehlerbehebungen.

### Breaking changes

Die Anwendung benötigt ab sofort iOS 15 oder höher, um zu funktionieren.
