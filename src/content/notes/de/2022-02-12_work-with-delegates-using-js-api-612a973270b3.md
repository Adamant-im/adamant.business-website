Die ADAMANT JavaScript-API-Bibliothek v1.4.0 stellt zwei neue Methoden zur Delegiertenverwaltung bereit: `delegateNew()` zum Registrieren eines neuen Delegierten und `voteForDelegate()` zum Abgeben von Stimmen für bestehende Delegierte. Diese Version enthält außerdem regelmäßige Aktualisierungen der Abhängigkeiten. Weitere Informationen zur Nutzung finden Sie in der [API-Spezifikation](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification).
