Diese Version führt zwei neue Methoden zum ADAMANT API JS Client hinzu: `delegateNew()` und `voteForDelegate()`. Diese Erweiterungen erweitern die funktionalen Möglichkeiten des Clients im Bereich Delegierte. Zusätzlich wurden die Projektabhängigkeiten auf ihre neuesten Versionen aktualisiert.
