ADAMANT ist eine dezentrale Messaging-Plattform mit integrierter Kryptowährungsübertragungsfunktion, nicht nur ein Messenger. Nachdem Anfang des Jahres die In-Chat-Überweisungen von Ethereum abgeschlossen wurden, unterstützt die Plattform nun auch In-Chat-Überweisungen von Binance Coin (BNB).

Binance Coin ist ein Token, der von Binance, der weltweit größten Kryptowährungsbörse, herausgegeben wurde, und kann zur Inanspruchnahme von Vergünstigungen auf dieser Plattform verwendet werden. Die Unterstützung von BNB dient als Ausgangspunkt für eine breitere Integration von ERC20-Token innerhalb von ADAMANT. Die Wahl von Binance Coin wurde zudem dadurch beeinflusst, dass der ADM-Token derzeit an Börsen gelistet wird und diese Integration dem Team hilft, die Mechanismen von Börsen besser zu verstehen.

![In-Chat-Überweisungen für Binance Coin werden jetzt unterstützt](/images/engineering-notes/medium/3b494b812e62/002-1-lw2mpy0kvqgsvgs68oubog-png.webp)

### So funktionieren In-Chat-BNB-Überweisungen

Um BNB in einem Chat zu senden, müssen Sie zunächst Ihre BNB-Wallet in ADAMANT Messenger aufladen. Dies können Sie tun, indem Sie BNB aus einer anderen Wallet (wie z. B. MyEtherWallet) oder von einer Börse wie Binance überweisen. Ihre Binance-Coin-Wallet-Adresse finden Sie im Reiter „Wallet“, wo Sie sie durch Antippen des Kopiersymbols kopieren können.

![In-Chat-Überweisungen für Binance Coin werden jetzt unterstützt](/images/engineering-notes/medium/3b494b812e62/003-1-cktw266pvgmcz6vcdip4g-png.webp)

Um eine Überweisung durchzuführen, öffnen Sie einen Chat mit dem gewünschten Empfänger, klicken Sie auf das Pluszeichen links neben dem Nachrichteneingabefeld, wählen Sie „Send BNB“, geben Sie den Betrag ein und bestätigen Sie die Transaktion.

Ein wichtiger technischer Hinweis: Da BNB ein ERC20-Token ist, sind zur Durchführung von Überweisungen Ether für die Netzwerkgebühren erforderlich. Sie müssen daher etwas ETH in Ihrer ADAMANT-Wallet halten, um BNB-Überweisungen ausführen zu können.

### Sicherheit

Die Unterstützung von Binance Coin wurde nach denselben Sicherheitsprinzipien implementiert wie die Unterstützung von Ethereum. Nur der Nutzer besitzt seine Wallets innerhalb seines ADAMANT-Kontos – die Plattform verwahrt keine Gelder. Das Team hat eine interne Sicherheitsprüfung abgeschlossen und einen Sicherheitswettbewerb gestartet; als nächster Schritt ist eine unabhängige externe Prüfung geplant.
