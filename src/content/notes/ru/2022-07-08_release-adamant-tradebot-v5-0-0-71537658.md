Этот выпуск сосредоточен на внутренних рефакторингах и обновлениях зависимостей для ADAMANT Tradebot. В рамках работ по обслуживанию добавлен файл `config.default.json`, чтобы обеспечить более понятную базовую конфигурацию для операторов.

Представлен новый параметр маркет-мейкера `mm_orderBookMaxOrderPercent`, позволяющий точнее контролировать участие в стакане заказов.

### Критические изменения
В этом выпуске нет критических изменений.
