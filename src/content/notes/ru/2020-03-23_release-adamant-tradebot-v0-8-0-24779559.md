Этот выпуск ADAMANT Tradebot добавляет поддержку биржи Bit-Z, расширяя перечень платформ, доступных для автоматической торговли. Пользователи теперь могут задавать точность вывода для торговых пар с помощью новых параметров конфигурации coin1Decimals и coin2Decimals. База данных ордеров была обновлена, чтобы включать поле exchange, что улучшает ведение записей на поддерживаемых платформах. Добавлена новая команда /params, отображающая текущие торговые настройки, что упрощает проверку активной конфигурации. Этот выпуск также включает несколько исправлений ошибок.
