Этот выпуск вносит несколько улучшений в обработку транзакций и функциональность обмена сообщениями. Пользователи теперь могут напрямую копировать идентификатор транзакции, поля отправителя и получателя из деталей транзакции, а кнопка обновления в этом представлении получила анимацию для лучшей визуальной обратной связи. Параметры отправки Dash были скорректированы с исправленными порогами максимальной и минимальной суммы, а также устранена проблема, возникающая при отправке небольших сумм Lisk. Сообщения об ошибках, отображаемые при отправке транзакций, были переформатированы для лучшей ясности.

База кода была обновлена с учётом хардфорка London ETH. Компонент snackbar был настроен для улучшенного поведения, а ссылка на сайт в сети Tor была заменена на новую. Внутренние зависимости обновлены до последних совместимых версий.
