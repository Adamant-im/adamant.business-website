Этот выпуск включает несколько улучшений функций в приложении ADAMANT Messenger. Приложение для Android теперь собирается с использованием Capacitor.js, уровень API изменён с `19` на `22`. В модуль чата добавлен тип транзакции, не поддерживаемый ранее, и представлен новый хук `useIsMobile`. В кошелёк добавлен токен KLY, а токен LSK исключён.

В этом выпуске также устранено несколько ошибок. Исправлена навигация по командам в чате, изменён цвет рамки поля ввода сообщения. Применены небольшие исправления интернационализации. Исправлено поведение прокрутки при возврате назад. Исправлен регулярное выражение для имён нод. Рабочие процессы GitHub перенесены на Node v20, а также устранена проблема, при которой `Object.groupBy` не был функцией в старых браузерах.

Также решены вопросы технического долга. Улучшена команда `wallets:generate` в adamant-wallets и обновлены зависимости Electron.
