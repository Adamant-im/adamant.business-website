Открытое программное обеспечение ADAMANT для маркет-мейкинга — это инструмент с самостоятельным хостингом для криптопроектов и бирж, предназначенный для создания объёмов торгов, поддержания спреда и ликвидности, а также формирования динамической книги заказов. Базовая версия бесплатна, расширенные функции доступны в виде платных модулей. Недавно проект выпустил версию 6.1.0, в которой представлен ряд улучшений функциональности и стабильности.

![Open-source Market-making software for cryptocurrency projects updated to v6.1.0](/images/engineering-notes/medium/8b8ed9a261c8/002-0-1vbp44c85yvdelg.webp)

Одним из ключевых обновлений в этом выпуске стало усовершенствование модуля Price Watcher. Теперь он включает механизм проверки актуальности цены токена, что помогает избежать решений по маркет-мейкингу на основе устаревших данных. База кода также была значительно реорганизована для повышения общей стабильности, производительности и удобства сопровождения по мере масштабирования проектов.

Добавлены новые настройки `dev` и `clear_db`. Параметр `dev` упрощает тестирование и разработку, а `clear_db` предоставляет быстрый способ очистки базы данных, что полезно при сбросе окружений. Зависимости были обновлены для обеспечения совместимости с последними библиотеками, что улучшает безопасность и производительность.

Дополнительные улучшения включают незначительные исправления ошибок, новую интеграцию ручных тестов для проверки установки перед развёртыванием и переработанный README с обновлёнными инструкциями по установке и использованию. Релиз и список изменений доступны в репозитории ADAMANT на GitHub.
