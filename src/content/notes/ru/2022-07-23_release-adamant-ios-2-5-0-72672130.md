Этот выпуск ADAMANT iOS представляет API Chatrooms, позволяющий более гибкую интеграцию с функциональностью чата. Получение криптокурсов было оптимизировано для более быстрого получения ценовой информации, что улучшает общую отзывчивость приложения.

Общая производительность была улучшена во всех аспектах, а сборка теперь включает оптимизации для оборудования MacBook M1. В этом выпуске также устранено несколько ошибок.
