Команда ADAMANT провела очную встречу под Владимиром в конце августа, чтобы обсудить будущее платформы мгновенных сообщений и определить приоритеты предстоящих задач дорожной карты по нескольким инженерным направлениям.

На уровне блокчейна приоритеты включают расширение тестового покрытия для нод ADAMANT и создание усовершенствованного API для чатов, пиров, загрузки (bootstrap), адресной книги и других записей KVS, а также разработку предложений AIP для стандартизации. Команда также планирует исследование групповых чатов, закрываемых чатов, хранения файлов на нодах и оценку обновлений Lisk, которые необходимо перенести обратно (backport).

Работа над инфраструктурой сосредоточена на создании сервиса пула форджинга ADM, внедрении мониторинга и уведомлений о сбоях, проведении тестов безопасности, проверке кода на уязвимости и запуске первого конкурса по безопасности.

Для прогрессивного веб-приложения (PWA) команда осуществляет миграцию на Vue 3.0 с улучшением скорости UX. Запланирована интеграция кошельков для Binance Coin и токенов ERC20, последние поддерживаются специальным репозиторием на GitHub для списков токенов. Дополнительные задачи по PWA включают списки транзакций для Ethereum и Binance Coin, обновлённый дизайн списка токенов, вход по паролю по выбору с безопасным хранением на устройстве и выпуск десктопной версии через Electron, ориентированный на Windows Store и Mac App Store.

В iOS-приложение добавляются интеграции Ethereum и Lisk для хранения и перевода этих активов, а также адаптация интерфейса для iPad, поддержка нескольких аккаунтов, приветственный онбординг, переход на новое API загрузки и KVS, поиск внутри сообщений, поддержка тёмной темы и генерация аватаров. На Android разрабатывается первая версия с Material Design 2.0, публичная тестовая сборка запланирована в Google Play.

Организационные задачи включают листинг ADM на дополнительных биржах, подачу заявки на обработку фиата, установление партнёрства с компанией по информационной безопасности, поиск международных конференций, контакт с авторами статей о сравнении мессенджеров и набор региональных представителей. Дизайнерские усилия охватывают руководства по дизайну, оформление одностраничных материалов, мерчандайзинг и обновлённые переводы на другие языки.

Многие из этих задач уже реализованы, остальные находятся в активной разработке.
