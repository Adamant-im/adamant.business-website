Этот выпуск клиента ADAMANT API JS вводит новый параметр `checkNodeTimeout`, который позволяет разработчикам настраивать продолжительность тайм-аута, используемого при проверке доступности ноды. Это улучшение обеспечивает больший контроль над тем, как клиент проверяет подключение к нодам ADAMANT, помогая приложениям более корректно обрабатывать медленные или неотвечающие ноды.
