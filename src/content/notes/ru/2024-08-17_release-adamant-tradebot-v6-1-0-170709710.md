Этот выпуск включает улучшения для системы отслеживания цен (Price Watcher) и более широкие улучшения надёжности. Модули теперь проверяют, является ли цена токена актуальной перед продолжением работы, что помогает предотвратить использование устаревших или неверных данных при принятии торговых решений. Проведена дополнительная рефакторизация для повышения общей стабильности.

Добавлены новые настройки `dev` и `clear_db`, которые дают операторам больший контроль над рабочими процессами разработки и управления базой данных. Зависимости обновлены до последних совместимых версий, исправлено несколько незначительных ошибок. Добавлены ручные тесты для дополнения существующего покрытия тестами, а README обновлён с новой ссылкой на сайт, а также обновлёнными инструкциями по установке и использованию.
