Этот выпуск улучшает соединения через сокеты, добавляя возврат поля `recipientPublicKey`. API транзакций, включая KVS и Чаты, теперь возвращает `block_timestamp` вместе с существующими данными ответа.

Был исправлен механизм миграций, устраняющий проблемы, которые ранее влияли на пути обновления базы данных. Эндпоинт `/states/get/` расширен для поддержки параметров `SenderIds` и `keyIds`, а также теперь принимает POST-запросы, расширяя способы, которыми клиенты могут запрашивать информацию о состоянии.

Документация была обновлена, чтобы отразить эти изменения.

### Критические изменения

В этом выпуске нет критических изменений.
