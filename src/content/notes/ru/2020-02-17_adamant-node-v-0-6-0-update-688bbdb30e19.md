Децентрализованный мессенджер работает на двух компонентах: системе блокчейна и клиентских приложениях. Блокчейн поддерживается нодами сети, которые предоставляют данные приложениям и обрабатывают входящие запросы. ADAMANT выпустила версию программного обеспечения ноды 0.6.0, доступную на странице релизов проекта в GitHub.

Этот выпуск улучшает соединения через сокеты и API транзакций. Соединения через сокеты теперь возвращают `recipientPublicKey`, а конечные точки API транзакций — включая KVS и Chats — теперь включают поле `block_timestamp` в своих ответах. Конечная точка `/states/get/` была расширена, чтобы поддерживать параметры `SenderIds` и `keyIds`, а также POST-запросы. Обновление также включает исправление для миграций и обновлённый комплект документации.

Обновление не является обязательным для всех операторов нод. Однако ноды, которым необходимо взаимодействовать с мессенджер-приложениями, должны быть обновлены до последней версии.
