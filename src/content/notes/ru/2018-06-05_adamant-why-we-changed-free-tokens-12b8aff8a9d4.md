Все аккаунты в блокчейне ADAMANT требуют небольшой баланс токенов для отправки сообщений, с минимальной комиссией в размере 0,001 ADM за сообщение. Эта комиссия поддерживает инфраструктуру сети и позволяет новым пользователям опробовать ADAMANT Messenger. Ранее для этой цели автоматический механизм выдавал 0,49 ADM каждому новому кошельку.

Команда разработчиков ADAMANT уменьшила стартовую сумму токенов, чтобы бороться с ботнетами, массово создающими кошельки для получения бесплатных токенов. Поскольку внедрение CAPTCHA внутри мессенджера нарушило бы анонимность пользователей, команда разместила процессор выдачи токенов на сайте по адресу [adamant.im/free-adm-tokens](https://adamant.im/free-adm-tokens/). Теперь бесплатные токены ADM получают только реальные пользователи.

Сумма бесплатных токенов снизилась с 0,49 ADM до 0,09 ADM, что позволяет пользователям отправить 90 сообщений для тестирования мессенджера. Это изменение также помогает сохранить токены для бонусных программ, максимизируя ценность каждого потраченного токена.
