Этот выпуск хранилища транзакций ETH добавляет дополнительные необязательные индексы, которые можно использовать для оптимизации производительности запросов в определённых сценариях. Эти индексы по умолчанию отключены, что позволяет пользователям включать их выборочно в зависимости от своих рабочих нагрузок.

Файл README также обновлён, чтобы отразить новые параметры индексирования и предоставить более чёткие инструкции по настройке и использованию.
