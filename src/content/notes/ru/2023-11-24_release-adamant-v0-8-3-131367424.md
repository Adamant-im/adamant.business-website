Этот выпуск ADAMANT обновляет список узлов по умолчанию для надежного подключения при обмене сообщениями и работе с блокчейном. Пользователям следует проверить конфигурацию узла после обновления, чтобы убедиться в подключении к обновлённым конечным точкам.

### Критические изменения

Файл `config.json` был удалён из репозитория. После настройки приложения его необходимо создать вручную, выполнив команду `cp config.default.json config.json`. Перед обновлением сохраните текущую конфигурацию, выполнив `cp config.json config.json.backup`, чтобы можно было использовать прежние настройки при повторной настройке.
