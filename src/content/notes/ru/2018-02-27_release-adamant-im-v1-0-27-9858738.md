Этот выпуск представляет начальную реализацию сканирования QR-кодов, которая может по-прежнему содержать ошибки, а также добавляет возможность отправки сообщений по нажатию Enter.

Включены несколько исправлений. Исправлена отправка сообщений через Cmd+Enter. Иногда максимальная сумма для перевода могла быть некорректной — эта проблема устранена. Также исправлена ошибка умножения с плавающей точкой.

Доступны новые переводы на японский, украинский и итальянский языки.
