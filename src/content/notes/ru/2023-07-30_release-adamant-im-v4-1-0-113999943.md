Это обновление ADAMANT Messenger добавляет цитирование и ответы на сообщения в чатах в соответствии с AIP-16, а также жест обновления при свайпе вниз для обновления баланса кошелька и кнопку «Отметить всё как прочитанное» для чатов. Диалог покупки токенов теперь включает биржу StakeCube, а при генерации QR-кода отображается индикатор загрузки. Ввод многострочных сообщений ограничен десятью строками, а во время загрузки баланса кошелька отображается многоточие. Добавлен скрипт для генерации кошельков из репозитория `adamant-wallets`, иконки голосования делегатов были стилизованы заново.

Было исправлено несколько ошибок. Исправлен разделитель заголовка на странице делегатов для светлой темы, таблица узлов теперь использует обычный цвет текста в светлой теме. Обновлены цвета иконок статуса узлов, анимация загрузки чатов теперь центрирована. Скрипт notarize для Electron переименован с `notarize.js` на `notarize.cjs`, ServiceWorker отключён в приложении Electron. Уведомления и список чатов теперь используют опцию рендеринга Markdown. ERC20-токены больше не нарушаются после обновления кошелька, список узлов обновляется корректно. Меню навигации Android больше не перекрывает форму чата, а `isScrolledBottom()` теперь корректно работает с устройствами, использующими субпиксели.

С техническим долгом: `electron-builder` обновлён с версии v23 до v24, версия NodeJS обновлена с v16 до v18. Устранены проблемы безопасности при выполнении `npm install`, а для нотариализации теперь используется `notarytool`.
