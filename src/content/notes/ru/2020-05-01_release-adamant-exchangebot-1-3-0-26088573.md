Этот выпуск добавляет поддержку Bit-Z Token (BZ), расширяя перечень поддерживаемых активов. Для улучшения обработки данных в реальном времени и взаимодействия с API бирж добавлены соединения по WebSocket. Зависимости обновлены до последних версий, а также применены несколько исправлений для устранения проблем, выявленных в предыдущих выпусках.
