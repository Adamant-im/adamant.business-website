В этом выпуске добавлена поддержка аргумента `returnAsset` для эндпоинтов `/api/transactions` и `/api/transactions/get`. При указании этот аргумент заставляет API включать актив транзакции в тело ответа. Это изменение закладывает основу для объединения эндпоинтов API в будущих выпусках, сокращая количество вызовов, необходимых для получения полных данных о транзакции.
