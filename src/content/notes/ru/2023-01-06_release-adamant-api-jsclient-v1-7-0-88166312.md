Этот выпуск представляет метод `api.setStartupCallback()`, позволяющий разработчикам регистрировать обратный вызов, который выполняется во время процесса запуска клиента. Кроме того, конструктор API теперь принимает обратный вызов в качестве третьего параметра, обеспечивая удобный способ передачи логики запуска непосредственно во время создания экземпляра.
