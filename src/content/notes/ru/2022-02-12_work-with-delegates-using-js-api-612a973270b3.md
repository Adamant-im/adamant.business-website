Библиотека JavaScript API ADAMANT версии v1.4.0 представляет два новых метода для управления делегатами: `delegateNew()` для регистрации нового делегата и `voteForDelegate()` для голосования за существующих делегатов. Этот выпуск также включает регулярные обновления зависимостей. Подробности использования см. в [спецификации API](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification).
