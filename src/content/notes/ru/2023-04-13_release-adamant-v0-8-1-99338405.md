В этом выпуске добавлено логирование скрипта миграции базы данных. Узел ADAMANT теперь записывает сообщения до и после выполнения скрипта миграции:

```
[inf] 2023-12-22 04:00:25 | Found 5 pending migrations. Start executing, this may take a while…
[inf] 2023-12-22 04:17:13 | Migrations have been successfully completed.
```

Значение по умолчанию для свойства `cors.origin` изменено с отражения источника запроса (`true`) на `*`. Подробнее о настройке CORS для узла ADAMANT см. в документации.

```json5
{
  "cors": {
    "origin": "*" // default
  }
}
```

Файл `config.json` добавлен в `.gitignore`. Теперь пользователи должны создать файл `config.json` самостоятельно или скопировать `config.default.json`, чтобы запустить узел ADAMANT.

```
cp config.default.json config.json
```

### Критические изменения

Значение заголовка CORS по умолчанию для `Access-Control-Allow-Origin` изменено с отражения источника запроса на `*`. Пользователям, полагающимся на прежнее поведение с отражённым источником, следует соответствующим образом обновить свой файл `config.json`. Кроме того, файл `config.json` теперь игнорируется git, поэтому пользователи должны вручную создать или скопировать этот файл перед запуском узла.
