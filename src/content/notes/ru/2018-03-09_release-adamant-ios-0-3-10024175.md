Этот выпуск добавляет поддержку фоновой загрузки в iOS для проверки новых сообщений в подключённых аккаунтах, а также локальные уведомления и мост NotificationService для взаимодействия с фреймворком UserNotifications. Теперь контроллер LoginViewController отображает кнопку входа, когда начинается редактирование пароля.

Выпуск также включает множество исправлений ошибок, очистку кода и рефакторинг по всему кодовому базе.

### Критические изменения

Сервис AccountService теперь удаляет сохранённый аккаунт, когда пользователь входит в другой аккаунт, что меняет прежнее поведение, при котором учётные данные сохранялись при переключении между аккаунтами.
