В этом выпуске обновлены зависимости проекта и переписана панель мониторинга с использованием Svelte. Полностью переработана и оптимизирована кодовая база, исправлены несколько ошибок.

### Критические изменения

Поддержка версий Node.js ниже 18.12.1, текущего выпуска с длительной поддержкой (LTS), прекращена. Теперь пул использует lowdb в качестве базы данных. Подробности см. в разделе миграции файла README.
