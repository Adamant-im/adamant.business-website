ADAMANT InfoServices обновлён до версии v3.5.0, предоставляя объединённое API курсов фиатных и криптовалют. Сервис агрегирует данные из нескольких внешних источников, включая MOEX, Currency-Api, ExchangeRates.host, CoinMarketCap, CryptoCompare и CoinGecko, затем сопоставляет результаты, чтобы выявлять расхождения и повышать надёжность.

Поскольку сервис работает на вашем собственном сервере, исторические данные о курсах остаются под вашим контролем и при этом остаются бесплатными для любого использования. Проект является открытым и размещён в организации ADAMANT на GitHub.

Документация по API доступна в [вики-странице InfoServices API](https://github.com/Adamant-im/adamant-currencyinfo-services/wiki/InfoServices-API-documentation).
