Этот выпуск представляет несколько новых возможностей API и улучшений инфраструктуры для программного обеспечения узла ADAMANT.

Добавлены новые конечные точки `/api/node`, расширяющие информацию, доступную клиентам, взаимодействующим с сетью. Также включена новая конечная точка `/api/chatrooms`. Кроме того, представлен сервер WebSocket, позволяющий лёгким клиентам подписываться и получать уведомления о новых транзакциях.

Поля `version` и `minVersion` были перемещены из файла конфигурации.

### Критические изменения

Поля `version` и `minVersion` больше не считываются из файла конфигурации. Развертывания, зависящие от установки этих значений через конфигурацию, должны соответствующим образом обновить свою настройку.
