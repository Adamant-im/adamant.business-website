Библиотека ADAMANT JavaScript API теперь поддерживает криптографические ключи Lisk наряду с комплексной реорганизацией кодовой базы. Выпуск v1.5.0 заменяет предыдущий HTTP-клиент на Axios, улучшая обработку запросов и удобство сопровождения библиотеки. Руководство по участию и стиль кода также были обновлены для упрощения совместной работы.

Полная документация по API доступна в [ADAMANT API wiki](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification), а подробности выпуска — на [странице выпуска v1.5.0](https://github.com/Adamant-im/adamant-api-jsclient/releases/tag/v1.5.0).
