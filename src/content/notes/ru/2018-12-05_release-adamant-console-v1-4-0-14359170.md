ADAMANT Console v1.4.0 представляет новую команду `rpc server`, которая позволяет операторам запускать RPC-сервер непосредственно из консоли. Этот выпуск также добавляет функциональность JSON-RPC сервера, позволяя внешним клиентам взаимодействовать с консолью посредством структурированных запросов JSON-RPC.
