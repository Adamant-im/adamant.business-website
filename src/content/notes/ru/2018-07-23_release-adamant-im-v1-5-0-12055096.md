Этот выпуск представляет новое окно списка транзакций наряду с несколькими изменениями стиля и улучшениями внутренней рефакторизации по всему коду.

Несколько ошибок было устранено в этой версии. Исправлена ошибка, возникающая при отправке ETH с переопределённым адресом контакта. Открытие чата без сообщений из списка транзакций теперь работает корректно. Исправлено поведение прокрутки к нижней части при открытии предыдущего чата. Кроме того, блок бесплатных токенов ADM на вкладке Кошелёк теперь скрыт, если баланс больше нуля.
