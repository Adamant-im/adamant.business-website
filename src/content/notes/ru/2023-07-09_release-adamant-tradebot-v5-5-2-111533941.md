Этот выпуск устраняет две проблемы, специфичные для бирж, и включает плановое техническое обслуживание. Исправлена отмена ордеров для StakeCube, что решает проблему, из-за которой сделки могли не отменяться корректно на этой платформе. Также исправлена функция getRates для Azbit, что обеспечивает надёжное получение курсов с этой биржи.

Кроме того, зависимости обновлены до последних совместимых версий, а файл readme проекта обновлён для отражения актуальной информации.
