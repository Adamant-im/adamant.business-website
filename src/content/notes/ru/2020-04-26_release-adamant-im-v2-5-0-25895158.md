Этот выпуск ADAMANT Messenger расширяет список доступных для подключения нод и добавляет проверку протокола ноды. Исправлены статусы нод и соединения сокетов на HTTP-хостах, а также обновлены базовые зависимости.

Интеграции с биржами скорректированы: добавлена ссылка на CoinDeal, удалена ссылка на IDCM, обновлены ссылки на Resfinex и Bit-Z. Решено несколько проблем, связанных с чатами, включая стили для встроенного `code` в чатах, статические названия чатов, проверку вставленных адресов при запуске нового чата, локализацию сообщений «no public key» и «no hash», а также генерацию QR-кодов для ADM-адресов в Windows и других приложениях.
