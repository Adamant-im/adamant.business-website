Этот выпуск полностью переписывает и обновляет способ хранения данных на устройстве пользователя. Пользователи, которые не хотят автоматического выхода из системы, теперь должны задать дополнительный пароль, который используется для шифрования локальных данных. Все данные, хранящиеся на устройстве, зашифрованы с помощью локального шифрования, а для хеширования паролей применяется Argon2. Благодаря этому хранение данных на персональных устройствах для быстрого доступа стало безопасным.

Защищённые данные также перемещены из LocalStorage в IndexedDb, что обеспечивает больше места для сообщений.
