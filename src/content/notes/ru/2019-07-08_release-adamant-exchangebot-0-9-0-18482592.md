Этот выпуск ADAMANT Exchangebot добавляет функциональность обмена между токенами ADM и ETH. Обновление также включает команды помощи для удобства навигации по доступным операциям, а также улучшения в обработке сообщений для более надёжной коммуникации.

В планах проекта — внедрение новой системы бонусов для пользователей и статистики обменов в будущих выпусках.
