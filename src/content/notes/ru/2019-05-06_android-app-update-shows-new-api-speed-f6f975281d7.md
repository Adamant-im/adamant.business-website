### Быстрое API в последнем выпуске для Android

После более ранних анонсов об улучшении скорости API для ноды ADAMANT команда разработки для Android выпустила обновление, включающее новое API чатов.

Сравнение старого и обновлённого API показывает, что операции в чатах и процедура входа выполняются значительно быстрее, чем раньше. Дальнейшие оптимизации запланированы.

Последняя версия APK доступна на [странице релизов для Android на GitHub](https://github.com/Adamant-im/adamant-android/releases/). Пользователям рекомендуется убедиться, что установлена последняя версия; если приложение не обновляется автоматически, удалите предыдущую версию и установите новый APK вручную.

Отзывы и отчёты об ошибках можно отправлять через репозиторий на GitHub. Команда также готовится к выпуску в Google Play.
