ADAMANT Node версии 0.5.1 теперь доступен на GitHub, завершая работу над API чатов, впервые анонсированную в январе. Главной особенностью является конечная точка `/api/chatrooms`, которая позволяет получать контакты и чаты значительно быстрее, чем предыдущее API. Разработчики должны обратиться к спецификации [AIP-14](https://aips.adamant.im/AIPS/aip-14) для деталей реализации.

Это обновление не является обязательным для всех владельцев нод. Однако ноды, которым необходимо подключаться к мессенджер-приложениям, должны обновиться до последней версии, чтобы воспользоваться новым API. Команда ADAMANT для Android уже продемонстрировала измеримое ускорение при использовании новой конечной точки, и аналогичные улучшения ожидаются во всех остальных приложениях ADAMANT по мере их внедрения.
