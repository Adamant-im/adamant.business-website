Этот выпуск обновляет зависимости проекта до Vue3-cli, сопровождаемый значительным рефакторингом по всему коду. Теперь в приложении поддерживается работа с произвольными токенами ERC20, а также добавлена функциональность кошелька и перевода BNB (Binance Coin). Сканер QR-кодов Instascan теперь загружается отложенно, что ускоряет первоначальную загрузку для пользователей.

В этот выпуск включено несколько исправлений. Префикс "U" теперь корректно добавляется к числовым адресам в окне создания нового чата. Устранены ошибки в списке транзакций, и отображаемые имена больше не перезаписываются при сканировании QR-кодов. Комиссии за переводы не-ADM теперь рассчитываются корректно, а также исправлено создание нового чата из списка транзакций. Обновлён также файл README.

### Критические изменения

Зависимости и CLI-команды теперь управляются с помощью yarn вместо npm. Разработчики, участвующие в проекте, должны использовать yarn для управления пакетами в дальнейшем, поскольку рабочие процессы на основе npm могут больше не соответствовать обновлённой инструментальной цепочке.
