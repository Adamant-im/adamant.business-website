ADAMANT Messenger v2.3.0 теперь доступен с версиями для Web, Tor, Windows, macOS и GNU/Linux. В релизе добавлена поддержка Bitcoin, а также несколько улучшений в обработке статусов транзакций и исправление ошибок.

Обработка транзакций была улучшена в нескольких цепочках. Количество подтверждений на экране деталей транзакции теперь обновляется динамически, а новый статус Registered для транзакций позволяет более эффективно обрабатывать обновления статусов. Исправлены ошибки в статусах транзакций Ethereum, которые ранее отображались некорректно. Кроме того, исправлена обработка ссылок на чаты msg.adamant.im.

Текущая кодовая база размещена в ветке master [репозитория ADAMANT Messenger на GitHub](https://github.com/Adamant-im/adamant-im). Разработчики, заинтересованные в участии в открытом проекте ADAMANT, могут найти там последний исходный код.
