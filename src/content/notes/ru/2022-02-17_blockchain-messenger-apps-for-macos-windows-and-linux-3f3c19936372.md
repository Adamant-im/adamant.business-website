ADAMANT Messenger v3.1.0 представляет автономные десктопные приложения для macOS, Windows и Linux. Этот выпуск уменьшает количество сетевых запросов при использовании сокет-соединений и обновляет зависимости проекта.

Приложение можно собрать из исходных кодов через [репозиторий ADAMANT](https://github.com/Adamant-im/adamant-im). Подробности о выпуске доступны на [странице релиза v3.1.0](https://github.com/Adamant-im/adamant-im/releases/tag/v3.1.0).
