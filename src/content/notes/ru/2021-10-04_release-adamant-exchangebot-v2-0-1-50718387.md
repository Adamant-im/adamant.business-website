Этот выпуск добавляет поддержку Dash с функцией InstantSend, а также обеспечивает совместимость с хардфорком London для Ethereum. Пользователям больше не нужно указывать валюту в комментариях.

База кода была полностью рефакторингована, включая линтинг и различные оптимизации. Приложение теперь использует ADAMANT API 1.1.0, а все зависимости обновлены.
