Этот выпуск вводит универсальную сборку для macOS и переводит проект на новую конфигурацию ESLint. В репозиторий добавлен файл с заметками о выпуске. Улучшена система проверки работоспособности: теперь перед попыткой HTTP-запроса отображается статус «Обновление…», начальная и последующие проверки работоспособности используют усовершенствованный алгоритм запросов, а система проверяет `alt_ip`, если веб-домен недоступен. Исправлено восстановление фоновой проверки после пробуждения системы, а также устранена некорректная передача статуса узла. Обновлен экран кошельков, скрипт для adamant-wallets и создан тестовый экран для разработчиков, предназначенный для этой инструментальной системы.

Сообщения теперь могут объединяться в группы с оформлением в виде пузырьков. По всему приложению применены несколько стилевых обновлений, включая стили поля ввода сообщений, компоновку формы экспорта ключей и улучшения общей визуальной согласованности. В чате «Добро пожаловать в ADAMANT» теперь присутствует отключённое поле ввода. Добавлены проверки Playwright только для разработчиков в рамках более масштабных усилий по унификации стилей и модернизации пользовательского интерфейса и пользовательского опыта.

Включено множество исправлений ошибок. Даты в чатах теперь обновляются корректно. Ложные индикаторы синхронизации больше не появляются при открытии экрана узлов. Исправлены некорректные суммы транзакций для BTC, DOGE и DASH при отправке на собственный адрес. Устранена ошибка разметки, возникавшая при ответе на сообщение, содержащее длинную ссылку. Устранена ошибка обращения к null, связанная с `maintainScrollPosition`. Статические чаты больше не становятся доступными только для чтения после повторного входа. Комиссии за транзакции для токенов ERC20 теперь одинаковы в PWA и на iOS. Исправлены навигация по транзакциям и регрессии локализованного интерфейса на экранах PWA. Устаревшие сохранённые символы кошельков теперь нормализуются, чтобы предотвратить сбой на маршруте `/options`. Исправлено имя выпущенного APK, а также устранены проблемы в исходном коде, выявленные ESLint 9. Устранена некорректная активность коммитов в репозитории.

Обновлены общие зависимости и инструменты, устранены предупреждения при сборке. Версия Node.js обновлена с 20 до 22. Внедрены Release Drafter для всей организации и повторно используемый рабочий процесс. Добавлен рабочий процесс GitHub Actions и git-хук Husky на postinstall для ESLint. Добавлена документация с рекомендациями по использованию AI-агента для PWA, обновлены README и метаданные выпуска.

### Критические изменения

Поддержка Klayr (KLY) полностью удалена из кошелька. Пользователи, владеющие активами KLY, должны учитывать, что эта криптовалюта больше не поддерживается, начиная с версии 4.11.0.
