Этот выпуск ADAMANT для iOS устраняет сбой, возникающий при использовании Face ID на iPhone X. Новые переводы теперь автоматически помечаются как непрочитанные, чтобы пользователи могли легче отслеживать последнюю активность. Кроме того, были исправлены фоновая выборка и push-уведомления для обеспечения надёжной доставки сообщений. Теперь приложение также поддерживает уведомления внутри приложения как для входящих сообщений, так и для переводов.
