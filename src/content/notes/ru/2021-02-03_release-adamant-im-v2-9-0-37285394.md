В этом выпуске представлен обновлённый логотип и новые иконки по всему приложению ADAMANT Messenger. В версии приложения для Tor добавлены ссылки в виде луковиц (onion) для покупки токенов и получения их бесплатно. Обновлены зависимости, а список ADM-нод обновлён до текущих участников сети.

Статус подтверждения переводов ADM через сокет теперь отображается корректно. Добавлен новый кошелёк Adoption, а также введён статический чат для пожертвований. Тексты в статических чатах пересмотрены, а положение времени в чатах скорректировано. Пользователи теперь могут выбирать отображение дат в полном формате. Таймауты всплывающих уведомлений (snackbar) улучшены для повышения удобства использования.

Содержимое сообщений теперь очищается с помощью DOMPurify для повышения безопасности. Применено множество исправлений в списках транзакций и в деталях транзакций. Ссылка на обозреватель Dash обновлена, чтобы указывать на корректный сервис.
