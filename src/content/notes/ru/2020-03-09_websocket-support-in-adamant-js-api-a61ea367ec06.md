ADAMANT — проект с открытым исходным кодом с публичным API блокчейна, консольным инструментом и клиентом JavaScript API. Клиент JavaScript API обновлён до версии 0.5.0, и ключевым нововведением стала поддержка WebSocket.

Соединения WebSocket позволяют приложениям мгновенно получать уведомления о новых транзакциях и сообщениях. Это снижает задержку по сравнению с опросом REST API и позволяет ботам и сервисам быстрее реагировать на запросы пользователей.

Релиз доступен на GitHub, а документация по JavaScript API поддерживается в вики проекта.
