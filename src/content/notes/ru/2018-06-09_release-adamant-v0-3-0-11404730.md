Этот выпуск добавляет поддержку сообщений с расширенным содержимым (Rich Content Messages), определённых в [AIP 5](https://aips.adamant.im/AIPS/aip-5), что позволяет использовать более богатое форматирование сообщений в платформе обмена сообщениями ADAMANT. Также внедрена поддержка Signal-сообщений в соответствии с [AIP 6](https://aips.adamant.im/AIPS/aip-6). Кроме того, в этом выпуске исправлены фильтры в API транзакций состояния (KVS).
