Этот выпуск представляет полную рефакторизацию ADAMANT Console, теперь построенного на основе adamant-api. Консоль получила возможность расшифровки сообщений и встроенную справку по доступным командам. Добавлен интерактивный режим для упрощения использования.

Теперь пакет можно подключать как библиотеку JavaScript, что расширяет возможности интеграции за пределами командной строки. Обновлены зависимости, включены исправления нескольких ошибок.

### Критические изменения

Полная рефакторизация означает, что внутренняя структура и API значительно изменились. Существующие скрипты или интеграции, основанные на предыдущей архитектуре, потребуют обновления для работы с новой основой на базе adamant-api.
