Это первый стабильный выпуск JavaScript-клиента API ADAMANT. Библиотека обеспечивает высоконадежный доступ к блокчейну ADAMANT, включая GET-запросы для чтения данных из цепи и методы отправки токенов и сообщений.

Клиент обрабатывает криптографические операции, такие как кэширование публичных ключей, шифрование и расшифровка сообщений, формирование и подписание транзакций, а также работа с парами ключей ADM. Он также поддерживает генерацию криптоадреса и ключей Ethereum, привязанных к аккаунту ADM, и работу со временем эпохи ADAMANT.

Дополнительные возможности включают поддержку соединений WebSocket и встроенное логирование предупреждений, ошибок и информационных сообщений.
