После листинга на централизованной бирже (CEX) эмитенты токенов часто сталкиваются со слабым стаканом, широким спредом и графиком, который наказывает мелкие сделки. Традиционное решение — передать токены и API-ключи третьей стороне, управляющей маркет-мейкингом на прозрачной инфраструктуре. Программное обеспечение для маркет-мейкинга ADAMANT v9.0.0 предлагает альтернативу: вы можете развернуть стек маркет-мейкинга на собственном сервере, как обычное приложение — без необходимости выполнять git clone и передачи контроля над активами.

### Модель: автономный, под полным контролем

Традиционный маркет-мейкинг обычно предполагает передачу токенов третьей стороне, обмен API-ключами с «чёрным ящиком» и надежду на то, что стакан будет выглядеть здоровым — и что вы сможете вернуть свои активы. ADAMANT меняет эту модель: вы запускаете стек маркет-мейкинга на собственном сервере, используя собственный аккаунт на бирже и свои ключи.

### Что делает v9 для вашего графика

Бесплатная версия с открытым исходным кодом (OSS) решает ключевые проблемы, возникающие сразу после листинга. Она формирует стакан, заполняя пробелы, чтобы он не выглядел пустым, поддерживает узкий спред bid/ask, чтобы избежать негативного первого впечатления, и обеспечивает глубину, чтобы мелкие сделки не вызывали резких колебаний цены. Программа отслеживает заданные вами ценовые диапазоны и применяет политики объёмов в режимах spread, order book, depth и optimal. Мониторинг прозрачен — балансы, ордера и статистика доступны через командное управление через ADAMANT Messenger, при этом панель администратора по умолчанию не открывается в публичный доступ.

Поддерживаемые коннекторы CEX в сборке OSS включают Azbit, P2PB2B, StakeCube, Coinstore, FameEX и NonKYC. Для других бирж доступны премиум- и кастомные коннекторы.

### Начало работы (путь через npm)

Вам понадобится сервер под Linux или Mac (или любая машина с поддержкой npm), Node.js 22+, MongoDB и API-ключ CEX от вашего собственного аккаунта.

![Ваш токен заслуживает настоящего рынка — а не передачи в аренду чёрному ящику](/images/engineering-notes/medium/42fcfb71beb3/003-1-duiackkmcteg8he9ccmqdq-png.webp)

Официальный реестр npm

Установите пакет глобально и создайте рабочую директорию:

```bash
npm install -g adamant-tradebot
mkdir my-mm && cd my-mm
```

Команда CLI — `mm`. Настройте бота с помощью интерактивного мастера, затем выполните проверку работоспособности:

```bash
mm init    # interactive wizard - exchange, pair, API keys
mm doctor  # checks config, MongoDB, exchange API
```

Токены не покидают вашу биржу для финансирования маркет-мейкера. Вы просто подключаете свои учётные данные API к боту на собственной машине. Запустите и проверьте статус:

```bash
mm on
mm status
```

Отправьте `/balances` вашему боту с вашего ADM-аккаунта администратора — и вы в строю. Остановите в любой момент командой `mm off`, а логи просмотрите с помощью `mm logs`.

### Альтернатива через Docker

Вы также можете загрузить опубликованный образ из GitHub Container Registry:

```bash
docker pull ghcr.io/adamant-im/adamant-tradebot:9.0.0
```

MongoDB запускается в Compose вместе с приложением, а конфигурация и логи хранятся в локальных томах, которые вы контролируете.

### Почему v9 — это веха

До v9 запуск требовал клонирования репозитория и самостоятельной настройки зависимостей — приемлемо для разработчиков, но создаёт барьер для основателей, которым просто нужен более здоровый стакан. v9.0.0 предлагает полноценную дистрибуцию через npm и GHCR, CLI `mm` с командами `init`, `on`, `off`, `doctor`, `status`, `logs` и `config`, публикацию через CI в npm и Docker при каждом релизе на GitHub, рефакторинг ядра, включающего трейдера, построитель стакана, поставщика ликвидности и ценового монитора, а также тестовые наборы на Jest и документацию.

ADAMANT — это open-source криптопроект с десятилетней историей публичной разработки.

### Кому подойдёт это решение

Это программное обеспечение предназначено для эмитентов токенов, у которых после листинга на CEX слабый или пустой стакан, команд, не готовых платить шестизначные суммы плюс передавать токены, основателей, не желающих передавать ключи непрозрачным третьим сторонам, и проектов, которым важна прозрачность — возможность читать код, просматривать логи и контролировать выключатель. Премиум-модули доступны для продвинутых стратегий, веб-интерфейса, дополнительных бирж или помощи в настройке. Бесплатная версия с открытым исходным кодом изначально полезна сама по себе.
