Этот выпуск включает несколько улучшений API и исправлений для программного обеспечения узла ADAMANT. Эндпоинт `/api/delegates/get` теперь принимает параметр `address` в дополнение к ранее поддерживаемым параметрам `username` и `publicKey`. Использование простое:

```bash
/api/delegates/get?address=U1234567890
```

Поддержка WebSocket была улучшена, чтобы разрешить подписку на несколько типов транзакций вместо только одного. Для транзакций сообщений новое событие `assetChatTypes` позволяет клиентам подписываться на определённые типы сообщений.

Эндпоинт `/api/node/status` теперь возвращает дополнительные свойства, объединяя данные из `/api/loader/status` и `/api/loader/status/sync`, что уменьшает количество запросов, необходимых для получения полной информации о состоянии блокчейна:

```json5
{
  // ...
  "loader": {
    "loaded": true,
    "now": 45345521,
    "syncing": false,
    "consensus": 100,
    "blocks": 0,
    "blocksCount": 0
  }
}
```

Было исправлено несколько ошибок. Устранена проблема подключения к пиром, чьи версии содержат метаданные предварительного выпуска или сборки (например, `+alpha` или `-dev.0`), что обеспечивает полное соответствие SemVer 2.0. Исправлено удаление пиров после однократных ошибок подключения, а также устранена ошибка проверки конфигурационного файла.

Эндпоинт `/delegates/get` оптимизирован для поиска по одной транзакции. Обновлены зависимости, внесены незначительные улучшения в логирование и добавлено руководство для новых участников по тестированию.
