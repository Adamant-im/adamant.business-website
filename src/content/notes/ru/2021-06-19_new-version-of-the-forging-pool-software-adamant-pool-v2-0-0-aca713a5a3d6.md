ADAMANT Pool v2.0.0 представляет полную переработку кода с акцентом на надежность и производительность. Теперь пул использует ADAMANT JS API v1.0.0, что гарантирует правильное и своевременное получение вознаграждений голосующими. Требования к ресурсам значительно снижены; пул теперь может работать на виртуальной машине с 1 виртуальным процессором и 512 МБ ОЗУ. Этой эффективности удалось достичь за счёт удаления ненужных зависимостей, обновления оставшихся и замены библиотеки `request` на `axios`.

Было внесено несколько изменений в конфигурацию. Порт по умолчанию теперь 36667 вместо 36668. В конфигурационный файл добавлены новая опция `log_level` и параметр `donatewallet` для передачи процента вознаграждений фонду ADAMANT. Периоды выплат теперь можно планировать по дням недели с помощью параметра `payoutperiod`. Кроме того, комиссия за транзакцию голосования теперь взимается с голосующих, что приводит к уменьшению вознаграждения на 0.5 ADM. Операторам следует скорректировать параметр `minpayout`, чтобы выплаты оставались разумными с учётом комиссии. Продуктивность делегата теперь учитывается при распределении вознаграждений.

Среди прочих обновлений — рефакторинг кода, удаление режима только для чтения и добавление функций Markdown для уведомлений. Обновлён также дизайн панели информации и интерфейса панели управления пулом.

![New version of the forging pool software ADAMANT Pool v2.0.0](/images/engineering-notes/medium/aca713a5a3d6/002-0-vrmyfc6ou3ue0xnb.webp)

При обновлении существующего пула рекомендуется удалить старую установку и выполнить новую чистую установку. Однако файл `/db/transactions`, содержащий историю транзакций, следует сохранить.
