В этом выпуске в эндпоинт `/api/chatrooms/U*` добавлено поле `confirmations`, которое предоставляет вызывающим сторонам дополнительный контекст о статусе подтверждения соответствующих транзакций. Это улучшение позволяет клиентам, использующим API чат-комнат, легче определять, насколько глубоко транзакция подтверждена в блокчейне, без необходимости отдельного поиска.
