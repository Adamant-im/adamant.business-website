Этот выпуск ADAMANT API JS Client добавляет поддержку блокчейна Lisk наряду с существующей функциональностью ADAMANT. Слой HTTP был перенесён на клиент на основе Axios, что улучшает обработку запросов и удобство сопровождения кода в поддерживаемых сетях.

Эти изменения сопровождаются масштабной рефакторингом кода, направленной на улучшение внутренней структуры и уменьшение дублирования. Проект также получил обновления руководств по стилю кода и документации для участников, чтобы упростить старт новым разработчикам.
