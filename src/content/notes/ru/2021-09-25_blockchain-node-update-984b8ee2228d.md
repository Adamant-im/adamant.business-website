ADAMANT Node v0.6.5 теперь доступен. Этот выпуск сосредоточен на качестве кода и инструментах, а не на изменениях протокола, поэтому обновление является необязательным и не влияет на консенсус сети.

Настройки широковещательной рассылки по умолчанию были изменены, а скрипт установки bash обновлён. Зависимости для разработки были обновлены, код был проверен с помощью линтера, а набор тестов исправлен.

Среди заметных дополнений — параметр командной строки `-genesis` и выделенный `portWS` 36665 для тестовой сети. Это упрощает запуск конфигураций genesis и тестовой сети без ручных переопределений.

Инструкции по установке для Ubuntu, macOS и Windows доступны в [примечаниях к выпуску ADAMANT Node v0.6.5](https://github.com/Adamant-im/adamant/releases/tag/v0.6.5).
