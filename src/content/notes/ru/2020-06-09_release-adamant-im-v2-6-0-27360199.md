В этом обновлении исправлены push-уведомления и изменено поведение при выходе: пользователи остаются в системе при закрытии вкладки, а не выходят автоматически. Тёмный режим теперь используется по умолчанию.

Было внесено несколько улучшений в обработку ссылок: ссылки на документацию по паролям пользователей теперь открываются в новом окне с атрибутом `noopener`, аналогично ссылки в сообщениях чата открываются в новых окнах с `noopener` для повышения безопасности.

Детали транзакций теперь отображают комментарии и показывают собственный адрес пользователя с пометкой «Me». В списке транзакций также отображаются комментарии, а имена рядом с адресами ADM показываются и для транзакций с другими криптовалютами. Кроме того, пользователи могут перейти напрямую в чат из записи в списке транзакций для других криптовалют.
