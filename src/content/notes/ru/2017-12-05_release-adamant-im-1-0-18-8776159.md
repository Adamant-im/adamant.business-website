Этот выпуск ADAMANT Messenger добавляет поддержку обработки одинарного переноса строки как реального переноса в сообщениях, что соответствует стандарту GitHub Flavored Markdown. Ранее одинарные переносы строк не отображались как переносы; это изменение приводит форматирование сообщений в соответствие с общепринятыми правилами markdown.

Было внесено исправление в счётчик непрочитанных сообщений, устраняющее проблему, из-за которой количество могло отображаться некорректно. Кроме того, временным исправлением решены проблемы совместимости с iOS 11, что улучшает стабильность работы для пользователей на этой платформе. Также выполнены небольшие корректировки логики хранения состояния для дальнейшего повышения надёжности.

### Критические изменения

В этом выпуске нет критических изменений.
