Этот выпуск посвящён улучшению обслуживания и стабильности ADAMANT Tradebot. Зависимости обновлены до последних версий, выполнена внутренняя рефакторизация для повышения качества кода. В этом выпуске также включены несколько исправлений ошибок.

Коннектор API P2PB2B был переписан для повышения надёжности и удобства сопровождения.
