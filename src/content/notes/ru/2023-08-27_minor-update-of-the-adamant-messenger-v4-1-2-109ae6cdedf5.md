ADAMANT Messenger v4.1.2 — это небольшой выпуск для технического обслуживания, ориентированный на исправление ошибок и общие улучшения стабильности. Обновление не вводит новых функций, но устраняет проблемы, влияющие на общую надежность мессенджера на всех поддерживаемых платформах.

Приложение представляет собой Progressive Web Application, доступное для Web, Tor, Android, Windows, Linux и macOS. Исходный код опубликован на GitHub в репозитории проекта ADAMANT, что позволяет пользователям собирать приложение из исходников для независимой проверки его свойств конфиденциальности и безопасности.

Заметки о выпуске и инструкции по сборке доступны в [теге релиза v4.1.2](https://github.com/Adamant-im/adamant-im/releases/tag/v4.1.2) на GitHub.
