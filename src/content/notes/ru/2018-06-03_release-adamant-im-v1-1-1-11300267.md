Этот выпуск включает изменения стиля, улучшающие удобство использования новых полей на мобильных устройствах. Теперь поддерживается вход с помощью парольной фразы, сохранённой в виде QR-кода, благодаря @chovy. Рабочий процесс разработки улучшен: команда `npm run dev` теперь поддерживает горячую перезагрузку.

Включены различные исправления и доработки. Сохранение данных в KVS теперь запускается только при достаточном количестве токенов ADM, что устраняет назойливые всплывающие окна. Исправлена ошибка, добавлены новые языковые константы и выполнены общие обновления локализации.

Поддержка украинского, филиппинского, шведского, хинди, турецкого, вьетнамского, румынского и арабского языков временно отключена до обновления соответствующих языковых файлов.
