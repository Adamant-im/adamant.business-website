Этот выпуск приложения ADAMANT iOS включает несколько улучшений обработки транзакций и работы чата.

Мессенджер теперь выполняет проверку состояния монет, выбирая только рабочие узлы для подключения. Пользователи могут выбирать между подключением к случайным узлам или предпочтением самого быстрого узла. Список транзакций также улучшен: приложение сохраняет транзакции локально и больше не перезагружает весь список при каждом открытии.

На экране чата короткое нажатие на имя получателя открывает QR-код его адреса, а длительное нажатие позволяет переименовать получателя.

Также включены дополнительные исправления для улучшения общего пользовательского опыта.
