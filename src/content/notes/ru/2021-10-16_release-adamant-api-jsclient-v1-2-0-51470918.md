Этот выпуск ADAMANT API JS Client добавляет поддержку генерации BTC-ключей. Обновление расширяет криптографические возможности клиента, включая создание пары ключей Bitcoin наряду с существующей функциональностью ключей ADAMANT.

В этой версии не внесено никаких прерывающих изменений.
