Библиотека ADAMANT JavaScript API v1.7.0 добавляет поддержку обратных вызовов для логики после инициализации. В этом выпуске представлены два изменения: новый метод `api.setStartupCallback()` и необязательный обратный вызов, передаваемый как третий параметр в конструктор `api`. Оба механизма позволяют выполнять пользовательский код после завершения инициализации библиотеки, что полезно для задач настройки, зависящих от готового экземпляра API.

Полная документация по API доступна в [вики ADAMANT API client](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification). Подробности выпуска — в [заметках к релизу v1.7.0](https://github.com/Adamant-im/adamant-api-jsclient/releases/tag/v1.7.0).
