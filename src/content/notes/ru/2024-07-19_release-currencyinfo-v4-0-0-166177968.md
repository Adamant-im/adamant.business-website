Этот выпуск представляет новый эндпоинт `GET /status` для мониторинга состояния сервиса. Эндпоинты `/get` и `/getHistory` теперь включают поля `version` и `last_update` в своих ответах. Уведомления о значительных изменениях можно отправлять в ADAMANT Messenger и Discord. Для упрощения развертывания добавлен Dockerfile.

### Критические изменения

Реализована новая система обнаружения значительных изменений для обработки различий курсов из нескольких источников. Кроме того, ExchangeRateHost теперь требует API-ключ для работы, что может потребовать обновления конфигурации существующих развертываний.
