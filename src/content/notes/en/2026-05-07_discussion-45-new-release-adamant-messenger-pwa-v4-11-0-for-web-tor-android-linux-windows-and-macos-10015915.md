ADAMANT has released version 4.11.0 of its Messenger PWA, now available across Web, Tor, Android, Linux, Windows, and macOS. The release focuses on reliability, polish, and maturity improvements rather than introducing major new features.

The latest build is tagged v4.11.1 on GitHub, indicating a quick follow-up patch after the initial 4.11.0 publication. Users running the PWA should update to the newest tag to pick up any post-release fixes.

ADAMANT Messenger is an open-source, blockchain-based messaging application. The PWA (Progressive Web App) build allows users to run the messenger in a browser or install it as a native-like application on supported desktop and mobile platforms, including Tor for users who need onion-routing access.
