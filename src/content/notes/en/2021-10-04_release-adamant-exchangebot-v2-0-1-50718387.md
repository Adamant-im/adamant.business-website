This release introduces Dash support with InstantSend functionality and compatibility with Ethereum's London hardfork. Users are no longer required to specify the currency in comments.

The codebase has undergone a total refactoring, including linting and various optimizations. The application now uses ADAMANT API 1.1.0, and all dependencies have been updated.
