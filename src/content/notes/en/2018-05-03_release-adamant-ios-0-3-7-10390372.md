This release of ADAMANT for iOS resolves a crash that occurred when using Face ID on iPhone X. New transfers are now automatically marked as unread to help users keep track of recent activity. Additionally, background fetch and push notifications have been fixed to ensure reliable message delivery. The app now also supports in-app notifications for both incoming messages and transfers.
