ADAMANT Messenger for iOS version 3.2.0 introduces message reactions alongside several usability and stability improvements. ADAMANT is an open-source, decentralized, anonymous messenger built on a blockchain foundation, and this release focuses on making conversations more expressive and the app more responsive.

### Message Reactions

The headline feature is the ability to react directly to individual messages and transfers. Instead of typing a reply, users can attach a quick reaction, which is useful for acknowledgements or confirmations where a full text response is unnecessary. This works for both chat messages and crypto transfers sent through the app.

### Usability Improvements

The context menu for messages has been redesigned to provide quicker access to common actions and smoother navigation. The chat header now displays your conversation partner's avatar, making it easier to distinguish between multiple chats at a glance. Swipe amplitude for the Quote/Reply gesture has been reduced, so less finger travel is needed to trigger the action—a subtle but meaningful refinement for one-handed use.

### Transaction Error Details

Failed transactions now surface more detailed information about what went wrong. This is particularly relevant for ADAMANT's built-in crypto transfers, where network fees, insufficient balances, or node issues can cause a transfer to fail. Clearer error messages help users diagnose and resolve problems without leaving the app.

### Stability and Design Fixes

The update addresses several design-related bugs and improves overall performance stability, reducing unexpected crashes and delays. These fixes apply across both casual chats and business-oriented conversations.

Release notes are available on [GitHub](https://github.com/Adamant-im/adamant-iOS/releases/tag/3.2.0). The app can be downloaded from the [App Store](https://itunes.apple.com/app/adamant-messenger/id1341473829).
