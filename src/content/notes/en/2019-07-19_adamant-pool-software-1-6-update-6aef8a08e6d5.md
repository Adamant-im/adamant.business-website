ADAMANT Pool version 1.6 addresses feedback from pool operators since the previous release. The update fixes minor and major bugs, improves overall system stability, and introduces an updated interface design.

Pool operators can update an existing pool or deploy a new one following the [official setup instructions](https://medium.com/adamant-im/create-your-own-adamant-forging-pool-a8574f5da43b#75bd).
