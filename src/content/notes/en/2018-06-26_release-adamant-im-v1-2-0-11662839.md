This release migrates the project build process to Webpack 4 and includes internal refactoring across the codebase. The back button in chat now consistently returns to the chat list rather than calling history.back. Welcome messages have been added, and users can now log in via QR-code.

Several fixes are included in this release. Camera scanning has been corrected, and ETH addresses are now stored in KVS when the balance is positive. Icon-related issues have also been resolved.
