This release addresses two exchange-specific issues and performs routine maintenance. Order cancellation for StakeCube has been fixed, resolving a problem that could prevent trades from being cancelled properly on that platform. The getRates function for Azbit has also been corrected, ensuring reliable rate retrieval for that exchange.

In addition, dependencies have been updated to their latest compatible versions and the project readme has been refreshed to reflect current information.
