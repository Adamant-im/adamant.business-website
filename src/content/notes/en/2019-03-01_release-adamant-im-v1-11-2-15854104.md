This is a bug fix release that addresses several issues across DOGE, ERC20, and ADM transactions. The update fixes DOGE transactions sent to the same address and adds validation for DOGE addresses. ERC20 transfer display issues are resolved, including incorrect direction icons in the current session and incorrect fee display.

ADM transfers with comments now display correctly, and incoming ADM transactions with comments no longer appear as two separate messages in notifications. Rounding errors in crypto transfers have been fixed. Incoming non-ADM transaction status is now correctly received in the chat list, and status icons are now shown only on outgoing messages instead of all messages. Additionally, an issue with login by password refetching messages has been resolved.
