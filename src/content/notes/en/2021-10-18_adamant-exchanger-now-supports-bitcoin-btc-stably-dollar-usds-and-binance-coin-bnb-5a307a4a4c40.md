The ADAMANT Exchanger, which performs cryptocurrency swaps directly within ADAMANT chats, has been updated to version 2.2.2. The exchanger now supports Bitcoin (BTC), Stably Dollar (USDS, ERC20), and Binance Coin (BNB, ERC20), alongside existing support for Ethereum (ETH), Dash (DASH), ADAMANT (ADM), and the Resfinex token (RES, ERC20). BZ coin has been removed from the supported asset list.

To initiate an exchange, users transfer cryptocurrency to the ADAMANT Exchange bot within a chat session. Sending **/help** to the bot returns current exchange parameters and available commands.

![ADAMANT Exchanger now supports Bitcoin (BTC), Stably Dollar (USDS) and Binance coin (BNB)](/images/engineering-notes/medium/5a307a4a4c40/002-0-xkzcwwyruweqjc1q.webp)

The exchanger is open source under the ADAMANT project, and operators can run their own instance. The full release is available on GitHub at the `adamant-im/adamant-exchangebot` repository under tag `v2.2.2`.
