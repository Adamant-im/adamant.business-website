The ADAMANT Trade & Market-making bot is free, open-source software for running trades on crypto exchanges, generating trade volume, maintaining spread and liquidity, setting price ranges, and building dynamic order books. Version 5.2.0 is a maintenance release focused on stability and connector reliability.

The most significant change in this release is a full rewrite of the P2PB2B API connector, which should improve exchange communication and reduce connector-related errors. The release also includes general refactoring, dependency updates, and bug fixes across the codebase.

Source code is available on [GitHub](https://github.com/Adamant-im/adamant-tradebot).
