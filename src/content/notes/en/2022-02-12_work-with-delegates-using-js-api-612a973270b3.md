The ADAMANT JavaScript API library v1.4.0 introduces two new methods for delegate management: `delegateNew()` to register a new delegate and `voteForDelegate()` to cast votes for existing delegates. This release also includes routine dependency updates. Refer to the [API specification](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification) for usage details.
