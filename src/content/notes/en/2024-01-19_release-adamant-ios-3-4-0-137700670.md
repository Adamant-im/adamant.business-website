This release of ADAMANT for iOS adds fiat currency display for transaction fees on the transaction details screen, giving users a clearer view of costs. The app now includes support for the STORJ token. A language switcher has been implemented, and Chinese language support has been introduced. Several UI bugs have also been resolved.
