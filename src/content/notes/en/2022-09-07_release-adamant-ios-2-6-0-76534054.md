This release introduces a node health check feature for ADAMANT, allowing the application to connect to the best available nodes rather than selecting them at random. Users can now view which nodes the application is currently connected to, along with their ping and status information.

The delegates screen has received a UX update, and several bug fixes and crash fixes have been applied to improve overall stability.
