This release of ADAMANT updates the default node list to ensure reliable connectivity for messaging and blockchain operations. Users should review their node configuration after upgrading to confirm they are connecting to the updated endpoints.

### Breaking changes

The `config.json` file has been removed from the repository. After setting up the application, you must create it manually by running `cp config.default.json config.json`. Before updating, save your existing configuration by running `cp config.json config.json.backup` so you can reference your previous settings when reconfiguring.
