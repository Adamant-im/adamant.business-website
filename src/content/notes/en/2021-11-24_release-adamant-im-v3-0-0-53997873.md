This release delivers a significantly faster Chatrooms API with lower RAM usage, achieved by replacing the underlying HTTP node implementation. The update also includes a workaround for a Lisk Service `includePending` bug and fixes a light/dark theme `background-color` issue. An infinite update loop warning that appeared in the ADM transaction list component render function has been resolved, and project dependencies have been updated.

### Breaking changes

Support for the Resfinex exchange and its associated RES token has been removed from the application.
