This release focuses on internal refactoring and dependency updates for ADAMANT Tradebot. As part of the maintenance work, a `config.default.json` file has been added to provide a clearer baseline configuration for operators.

A new market maker option, `mm_orderBookMaxOrderPercent`, is introduced, allowing finer control over order book participation.

### Breaking changes
No breaking changes are introduced in this release.
