This release of the ADAMANT iOS app introduces several improvements to transaction processing and the chat experience.

The messenger now performs a coins health check, selecting only working nodes to connect. Users can choose between connecting to random nodes or preferring the fastest node. The transactions list has also been improved: the app stores transactions locally and no longer reloads the full list every time it is opened.

On the chat screen, a short tap on a recipient's name opens the QR code of their address, while a long tap allows renaming the recipient.

Additional fixes are included to improve the overall experience.
