This release introduces an initial QR code scanning implementation, which may still contain bugs, and adds an option to send messages on Enter.

Several fixes are included. Sending messages on Cmd+Enter has been corrected. The maximum amount available for transfer could sometimes be incorrect, and this has been resolved. A floating point multiplication bug has also been fixed.

New translations are available for Japanese, Ukrainian, and Italian.
