ADAMANT Messenger for iOS version 1.9.2 ships with a round of bug fixes and several convenience improvements. New accounts now include static chats for the Exchanger and Betting bot, and a prompt offers free ADM tokens when a new account sends its first message. On the Account tab, a swipe-down gesture refreshes wallet data. Wallet history screens now display a "No transactions yet" placeholder when empty and show the year for older transactions.

On the messaging side, a scroll bug triggered when sending a message has been fixed. The app also checks whether a chat partner's account has a public key, and an issue causing misbehavior when an account has an empty name has been resolved. Delegate voting no longer fails on a second vote, and a crash that occurred on re-login is fixed.

Release 1.9.2 is available on [GitHub](https://github.com/Adamant-im/adamant-iOS/releases/tag/1.9.2).
