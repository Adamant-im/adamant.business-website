This release of ADAMANT for iOS introduces message reactions, which also work with transfers, and an enhanced context menu for messages. Users can now view a partner's avatar directly in a chat, and the app provides detailed information for unsuccessful transactions.

Several design-related bugs have been addressed, and the swipe amplitude has been reduced for a smoother interaction. Overall application performance is now stable.
