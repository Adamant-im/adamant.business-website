This release introduces style changes to make new fields more usable on mobile devices. Login with a passphrase stored as a QR code is now supported, thanks to @chovy. The development workflow has been improved so that `npm run dev` now supports hot reload.

Several fixes and refinements are included. Storing data in KVS is now triggered only when enough ADM tokens are available, eliminating nagging windows. A bug has been fixed, and new language constants have been added alongside general language updates.

Ukrainian, Filipino, Swedish, Hindi, Turkish, Vietnamese, Romanian, and Arabic languages have been temporarily turned off until someone updates the corresponding language files.
