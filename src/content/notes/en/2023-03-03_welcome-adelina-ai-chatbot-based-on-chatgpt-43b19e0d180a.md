ADAMANT Messenger has launched Adelina, an AI chat assistant based on ChatGPT. Adelina uses natural language processing to understand and respond to user queries. It can draft emails and other writing, generate Python code, answer questions against a set of documents, hold conversational exchanges, tutor across various subjects, translate languages, compose poetry, simulate video game characters, and more.

![Welcome Adelina, AI chatbot based on ChatGPT](/images/engineering-notes/medium/43b19e0d180a/002-0-ilrmqdyclcgkby8t.webp)

During the testing period, Adelina is free to use. No OpenAI ChatGPT account, API keys, phone numbers, or other account identifiers are required. However, OpenAI may read conversations to improve the chat engine; refer to the ChatGPT FAQ for details.

Adelina ships in PWA v3.3.0 and iOS app v2.9.0. The Bet on Bitcoin price chat feature has been removed in the same releases. If you have used the ADAMANT web app previously, your browser may cache an older version — clear it with Ctrl + Shift + R in Chrome, for example. iOS App Store delivery may lag by one to two days. You can also start a chat with Adelina manually using her address: U11138426591213238985.
