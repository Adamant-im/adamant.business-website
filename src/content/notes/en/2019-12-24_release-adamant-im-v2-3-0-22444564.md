This release adds support for Bitcoin within the application. The transaction details screen now dynamically updates the confirmation count. To improve status management, a Registered status has been added for transactions, allowing statuses to be updated more effectively. Additionally, the release fixes status reporting for Ethereum transactions and resolves an issue with processing links to msg.adamant.im chats.
