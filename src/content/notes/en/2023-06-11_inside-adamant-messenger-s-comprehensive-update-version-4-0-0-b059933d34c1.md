ADAMANT Messenger version 4.0.0 introduces a comprehensive technology stack update aimed at improving overall performance and reducing memory consumption. This release serves primarily as a foundational upgrade to facilitate future feature development.

Users will experience faster and smoother messaging due to optimized performance and various bug fixes that enhance stability. ADAMANT Messenger continues to operate as a Progressive Web Application (PWA), maintaining compatibility across Web, Tor, Android, Windows, Linux, and macOS environments. 

Further technical details are available in the official release notes on the ADAMANT GitHub repository.
