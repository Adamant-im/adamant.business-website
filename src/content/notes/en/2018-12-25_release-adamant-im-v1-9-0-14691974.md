This release rewrites and updates how data is stored on the user's device. Users who prefer not to log out automatically must now set an additional password, which is used to encrypt local data. All data stored on the device is encrypted using local encryption, and Argon2 is used for password hashing. As a result, storing data on personal devices for faster access is now secure.

Secured data is also moved from LocalStorage to IndexedDb, which provides more space for messages.
