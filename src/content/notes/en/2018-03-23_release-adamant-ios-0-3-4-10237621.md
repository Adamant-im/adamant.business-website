This release of ADAMANT iOS introduces the ability to read QR codes directly from photos and images stored in your Photo Library. Users can also save generated QR codes back to the Photo Library for convenient access. Several UI improvements are included, though this work remains in progress.

Network reachability handling has been improved to prevent unnecessary update requests when connectivity is lost. An issue causing service updates to continue while the app was in the background has been fixed. Additionally, several minor bugs have been resolved.
