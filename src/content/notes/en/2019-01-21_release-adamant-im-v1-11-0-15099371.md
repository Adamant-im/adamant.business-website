This release of ADAMANT introduces DOGE token wallet functionality and transfer capabilities for the web application. Users can now manage and transfer DOGE directly within the platform.

Additionally, message sending for ETH-like currencies has been optimized for faster performance. A transaction spinner has been added to provide visual feedback during transaction processing, along with various other improvements to the overall user experience.
