This release of ADAMANT Messenger introduces tooltips for icons in the desktop version, improving usability by clarifying the purpose of each interface element. Additional scam protection measures have been implemented to help safeguard users against malicious activity. The update also includes various style changes and updates to language constants.

### Breaking changes

No breaking changes are introduced in this release.
