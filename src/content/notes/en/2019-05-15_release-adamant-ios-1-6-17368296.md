This release introduces currency and fiat rate displays across the account, transfers, and transactions views, including historic values starting from March 20, 2019. Users can now generate and export private keys directly within the app.

Several bugs and memory leaks have been addressed in this version. A `firstRun` flag has also been set; this flag will be used in a future update to purge the keychain store on first run.
