ADAMANT Messenger v1.6.0 introduces several user interface and performance improvements across the application. The release includes multiple style changes, an optimization of delegate list fetching, and an internal message queue for improved message handling.

Users will notice immediate UI responses when sending messages, along with new message statuses displayed in both individual chats and the chat list. These enhancements provide better visibility into message delivery and overall chat state.

This release also fixes a scrolling bug that affected chats, improving the reliability of navigation within conversation views.
