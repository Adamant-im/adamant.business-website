ADAMANT iOS 2.7.0 brings a dark theme for more comfortable viewing in low-light conditions. Chat loading has been improved for faster message retrieval and a smoother user experience.

This release also adds notification settings, giving users more control over alerts, and now displays the ERC20 network for tokens so users can clearly identify which network a token operates on.

Several stability improvements are included. A crash that occurred when copying a message has been fixed, along with a crash triggered during QR scanning with the camera. Bugs specific to MacBook M1 devices have been resolved, and additional general bug and crash fixes round out the release.
