ADAMANT Node v0.7.0 focuses on cleanup and dependency maintenance. The release updates project dependencies, moves legacy code into a dedicated `/legacy` directory, and removes outdated documentation. These changes do not affect network consensus, so node owners are not required to upgrade.

Release details are available on the [v0.7.0 GitHub release page](https://github.com/Adamant-im/adamant/releases/tag/v0.7.0).
