This release adds support for Rich Content Messages as defined in [AIP 5](https://aips.adamant.im/AIPS/aip-5), enabling richer message formatting within the ADAMANT messaging platform. Support for Signal Messages has also been introduced per [AIP 6](https://aips.adamant.im/AIPS/aip-6). Additionally, this release fixes filters in the state(KVS) transaction API.
