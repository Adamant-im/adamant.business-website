## Changes

This release reduces the message sending fee from 0.005 to 0.001 AMD, lowering the cost of communication on the network. The application now checks for incoming messages more frequently, improving responsiveness. A button linking to the ICO page has been added, and users now receive ICO messages instead of the previous Pre-ICO messages. The favicons have been updated, along with some minor internal changes.

## Fixes

An issue causing messages to disappear has been resolved. A temporary fix has been applied for Local Storage, which is expected to be completely removed in upcoming releases. The link rendering issue (#16) that could potentially be used to trick users has been addressed.

## Translations

New translations have been added for Arabian, Deutsch, Portuguese (contributed by @DataDrug via PR #17), Romanian, Swedish (contributed by @pdahlberg via PR #29), and Vietnamese.
