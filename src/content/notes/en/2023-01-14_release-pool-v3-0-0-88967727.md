This release updates the project dependencies and rewrites the dashboard using Svelte. The entire code base has been redesigned and optimized, and several bugs have been fixed.

### Breaking changes

Support for Node.js versions below 18.12.1, the current LTS release, has been dropped. The pool now uses lowdb for its database. Refer to the migration section in the README file for details.
