The ADAMANT JavaScript API library now supports Lisk cryptographic keys alongside a comprehensive codebase refactor. The v1.5.0 release replaces the previous HTTP client with Axios, improving request handling and maintainability across the library. Contributing guidelines and code style have also been updated to streamline collaboration.

Full API documentation is available on the [ADAMANT API wiki](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification), and release details are on the [v1.5.0 release page](https://github.com/Adamant-im/adamant-api-jsclient/releases/tag/v1.5.0).
