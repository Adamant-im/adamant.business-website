This maintenance release for the ADAMANT iOS application removes the node btcnode2.adamant.im from the network configuration and adds btcnode3.adamant.im as a replacement. The icon for the MANA token has been replaced with an updated asset.

A message height issue has been fixed, and the dialog screen performance has been improved for smoother operation. The reaction background color has been adjusted for better visual consistency. The display of notification content, which had been affected in a prior release, has been restored.

Several additional user interface enhancements round out this release.
