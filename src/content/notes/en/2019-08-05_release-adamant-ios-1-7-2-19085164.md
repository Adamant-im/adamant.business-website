This release of ADAMANT iOS addresses several bugs affecting data handling, chat visibility, address book entries, and application stability.

Fixed an issue where user data was not fully cleared on logout. ADM transfers without comments now correctly appear in the Chat and Transactions list. Names read from QR codes are now properly saved to the Address Book. Resolved a crash that occurred when the app operated without a network connection.
