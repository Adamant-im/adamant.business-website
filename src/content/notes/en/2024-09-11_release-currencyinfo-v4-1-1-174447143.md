This release adds a notification error message for cases when there are no previous rates to fall back to.

Several fixes are included. The base and quote coins in the `getHistory` endpoint were swapped, so the response now correctly shows `'ADM/USD': 0.03` instead of `'USD/ADM': 0.03`. Detection of significant differences when merging sources has been corrected. The calculation of each coin's price relative to every coin from the `base_coins` has also been fixed.
