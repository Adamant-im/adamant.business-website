This hotfix release for ADAMANT iOS addresses several crash-related issues and includes minor chat fixes.

The most significant fix resolves a crash that occurred when sharing a QR code without a sharing tip. Another crash has been fixed that was triggered when tapping on an email address or URL containing an unknown protocol within a chat. Additionally, several smaller chat-related issues have been resolved to improve overall messaging stability.
