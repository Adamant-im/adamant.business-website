This is a hotfix release that resolves several issues affecting the ADAMANT Messenger application.

The release fixes an issue with voting functionality. It also addresses a problem with incorrect ordering of requests, which had prevented some users from receiving free tokens. Additionally, an issue with new chats has been resolved.
