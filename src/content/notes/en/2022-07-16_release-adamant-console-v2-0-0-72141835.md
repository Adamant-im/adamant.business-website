This release delivers a full refactoring of ADAMANT Console, now built on top of adamant-api. The console gains the ability to decrypt messages and provides built-in help for available commands. An interactive mode has been introduced to streamline usage.

The package can now be included as a JavaScript library, broadening integration options beyond the command line. Dependencies have been updated and several bug fixes are included.

### Breaking changes

The complete refactoring means the internal structure and APIs have changed significantly. Existing scripts or integrations relying on the previous architecture will need to be updated to work with the new adamant-api-based foundation.
