This release introduces a new Wallets screen in the settings, allowing users to configure the order and display of coins. In the chat module, users can now select previous bot commands using keyboard arrows, and an emoji picker has been added. The FLOKI token has been added to the supported wallets. The crypto transfer and pending transaction scenarios have been improved, along with the user interface for nodes health-checks, which now also displays the coin node's version. Three new exchanges have been added, and the structure of nodes and services has been updated. Additional minor features and fixes are also included.

A bug causing an infinite loading spinner in the chat after a page refresh has been fixed.

Technical debt has been addressed by upgrading dependencies, including Vue, Vuetify, Electron, and other libraries such as lisk-* and web3-*.
