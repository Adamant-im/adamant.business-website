This release introduces Resfinex Token (RES) support and updates the Stably Dollar (USDS) name and logo. The Send Token menu height has been adjusted for better usability, and the Buy & Sell tokens dialog has been updated.

A client-node socket connection is now available, along with a new "In blockchain" status for messages. BTC address validation has been added, and a fix addresses paste-from-clipboard address validation. An additional fix resolves an issue with negative send amounts.

### Breaking changes

No breaking changes are introduced in this release.
