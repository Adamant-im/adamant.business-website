This release updates the moment-js dependency, which had been flagged as insecure. Since it was only used as a dev-dependency for migration generation, this did not present an actual security issue. The README has also been updated.

A bug in the state(KVS) transaction signing mechanism has been fixed, and the mechanism has been made compliant with AIP-3.
