This release optimizes transaction update timings across different coins and fixes UTXO handling for Doge. A new txVerify module checks transaction consistency, and transaction details can now be refreshed from both the chat screen and the transaction details screen.

Crypto address validation has been added for both user and partner accounts, and transaction consistency is verified in the transaction details view. The Nodes view now displays the node version. A bug causing an empty transaction list has been fixed, along with other minor bug fixes.
