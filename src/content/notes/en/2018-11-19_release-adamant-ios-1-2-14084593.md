This release introduces the first built-in crypto wallet to the ADAMANT iOS application, starting with Ethereum. Users can now send, receive, and view transaction history directly within the app. The update also adds the ability to send supported tokens, including ADM and ETH, within chat conversations.

Significant performance improvements have been achieved by migrating the core to native code. Additional refinements include relocating the "Stay in" and "Touch/Face ID" options to the Account page, along with various bug fixes. The project has also been updated to Swift 4.2.
