The ADAMANT development team has released version 0.6.1 of the ADAMANT Messenger Android application. This release focuses on visual polish and stability as the team continues preparing the app for publication on Google Play.

The updated version brings a refreshed message item design and a redesigned settings screen. QR-code functions, which were previously unavailable, have been restored. The registration screen has been refactored for improved reliability, and avatar effects have been added. Messages from the current day are now highlighted for easier navigation, and the app introduces a new color schema across its interface.

Several bugs have been addressed in this release. The login window now responds correctly to valid passphrase input, and the navigation issue when returning back after login has been resolved. A bug where a companion's avatar failed to display if they had not yet replied has been fixed, along with incorrect dialog window styles and other minor issues.

The Android app is available for download from the [ADAMANT GitHub releases page](https://github.com/Adamant-im/adamant-android/releases).
