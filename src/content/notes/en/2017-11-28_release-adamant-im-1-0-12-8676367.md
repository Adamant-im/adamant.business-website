This release brings a refreshed visual style to the ADAMANT Messenger application, along with targeted fixes for Android devices. The logout flow now includes a redirect to improve the user experience when ending a session.

Transaction state now auto-updates, allowing users to see the latest status of their transfers without manual refresh. Additionally, chat sorting has been improved to provide a more intuitive ordering of conversations.
