ADAMANT Messenger, a blockchain-based communication platform where every message is a transaction, has introduced a Reply/Quote feature in its latest iOS app update. Because ADAMANT operates on a decentralized architecture, implementing standard messaging features like quoting requires unique technical approaches compared to conventional centralized messengers.

The Reply/Quote feature can be activated by swiping left or long-pressing a message. Alongside this, the update introduces a Contribute section, allowing users to support the platform's development directly. Optional crash analytics have also been added; disabled by default, users can enable Crashlytics from the Contribute section to help developers gather insights and improve app stability.

![The Reply/Quote feature landed on the ADAMANT Messenger iOS app](/images/engineering-notes/medium/1ac2e43cbef1/002-0-pdyl0ofrnylbkm4e.webp)

Several wallet and transaction issues have been resolved in this release. BC1 (SegWit) addresses are now correctly operational for BTC wallet transactions. Gas limit issues for ETH and ERC20 transactions have been fixed, and a switch to increase gas for ETH, ERC20, and BTC transactions has been added. Address validation for coins has improved, including support for the "amount" parameter in QR codes.

Additional enhancements include fixing a scrolling bug in messages, allowing users to start a new chat directly from the search field, and resolving design bugs for macOS, iPad, and iPhone. New nodes have been added for enhanced stability and performance; users can access a fresh list of nodes by tapping 'Reset' on the Nodes screen.
