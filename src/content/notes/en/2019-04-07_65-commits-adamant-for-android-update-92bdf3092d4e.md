ADAMANT for Android release 0.9.0 includes more than 30 changes across design and functionality, along with bug fixes. The release introduces PIN-code authorization (in beta), a "Message not sent" status indicator, and a "Last transactions" section that displays transactions with dates. The push notification mechanism has been reworked, and a new welcoming screen has been added. Login-related bugs were fixed alongside seven other resolved errors.

Users already running the app should verify they are on version 0.9.0; if not, the previous version should be removed and the new build installed in its place. The APK is available from the [GitHub release page](https://github.com/Adamant-im/adamant-android/releases/tag/release-0.9.0) under the Assets section.
