ADAMANT Tradebot v5.6.0 introduces support for the Coinstore exchange, expanding the range of platforms the bot can operate on. This release also adds Discord as a notification channel, allowing users to receive trade alerts and status updates through their Discord servers.

A new health check endpoint has been added to the bot, making it easier to monitor whether the service is running correctly. This is particularly useful for deployments behind reverse proxies or orchestration systems that perform periodic liveness checks.

In addition to these features, the release includes general bug fixes and updates to project dependencies to maintain compatibility and security.
