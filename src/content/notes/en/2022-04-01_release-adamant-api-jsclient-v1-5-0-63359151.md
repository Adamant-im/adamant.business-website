This release of the ADAMANT API JS Client introduces support for the Lisk blockchain alongside the existing ADAMANT functionality. The HTTP layer has been migrated to an Axios-based client, improving request handling and maintainability across supported networks.

A broader refactoring pass accompanies these changes, cleaning up internal structure and reducing duplication. The project also receives updates to its style guidelines and contributing documentation to help new contributors get started more easily.
