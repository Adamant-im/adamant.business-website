This release introduces liquidity maintenance and spread maintenance capabilities, along with a new `/enable liq` command for configuring both. Price watching functionality has been added, with `/make price` and `/enable pw` commands available for price setup.

The `/balances` command now provides additional information. Logging has been updated, request timeouts are now processed, and market making trends have been introduced.
