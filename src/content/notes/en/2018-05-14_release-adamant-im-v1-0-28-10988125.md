This release of ADAMANT Messenger introduces the ability to display transfers directly within chats, making it easier to track transaction history alongside conversation context.

Several fixes have been applied in this version. Manifest files have been updated, and the service worker now caches web fonts to improve offline reliability and loading performance. A number of style updates have also been included to refine the user interface.

Translation updates are provided for Korean, Spanish, and French, improving localization coverage for those language communities.
