This release of ADAMANT iOS introduces system accounts, local messages, and chatrooms, expanding the messaging capabilities of the application. The wallet scene now includes a "Get free tokens" row, making it easier for users to acquire tokens directly from the wallet interface.

System messages now support Markdown formatting, improving readability for automated communications. The login scene has been refined with better error handling, providing clearer feedback when authentication issues occur.

Notifications have been improved with better app badge number handling, ensuring unread message counts are accurately reflected. This release also includes many internal improvements for better stability and performance.
