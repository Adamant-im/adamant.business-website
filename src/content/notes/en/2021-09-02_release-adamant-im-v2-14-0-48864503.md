This release introduces several improvements to transaction handling and messaging functionality. Users can now copy the transaction ID, sender, and recipient fields directly from transaction details, and the update button in that view has been animated for better visual feedback. Dash sending parameters have been adjusted with corrected maximum and minimum amount thresholds, and a fix addresses issues when sending small amounts of Lisk. Error messages displayed during transaction sending have been reformatted for improved clarity.

The codebase has been updated to account for the London ETH hardfork. The snackbar component has been tuned for better behavior, and the Tor website link has been replaced with a new destination. Underlying dependencies have been updated to their latest compatible versions.
