This release of ADAMANT iOS introduces support for Dash, expanding the range of cryptocurrencies available within the app. New installations will now start with a clean configuration by dropping any previously stored settings, ensuring a fresh initial setup experience.

Several bug fixes have been applied in this version to address issues reported by users and to improve overall application stability.
