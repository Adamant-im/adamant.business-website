ADAMANT Messenger v2.9.0 introduces several enhancements focused on Tor integration, security, and user interface improvements. The Tor version of the app now includes direct onion links to the Buy tokens and Free tokens pages, furthering privacy for users operating over the Tor network.

To maintain stability and security, this release updates package dependencies and refreshes the list of available ADM nodes. Message sanitization has been improved by integrating DOMPurify, which helps prevent cross-site scripting vulnerabilities in chat messages. Additionally, the confirmed status for socket-based ADM transfers is now displayed correctly.

The user interface receives a notable update with a refreshed logo and icons. Chat usability is improved with a new option to display full dates, alongside a repositioned timestamp for messages. Snackbar notification timeouts have been enhanced for better visibility. 

Wallet and transaction features are also refined in this version. A new Adoption wallet and a Donate static chat have been introduced, with updated texts for static chats. The transaction list and details views have received multiple fixes, and the Dash explorer link has been updated to ensure accurate transaction tracking.
