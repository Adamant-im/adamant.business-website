This release refactors the ApiService for improved internal structure. Avatars are now displayed for ADAMANT contacts, though user avatars are not yet supported by the ADAMANT network.

The Chat scene has been updated with a numeric keyboard for the text field, and a copy-to-clipboard button for partner addresses has been added. The fee structure has also been updated.

Message duplicating and disappearing issues have been fixed via an updated API.
