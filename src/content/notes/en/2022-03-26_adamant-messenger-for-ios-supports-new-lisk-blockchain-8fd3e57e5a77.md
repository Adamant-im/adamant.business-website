The ADAMANT Messenger iOS application has been updated to support the new Lisk blockchain. This release, version 2.2.0, implements native Swift support for Lisk transactions. Under the hood, the application benefits from updated libraries and code refactoring, resulting in improved speed and stability. Additionally, legacy KCS, BZ, and RES wallets have been removed.
