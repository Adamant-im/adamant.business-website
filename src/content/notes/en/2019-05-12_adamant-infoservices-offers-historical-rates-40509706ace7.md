ADAMANT InfoServices has been updated with a new `getHistory` endpoint that provides historical currency rates. Developers can refer to the [InfoServices API documentation](https://github.com/Adamant-im/adamant-currencyinfo-services/wiki/InfoServices-API-documentation) for integration details.

Fiat currency display for crypto balances is planned across ADAMANT applications, starting with the iOS app. For background on how exchange rates work within ADAMANT, see the earlier [overview of InfoServices](https://medium.com/adamant-im/exchange-rates-in-adamant-8b6a1713d2b0).
