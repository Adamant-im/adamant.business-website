The ADAMANT Exchanger has been updated to support Dash transactions using InstantSend. Because Dash transfer fees are only 0.0001 DASH, the minimum exchange amount restriction has been removed. Additionally, the exchange fee has been lowered to 4% and the daily limit increased to $1000.

To make an exchange, transfer cryptocurrency to the ADAMANT Exchange directly within the chat. You can retrieve exchange parameters and command help by sending `/help`.

![Exchange Dash in the ADAMANT Exchanger, InstantSend supported!](/images/engineering-notes/medium/c9c2344bbb/002-0-ev3nbcqy93qlrrb.webp)

The ADAMANT Exchanger is open source, allowing anyone to operate their own instance. The v2.0.1 release introduces Dash support with InstantSend and compatibility with Ethereum's London hardfork. It also removes the requirement to specify the output currency in the transaction comment. This version includes a total code refactoring, linting, and general optimizations, alongside a switch to ADAMANT API 1.1.0 and updated dependencies.
