The iOS release of ADAMANT Messenger v3.10.1 focuses on improving stability and resolving important bugs. This update addresses an issue where Ethereum and ERC20 transaction details failed to display block and confirmation information correctly, causing valid transactions to appear as failed. Additionally, the release fixes incorrect processing of v0.9.0 ADM node responses, which previously prevented the chat list from loading.
