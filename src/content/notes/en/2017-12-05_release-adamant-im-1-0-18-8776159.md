This release of ADAMANT Messenger introduces support for treating a single line break as a line break in messages, consistent with GitHub Flavored Markdown. Previously, single line breaks were not rendered as line breaks; this change brings message formatting in line with common markdown conventions.

A fix has been applied to the unread messages counter, resolving an issue where the count could display incorrectly. Additionally, a temporary fix addresses compatibility issues on iOS 11, improving stability for users on that platform. Small adjustments have also been made to the state storage logic to further refine reliability.

### Breaking changes

There are no breaking changes in this release.
