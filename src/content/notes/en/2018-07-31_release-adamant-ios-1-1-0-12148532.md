This release of ADAMANT iOS introduces the ability to rename a partner directly from the Address Book, giving users more control over their contact labels. The "To Chat" button is now hidden when transaction details are presented from within a chat, reducing unnecessary interface clutter.

A new "Free tokens" icon has been added to the interface. Balance formatters have been fixed to ensure values display correctly across the app. The UIColor.adamant component was refactored for improved code maintainability. Additionally, a display issue with the biometry row on older devices without Touch ID has been resolved.
