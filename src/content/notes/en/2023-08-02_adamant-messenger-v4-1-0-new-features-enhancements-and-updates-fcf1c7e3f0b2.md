ADAMANT Messenger v4.1.0 introduces quote and reply messages per AIP-16, a "Mark all as read" button for notification cleanup, and a ten-row cap on multiline message input to keep the chat interface compact. These changes make conversations easier to follow and reduce visual clutter.

On the wallet side, a generation script from the adamant-wallets repository simplifies adding support for new tokens. A pull-down gesture now refreshes balances, with an ellipsis indicator shown while loading. The supported coin list has expanded to include Tether (USDT), USD Coin (USDC), and Dai alongside existing support for ADM, BTC, ETH, Doge, Dash, and Lisk.

![ADAMANT Messenger v4.1.0: New Features, Enhancements, and Updates](/images/engineering-notes/medium/fcf1c7e3f0b2/002-0-rgxdqzzrdjk3m7qj.webp)

Interface refinements include a spinner during QR code generation and the addition of StakeCube exchange to the BuyTokensDialog. Delegate voting icons have been restyled for clearer visual feedback.

![ADAMANT Messenger v4.1.0: New Features, Enhancements, and Updates](/images/engineering-notes/medium/fcf1c7e3f0b2/003-0-xor1ka7rgr2ppf0e.webp)

The release also bundles bug fixes and library updates aimed at stability and compatibility. ADAMANT Messenger is a Progressive Web Application available for Web, Tor, Android, Windows, Linux, and macOS. Full release details are available at the GitHub release page: https://github.com/Adamant-im/adamant-im/releases/tag/v4.1.0
