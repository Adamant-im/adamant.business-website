The Market-making App is an open-source, self-hosted solution developed by the ADAMANT project for maintaining token liquidity, trading volume, spread, and order books. It provides token owners with a tool to attract investors and traders by ensuring healthy market activity.

Unlike many market-making solutions, this application is entirely self-hosted and self-controlled, giving token owners full custody of their operations. The core features are available for free, making it a cost-effective option for full-cycle market making.

The project's [website](https://marketmaking.app/) provides comprehensive documentation, including an installation guide, a description of the free market-making bot, details on custom features, and a command reference. The source code is available on [GitHub](https://github.com/Adamant-im/adamant-tradebot).
