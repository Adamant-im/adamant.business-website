This release addresses several bug fixes in ADAMANT Messenger. It resolves an issue with incorrect sender names being displayed, and fixes console errors related to service workers that occurred during page reloading. The update also fixes the incoming message scroll behavior and resolves a bug with the status message when sending ADM tokens.
