This release of ADAMANT Messenger expands the list of nodes available for connection and introduces a check for node protocol. Node statuses and socket connections on HTTP hosts have been corrected, and underlying dependencies have been updated.

Exchange integrations have been adjusted: a link to CoinDeal has been added, the IDCM link has been removed, and links to Resfinex and Bit-Z have been updated. Several chat-related issues are resolved, including styles for inline `code` in chats, static chat names, pasted address validation when starting a new chat, localization of "no public key" and "no hash" messages, and QR generation for ADM addresses on Windows and other apps.
