ADAMANT Messenger launched in April 2017 as a concept that progressively evolved into a full technical architecture, blockchain network, registered entity, and set of applications. The project now positions itself as the only messenger operating entirely via a blockchain, providing a secure and anonymous platform for online messaging.

A notable technical feature already implemented is in-chat Ethereum transfers, alongside ADM token transfers. This capability allows users to conduct cryptocurrency transactions directly within conversations, and the team plans to continue expanding the messenger's feature set on this foundation.

Fundraising for the project has been successfully completed. The next development phase focuses on continued active work on ADAMANT Messenger, with the ADM token scheduled to enter exchanges during the third quarter of 2018.
