This release fixes push notifications and changes the logout behavior so that users stay logged in when closing a tab, rather than being signed out automatically. Dark Mode is now the default appearance for the application.

Several link-handling improvements have been made: links to the User passwords documentation now open in a new window with `noopener`, and links within chat messages also open in new windows with `noopener` for improved security.

Transaction details now display comments and show the user's own address labeled as "Me." In the transaction list, comments are also shown, and names alongside ADM addresses are displayed for transactions involving other cryptocurrencies. Additionally, users can navigate directly to a chat from a transaction list entry for other cryptos.
