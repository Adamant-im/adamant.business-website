This release introduces static chats for the Exchanger and Bet bot, along with a new popup for receiving free ADM tokens. Wallets now include a refresh feature, and the wallets history screens display a "No transactions yet" message when no transactions are present and show the year for older transactions. The app now also checks whether a partner's account has a public key.

Several bug fixes are included in this release. A scroll issue when sending a message has been resolved, as has misbehaviour when an account has an empty name. A failure that occurred when casting a second vote for a delegate has been fixed, and a crash on re-login has been addressed.
