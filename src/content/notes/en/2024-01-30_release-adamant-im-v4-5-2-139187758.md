### Features

This release expands wallet support by adding STORJ and Gate tokens to the ADAMANT wallet integration. An internal page was added for testing `navigator.vibrate` patterns, aiding development of haptic feedback across supported platforms. Additionally, the `Exo 2` typeface was replaced with system fonts to improve rendering performance and reduce bundle size.

### Bug fixes

A search issue on the Delegates page has been resolved, restoring expected filtering behavior when looking up delegate accounts.
