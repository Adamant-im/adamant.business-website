This release of ADAMANT iOS introduces new error messages that provide clearer feedback to users when issues arise. The transactions list has been fixed, resolving problems that affected the display and reliability of transaction history.

A list of nodes is now available, giving users better visibility into available connection points. The release also includes many internal improvements and fixes that enhance overall stability and performance.

### Breaking changes

There are no breaking changes in this release.
