This release introduces a new transaction list window along with multiple style changes and internal refactoring improvements across the codebase.

Several bugs have been resolved in this version. The error that occurred when sending ETH with an overridden contact name address has been fixed. Opening a chat without messages from the transactions list now works correctly. Scroll-to-bottom behavior when opening a previous chat has been fixed. Additionally, the Free ADM tokens block on the Wallet tab is now hidden when the balance is greater than zero.
