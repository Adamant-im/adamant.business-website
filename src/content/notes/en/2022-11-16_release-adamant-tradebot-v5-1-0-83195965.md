This release of ADAMANT Tradebot introduces support for `.jsonc` configuration files, allowing comments and trailing commas in your bot configuration for easier maintenance. The notification system has been expanded to send messages to several channels simultaneously, and a new priority notification mechanism ensures critical alerts reach operators promptly.

Command aliases are now available, letting users define shorthand names for frequently used commands. Interactive confirmations have been added for the `/buy`, `/sell`, and `/fill` commands to help prevent accidental trades. In-orderbook trading has also been improved for more reliable execution when working directly against the order book.
