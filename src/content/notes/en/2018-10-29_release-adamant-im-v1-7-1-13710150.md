This release introduces a node list health check feature and rewrites the internals of the ADM API client for improved reliability.

Several fixes are included in this build. The bottom bar no longer disappears during use, and the empty messages bug reported in issue #139 has been resolved. Markdown links now render correctly as described in issue #141. Voting-related issues have been addressed, and the Travis build script has been fixed.
