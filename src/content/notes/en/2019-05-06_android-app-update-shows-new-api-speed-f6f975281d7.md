### Faster API in the Latest Android Release

Following earlier announcements about API speed improvements for ADAMANT node, the Android development team has released an update that includes a new chats API.

A comparison between the old and updated API shows that both chat operations and the login procedure complete significantly faster than before. Further optimizations are planned.

The latest APK is available from the [GitHub releases page for Android](https://github.com/Adamant-im/adamant-android/releases/). Existing users should ensure they are running the latest version; if the app does not update automatically, uninstall the previous version and install the new APK manually.

Feedback and bug reports can be submitted through the GitHub repository. The team is also preparing a Google Play release.
