This release of ADAMANT Messenger introduces support for AIP-8, AIP-9, and AIP-15. Official Exchangebot and Betbot have been added to the address book for easier access.

Crypto addresses now feature a Share button that allows users to quickly copy an address or generate a QR-code. All address input fields can now be populated using a QR-code reader, streamlining the transfer process. Input fields on desktop devices display a hint regarding emoji usage. Additionally, transfer statuses are rechecked when logging in via LoginByPassword.
