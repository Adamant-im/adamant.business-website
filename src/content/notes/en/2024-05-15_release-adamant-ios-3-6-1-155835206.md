This release introduces partial text copying within messages and the ability to rename a contact directly from the Partner screen.

The user interface has been refined with improved display of multi-line messages, updated Welcome screens on macOS, and a revised warning about node availability.

Several bug fixes are included. QR code reading has been corrected, the ADAMANT Bounty Wallet now displays its name properly in the Transaction list, and a message's menu position has been fixed for certain cases.
