Previous releases had some issues that prevented them from being ready for production use. This release addresses those concerns with a set of refinements aimed at stability and usability.

The update introduces slight design changes and relocates dialog messages for improved layout. Date formatting has been adjusted, and chat loading behavior has been modified to provide a smoother experience. Performance optimizations have been applied throughout the application, and connection errors that affected earlier builds have been resolved.

New public nodes have been added to the server list, expanding connectivity options for users. New message notifications have also been fixed to ensure timely and reliable alerts.
