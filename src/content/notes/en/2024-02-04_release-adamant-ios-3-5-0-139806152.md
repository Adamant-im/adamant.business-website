This release of ADAMANT iOS introduces several safety enhancements and bug fixes. Users can now identify duplicate transfers within the chat interface, helping to prevent accidental or fraudulent repeated transactions. The app also incorporates support for the GT token and connects to new Ethereum and Lisk nodes.

Additional improvements include modified node height formatting and a requirement that only nodes running a version above the minimum supported version are utilized. The release addresses keyboard-related issues and resolves various UI bugs. Additional causes for inconsistent status indicators have also been introduced to improve diagnostic clarity.
