This release adds a LICENSE file to the repository. Contact list storage now follows the secure recommendations outlined in AIP-3, with backward compatibility maintained for contacts stored in previous versions.

Several fixes are included in this release. A bug affecting the synchronisation of multiple contact list changes across different devices has been resolved. Empty and unused component lifecycle stages have been removed, along with other unused components. An issue causing an infinite loading GIF during incorrect load has also been fixed.
