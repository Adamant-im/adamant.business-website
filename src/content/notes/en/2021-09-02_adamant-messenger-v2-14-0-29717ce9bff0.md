ADAMANT Messenger version 2.14.0 introduces several stability and usability improvements for cryptocurrency transfers, alongside updated Tor-onion links in the Progressive Web App (PWA). Transaction details now support copying the transaction ID, sender, and recipient addresses directly. For Dash, the maximum and minimum send limits have been adjusted, and a bug preventing the transfer of small Lisk amounts has been resolved. The application also includes compatibility updates for the London Ethereum hard fork. Error messages encountered during transaction sending are now formatted more clearly, and snackbar notifications have been refined. The update button within the transaction details view has been animated to improve user experience. Finally, the Tor website link has been updated and underlying dependencies have been refreshed.
