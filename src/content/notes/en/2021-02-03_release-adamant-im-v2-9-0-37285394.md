This release introduces a refreshed logo and updated icons across the ADAMANT Messenger application. Onion links for purchasing and obtaining free tokens have been added to the Tor version of the app. Dependencies have been updated, and the ADM node list has been refreshed to current network participants.

The confirmed status for socket-based ADM transfers is now reported correctly. A new Adoption wallet has been added, and a static donation chat has been introduced. Texts for static chats have been revised, and the time position within chat conversations has been adjusted. Users now have the option to display dates in full format. Snackbar timeouts have been enhanced for better user experience.

Message content is now sanitised using DOMPurify to improve security. Numerous fixes have been applied to the transaction list and transaction details views. The Dash explorer link has been updated to point to the correct service.
