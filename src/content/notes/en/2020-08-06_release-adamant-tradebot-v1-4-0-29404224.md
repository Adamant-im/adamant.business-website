This release of ADAMANT Tradebot introduces support for the Atomars exchange, expanding the range of platforms the bot can operate on. Order book building has been made dynamic, improving how the bot reacts to changing market conditions.

A silent mode for notifications has been added, allowing operators to suppress notification output when desired. A new configuration parameter is also available: an interval in minutes after which all opened orders are automatically cleared.
