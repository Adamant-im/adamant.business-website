This release brings transfer transaction display directly into the chat view, allowing users to see transfers inline within their conversations. Tapping on a transfer bubble reveals additional transaction details.

Under the hood, the CoreData models have been refactored so that `ChatTransaction` now serves as an abstract model for all in-chat transactions. The `TransfersProvider` has been updated accordingly to work with the revised data layer.

Localization has been updated for both English and Russian.
