This release of ADAMANT iOS introduces support for several new tokens, expanding the wallet's multi-exchange capabilities. Users can now hold and transact Bit-Z, Stably USD, Resfinex, KuCoin, and Binance tokens directly within the app.

The node list has been updated to reflect the current network infrastructure, improving connection reliability and transaction propagation. Additionally, this version includes general bugfixes to enhance overall stability and user experience.
