This release brings several improvements to the ADAMANT client. The login screen now includes a loading animation, and the address book is stored in the Key-Value Store (KVS) as requested in issue #28. Users can now vote for delegates and view a delegate list with forging information. Language constants gracefully fall back to English when a localized string is missing. The ICO link has been removed.

Ethereum transaction confirmations count is now accurate, and ETH transaction rejection behavior has been corrected. The rendering of ADM transaction fees was fixed, and language files have been updated.
