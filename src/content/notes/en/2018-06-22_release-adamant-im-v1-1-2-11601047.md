This release introduces finished ETH sending functionality and adds support for AIP-5. Users can now send tokens directly from the chat window. When a user chooses to store a passphrase on device, it is stored encoded. The release also adds the ability to save a passphrase QR code, and a Travis build script has been added.

Several fixes are included, covering icon changes and small style updates.
