This release of ADAMANT iOS enhances transaction processing mechanics and implements a secure coin transfer algorithm. Performance when working with large messages has been improved, and FLOKI token support has been added. The transaction list now features a new design, and users can save texts and replies in input fields as message drafts. The app now presents the coin nodes version. Several UI-related bugs have been resolved and localizations have been fixed.
