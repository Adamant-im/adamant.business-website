Blockchain messengers inherently trade speed for security and anonymity, as message delivery is typically constrained by block generation frequency. ADAMANT has addressed this limitation, reducing message delivery times to between 0 and 2 seconds.

This performance gain was achieved through two primary optimizations. Initially, ADAMANT implemented client-node socket connections to facilitate peer-to-peer communication speeds. Subsequently, node configurations were updated to exchange transactions more rapidly, with a specific focus on the propagation of unconfirmed transactions across the network. As a result, users connected to different network nodes now receive messages almost instantly, bringing the communication experience in line with classic centralized instant messengers.
