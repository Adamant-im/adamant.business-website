ADAMANT iOS version 1.3 introduces Lisk wallet support alongside user avatars for a more personalized messaging experience. Users can now send ADM transfers with comments and enjoy full iPad support for broader device compatibility.

This release also adds Markdown rendering for richer message formatting. Message sending has been improved for reliability, and QR codes have been enhanced for better scanning performance. Overall stability improvements and bug fixes round out the update.
