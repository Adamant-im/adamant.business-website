This release introduces the ability to save draft replies and messages in the chat, adds Simplified Chinese language support, and integrates the STORJ token into the wallet.

Several bugs have been addressed in this version. The wallet's left chevron is now clickable, and its colors have been adjusted for better visibility. A pluralization issue with addresses in the Russian locale has also been fixed.

On the technical side, dependencies including Vue, Vuetify, and Electron have been upgraded to newer versions. The README has been updated with shields, and the default locale can now be configured using an environment variable.

### Breaking changes

There are no breaking changes in this release.
