ADAMANT Messenger v3.1.0 introduces standalone desktop applications for macOS, Windows, and Linux. This release reduces network requests when using socket connections and updates project dependencies.

The application can be built from source via the [ADAMANT repository](https://github.com/Adamant-im/adamant-im). Release details are available on the [v3.1.0 release page](https://github.com/Adamant-im/adamant-im/releases/tag/v3.1.0).
