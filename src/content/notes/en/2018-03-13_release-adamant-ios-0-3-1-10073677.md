This maintenance release introduces a notification sound feature and updates the Russian localization. Several bug fixes are included, addressing an issue with account balance updates and a problem where the first message in a chat was not being marked as unread. As the release notes wryly observe, many bugs were fixed and introduced.

### Breaking changes

Account balances are now represented using the Decimal type, correcting the previous implementation. Any code or integrations relying on the former balance type will need to be updated accordingly.
