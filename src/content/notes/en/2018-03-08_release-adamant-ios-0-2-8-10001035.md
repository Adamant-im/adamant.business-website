This release of ADAMANT iOS introduces several user authentication and interface improvements. Users can now stay logged in across sessions, with PIN code authorization providing an additional security layer. Biometric authentication is supported through both Touch ID and Face ID.

The navigation bars have been updated to use large titles in the iOS 11 style, offering a more modern and readable interface. General code cleanups were also performed to improve maintainability.

### Breaking changes

No breaking changes were introduced in this release.
