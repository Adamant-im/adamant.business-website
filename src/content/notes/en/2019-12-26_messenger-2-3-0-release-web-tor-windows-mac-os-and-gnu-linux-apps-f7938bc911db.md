ADAMANT Messenger v2.3.0 is now available with builds for Web, Tor, Windows, macOS, and GNU/Linux. The release introduces Bitcoin support alongside several improvements to transaction status handling and bug fixes.

Transaction processing has been refined across multiple chains. The confirmation count on transaction detail screens now updates dynamically, and a new Registered status for transactions allows status updates to be handled more effectively. Ethereum transaction statuses, which were previously incorrect, have been fixed. Additionally, processing of links to msg.adamant.im chats has been corrected.

The current codebase resides in the master branch of the [ADAMANT Messenger GitHub repository](https://github.com/Adamant-im/adamant-im). Developers interested in contributing to the open-source ADAMANT project can find the latest source there.
