This release of ADAMANT for iOS introduces support for the Klayr coin (KLY), which represents an evolution from the Lisk L1 blockchain. Users can now manage KLY alongside their existing assets within the ADAMANT wallet application on iOS.

In addition to the new coin listing, this version includes various bug fixes and stability improvements to enhance the overall reliability and user experience of the application.
