This release introduces QR code generation for passphrases, enabling fast login by scanning a printed code that you can keep in a secure physical location. Users can also generate and share QR codes containing their ADAMANT address, and scan a friend's code directly from the New Chat window to start a conversation quickly.

The app icon has been improved, and the notification style has been switched to light. Additionally, this release includes numerous internal fixes and cleanups.
