This release introduces a reply and quote feature, allowing users to swipe left or long-press a message to reply. A new Contribute section provides options to support the future of the messaging platform. Crash analytics have been added to help improve the project, though they remain disabled by default.

Several fixes and improvements target cryptocurrency wallets and transactions. A BTC wallet issue has been resolved, ensuring BC1 (SegWit) addresses work correctly. Gas limits for ETH and ERC20 transactions have been fixed, and a new switch allows users to increase gas for ETH, ERC20, and BTC transactions. Address validation for coins has been improved to support the "amount" parameter for QR codes.

The update also addresses user interface issues. A scrolling bug in messages has been fixed, and users can now start a new chat directly from the search field. Design bugs affecting macOS, iPad, and iPhone have been resolved. Network stability and performance are enhanced with the addition of new nodes, which can be refreshed by tapping Reset on the Nodes screen. Finally, StakeCube has been added as a new exchange option in the "Buy ADM" section.
