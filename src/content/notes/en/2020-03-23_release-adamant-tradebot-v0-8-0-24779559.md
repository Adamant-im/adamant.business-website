This release of ADAMANT Tradebot introduces support for the Bit-Z exchange, expanding the range of platforms available for automated trading. Users can now set output precision for trading pairs via the new coin1Decimals and coin2Decimals configuration options. The orders database has been updated to include the exchange field, improving record keeping across supported platforms. A new /params command displays the current trading settings, making it easier to review active configuration. This release also includes several bug fixes.
