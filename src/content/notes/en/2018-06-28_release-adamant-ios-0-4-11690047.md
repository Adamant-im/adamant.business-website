This release of ADAMANT for iOS introduces push notifications, keeping users informed of incoming messages and updates. The Node editor has been fixed to resolve a "Checking..." hang that could occur during validation, and a bug preventing nodes from being saved has been corrected.

System chatrooms are now hidden to avoid duplication, and avatars for system chats have been updated.

No breaking changes are introduced in this release.
