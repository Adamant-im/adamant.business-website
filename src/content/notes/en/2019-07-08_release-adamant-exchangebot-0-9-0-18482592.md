This release of ADAMANT Exchangebot introduces exchange functionality between ADM and ETH tokens. The update also adds help commands to assist users in navigating available operations, along with improvements to message handling for more reliable communication.

Looking ahead, the project plans to introduce a new user bounty system and exchange statistics in future releases.
