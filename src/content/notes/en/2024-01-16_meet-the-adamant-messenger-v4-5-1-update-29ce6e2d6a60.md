ADAMANT Messenger v4.5.1 introduces draft message persistence, Simplified Chinese localization, STORJ token support in wallets and chats, and several dependency upgrades and bug fixes.

### Draft messages

Users can now compose a message and save it as a draft for later sending. This prevents loss of partially written text when a message cannot be completed in one session, improving the messaging workflow for intermittent use.

![Meet the ADAMANT Messenger v4.5.1 update](/images/engineering-notes/medium/29ce6e2d6a60/003-0-k0dduujxur7pnr9.webp)

### Simplified Chinese language support

The application now supports Simplified Chinese, extending localization coverage for Chinese-speaking users. This update also includes corrections to pluralization errors in the Russian locale for transaction-related strings.

![Meet the ADAMANT Messenger v4.5.1 update](/images/engineering-notes/medium/29ce6e2d6a60/002-0-l2314o6c9trohor6.webp)

### STORJ token integration

STORJ is now available as a supported wallet asset, allowing users to transfer STORJ directly within chats alongside previously supported cryptocurrencies.

![Meet the ADAMANT Messenger v4.5.1 update](/images/engineering-notes/medium/29ce6e2d6a60/004-0-n9t5myjiuilzj6-d.webp)

### Bug fixes and technical updates

The wallet left chevron navigation issue has been resolved. Underlying libraries including Vue, Vuetify, and Electron were upgraded, and the README and internationalization configurations received updates.

The release is available on GitHub, where the application can be built from source. ADAMANT Messenger is a Progressive Web Application targeting Web, Tor, Android, Windows, Linux, and macOS.
