## Hotfix release

This emergency release `v4.11.1` addresses a critical production bug discovered after the previous rollout, specifically fixing BTC, DOGE, and DASH transfers. The application version has been bumped to `4.11.1` across all release artifacts, and in-chat coin transaction statuses have been improved.

The release also includes synchronization of the Massa DeWeb workflow fix from master to dev in the CI/CD pipeline, as well as syncing Android version metadata fixes from master to dev. Additionally, it resolves send funds and crypto transfer status regressions in the PWA.
