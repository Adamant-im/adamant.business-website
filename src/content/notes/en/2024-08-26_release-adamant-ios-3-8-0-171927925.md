The 3.8.0 release of ADAMANT for iOS introduces comprehensive file sharing enhancements. Users can now send any file type, with media files displayed separately in chat. Drag-and-drop functionality is available on both iOS and macOS, and images can be pasted directly from the clipboard.

File preview and management capabilities have been expanded. Users can preview files before sending, and selected files are automatically saved even when exiting the chat. File upload and download progress tracking provides visibility into transfer status.

Advanced file handling allows users to view file content directly within the app. Previews or full media are downloaded automatically based on user settings, which can be configured for contacts, everyone, or no one. Storage management includes a space usage overview and cleaning options, and all files can be downloaded in bulk with a single click on a side button.

Security and storage options include the ability to locally save files with encryption based on user settings.
