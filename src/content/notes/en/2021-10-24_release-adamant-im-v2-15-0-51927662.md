This release fixes an issue with sending the full DOGE balance and adds an ERC20 mark for supported tokens. Dependencies have been updated. The Lisk explorer has been switched from testnet to mainnet. An Exchanger has been added to the Buy-Sell ADM feature, while Bit-Z has been removed from it.

### Breaking changes

The KCS and BZ wallets have been removed from the application. These wallet types will no longer be available after upgrading.
