This release of ADAMANT Messenger introduces quote and reply messages in chats according to AIP-16, along with a pull-down gesture to update wallet balances and a "Mark all as read" button for chats. The Buy Tokens dialog now includes the StakeCube exchange, and QR code generation shows a spinner while rendering. Multiline message input is capped at ten rows, and an ellipsis indicator appears while wallet balances load. A script for generating wallets from the `adamant-wallets` repository has been added, and delegate vote icons have been restyled.

Several bugs have been addressed. The delegates page header separator is fixed for the light theme, and the nodes table now uses the regular text color in light theme. Node status icon colors have been updated, and the loading chats animation is now centered. The Electron notarize script has been renamed from `notarize.js` to `notarize.cjs`, and ServiceWorker has been disabled in the Electron app. Push notifications and the chat list now use the Markdown rendering option. ERC20 coins no longer break after a wallet refresh, and the nodes list updates correctly. The Android navigation menu no longer overflows the chat form, and `isScrolledBottom()` now handles devices that use subpixels correctly.

On the technical debt front, `electron-builder` has been upgraded from v23 to v24, and the NodeJS version has been upgraded from v16 to v18. npm install security issues have been resolved, and notarization now uses `notarytool`.
