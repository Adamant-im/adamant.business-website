This release introduces support for a wide range of ERC20 tokens, including Binance Coin (BNB), Binance USD (BUSD), Dai (DAI), Ethereum Name Service (ENS), Holo (HOT), Injective (INJ), Chainlink (LINK), Decentraland (MANA), Polygon (MATIC), PAX Gold (PAXG), Quant (QNT), Ren (REN), SKALE (SKL), Status (SNT), Synthetix Network (SNX), TrueUSD (TUSD), Uniswap (UNI), USD Coin (USDC), PAX Dollar (USDP), Stably USD (USDS), Tether (USDT), WOO Network (WOO), and Onyxcoin (XCN). Users can now manage visible coins in a wallet, and ETH and ERC20 transfers have been optimized for lower fees and smoother operation.

The update also brings improved in-chat transaction handling, faster transaction processing, and a cleaner, faster UI for messages. ADM addresses are now rendered as markdown hyperlinks. macOS support has been improved with push notifications and CPU usage optimization. Additionally, the app features a more stable connection with health checks, better chat updating when returning from the background, and overall performance improvements alongside various minor bug fixes.

### Breaking changes

The application now requires iOS 15 or later to function.
