ADAMANT Messenger v4.1.2 is a minor maintenance release focused on bug fixes and general stability improvements. The update does not introduce new features but addresses issues that affect the overall reliability of the messenger across supported platforms.

The application is a Progressive Web Application available for Web, Tor, Android, Windows, Linux, and macOS. Source code is published on GitHub under the ADAMANT project repository, allowing users to build the app from source to independently verify its privacy and security properties.

Release notes and build instructions are available in the [v4.1.2 release tag](https://github.com/Adamant-im/adamant-im/releases/tag/v4.1.2) on GitHub.
