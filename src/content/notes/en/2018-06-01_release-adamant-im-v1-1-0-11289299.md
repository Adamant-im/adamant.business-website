This minor version update introduces a significant new feature: all ADM users now have an ETH account automatically generated for them. Users can utilize this account to transfer ETH funds. This feature is expected to evolve further in future releases. Credit to @MaaKut for implementing this functionality.

Additional changes include adding a Free ADM Tokens link, the ability to start dialogs directly from transaction history, and the ability to send ETH tokens. Language messages have also been updated as part of this release.
