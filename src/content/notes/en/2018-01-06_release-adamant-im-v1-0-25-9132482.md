This release adds tooltips to buttons on the desktop interface. Several fixes are included: tab titles now update correctly, the chat scrolls to the last message as expected (resolving issue #8), the login screen title is localized for Russian, and the language selector on the Options screen renders correctly. New translations are available for Turkish, Filipino, Hindi, and Traditional Chinese.
