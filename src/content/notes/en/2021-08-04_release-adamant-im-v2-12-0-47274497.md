This release introduces Dash InstantSend support and refines the status logic across the application. A static chat for the ADM Bounty bot has been added, and non-English BIP39 wordlists have been removed to streamline the codebase.

Startup performance is now roughly six times faster thanks to seed caching. Dependencies and linters have been updated, and Ethereum libraries have been upgraded. The Atomars exchange link has been removed. Several bugfixes are also included.
