This release introduces quick reactions in chat, along with haptic vibration feedback when replying to, long-pressing, or swiping messages. The wallet gains support for the Coinstore exchange as well as two new digital assets: XCN coin and Verse token. Login form inputs now include a toggle to show or hide the passphrase.

Several bug fixes address issues across the application. Clicking a reply message previously loaded the entire chat history, which has been corrected. The "stay logged in" setting now functions as expected. A problem causing the app to send infinite transaction requests has been resolved, and unsupported characters in VueI18n have been removed.

On the technical side, the API client is now generated with TypeScript types from adamant-schema, and TypeScript types have been added to adamant-api. The marked library, Electron, and other dependencies have been updated to their latest versions.
