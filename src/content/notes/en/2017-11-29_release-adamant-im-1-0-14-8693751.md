This release introduces several style changes across the application to refine the user interface. Compatibility fixes have been applied to resolve issues experienced by Firefox users. Notification support has been improved with the addition of sound alerts and title bar notifications. The messenger now also handles ICO and pre-ICO messages.

### Breaking changes

No breaking changes are introduced in this release.
