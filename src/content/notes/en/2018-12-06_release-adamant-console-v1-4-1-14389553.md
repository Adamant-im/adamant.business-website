This release of ADAMANT Console introduces a new `client version` command and a `node` commands subgroup containing `node height` and `node version` commands. On the JSON-RPC side, the methods `clientVersion`, `nodeHeight`, and `nodeVersion` have been added to support these new console features programmatically.
