This release introduces a dedicated service nodes tab with staking support and health checks for indexers. The Electron build now produces separate binaries for Apple M1 and Intel chips. Chat reactions are animated, tables have been restyled, and wallet balances are formatted on the account tab. Coinmarketcap and Coingecko links have been added to exchanges.

On the chat side, a nodes offline dialog has been added, and an infinite spinner issue in ADAMANT chats has been resolved. Login fixes address camera switching between front and back cameras and a no camera access bug. A missing "Message rejected" translation has been added.

Technical debt work includes saving only app binary artifacts in the Electron build, rewriting the transaction fetching mixin into hooks using VueQuery, removing an unknown Vuex action, and creating the `.well-known` directory after build.
