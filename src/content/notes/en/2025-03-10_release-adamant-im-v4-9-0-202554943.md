This release of ADAMANT Messenger introduces file and image sharing in chat, along with a faster first-message experience. Node management gains the ability to check or uncheck all nodes at once, updated health-check thresholds, and fixes to ADM node status reporting. The login scenario has been refreshed, and line separators are now treated as line feeds for better message formatting.

On the wallet side, users can now open a wallet directly in a blockchain explorer, and transaction fees are estimated in fiat currency. Several wallet bugs were addressed, including ERC20 transaction fees now displayed in ETH, corrected transaction statuses for USDT transfers, repaired transaction lists, proper ETH transfer fee display, and a fix for KLY amount handling. Snackbar texts when sending coins were also updated.

Notable bug fixes include preventing data from a previous account from appearing in chat, resolving an infinite request queue when there is no connection, and a range of smaller UI and miscellaneous fixes across the application.

Technical debt was reduced by migrating icons from fonts to SVG, updating dependencies, removing TypeScript file extensions, and adding a CODEOWNERS file.
