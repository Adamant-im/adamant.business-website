ADAMANT Messenger v2.10.0 introduces support for LSK tokens, enabling users to store and transfer LSK within chats across Web, Tor, Android, and iOS platforms. This addition joins existing support for ADM, Bitcoin, Ethereum, Doge, Dash, BNB, KSC, RES, and USDS.

The ADAMANT wallet is fully open source, allowing users to build from source rather than relying on closed-source alternatives. Private keys never leave the user's device, and a single passphrase grants access to all integrated crypto wallets. Users can also export private keys for backup purposes. A standout feature of ADAMANT is the ability to transfer cryptocurrencies directly within messenger chats.

The ADAMANT blockchain itself is built on Lisk code, modified with specific block times, fees, and a chatrooms API for decentralized messaging. Future plans include supporting the new Lisk SDK, new address types, and various Lisk sidechains within the ADAMANT ecosystem. There is also consideration for creating a Wrapped ADM token on the Lisk platform to expand services and decentralized trading options.

ADM functions as a utility token, with its value underpinned by fees for message and data transfers designed to cover infrastructure costs for anonymous and secure transmission. ADAMANT employs a delegated Proof of Stake (dPoS) consensus model with improvements termed "Fair dPoS." This model divides the vote weight of each wallet by its number of votes, reducing the influence of large wallets and lowering the barrier to entry for forging. Additionally, missed blocks negatively impact a delegate's productivity, which in turn affects their vote weight.

Lisk operates as a blockchain application platform utilizing a JavaScript/TypeScript SDK and a Node.js core. It relies on a PoS/dPoS consensus mechanism. Compared to Ethereum, Lisk offers a broader developer audience due to JavaScript's popularity, easier onboarding for beginners, lower transaction fees, and energy-efficient consensus. The Lisk SDK is currently in beta, and a security audit has been published.
