This release introduces a universal macOS build and migrates the project to a new ESLint configuration. A release notes file has been added to the repository. The health-check system has been improved: an "Updating…" status is now shown before HTTP request attempts, initial and subsequent health checks follow a refined request algorithm, and the system checks `alt_ip` when a web domain is not accessible. Background health-check recovery after system sleep has been fixed, and incorrect node status reporting has been resolved. The wallets screen has been updated, along with the script for adamant-wallets, and a dev test screen has been created for that tooling.

Messages can now be joined into groups with a bubble layout. Several style updates have been applied across the app, including message input styles, export keys form layout, and general visual consistency improvements. The Welcome to ADAMANT chat now includes a disabled input field. Dev-only Playwright checks have been added as part of a broader effort to unify styles and modernize the UI/UX.

Numerous bug fixes are included. Dates in chats now refresh correctly. False sync indicators no longer appear when the nodes screen is opened. Incorrect transaction amounts for BTC, DOGE, and DASH when sending to one's own address have been fixed. Markup that broke when replying to a message containing a long link has been corrected. A null reference error related to `maintainScrollPosition` has been resolved. Static chats are no longer read-only after re-login. Transaction fees for ERC20 coins are now consistent between PWA and iOS. Transaction navigation and localized UI regressions across PWA screens have been fixed. Legacy persisted wallet symbols are now normalized to prevent a crash on the `/options` route. The released APK name has been corrected, and source code issues detected by ESLint 9 have been resolved. Wrong commit activity in the repository has been addressed.

General dependencies and tooling have been upgraded with build warnings removed. Node.js has been upgraded from version 20 to 22. An org-wide Release Drafter and reusable workflow have been implemented. A GitHub Actions workflow and Husky postinstall git hook for ESLint have been added. Documentation has been added with AI agent operating guidelines for the PWA, and the README and release metadata have been refreshed.

### Breaking changes

Klayr (KLY) support has been removed in full from the wallet. Users holding KLY assets should be aware that this cryptocurrency is no longer supported in version 4.11.0 and beyond.
