ADAMANT Messenger for iOS version 2.0.0 now supports storing and transferring Binance Coin (BNB), Bit-Z token (BZ), Stably USD (USDS), Resfinex token (RES), and KuCoin Shares (KCS) directly in chats. This update aligns the iOS application's functionality with the existing Android, Web, and Tor clients.

The ADAMANT wallet is fully open-source, allowing users to build the application from source and verify its security, unlike many closed-source alternatives. Private keys are stored locally and never leave the user's device. A single passphrase grants access to all integrated crypto wallets, and users can export private keys for backups.

Transferring cryptocurrencies directly within chat conversations remains a core feature of the ADAMANT platform, offering a streamlined method for sending funds between users.

Beyond the newly added token support, this release includes an updated node list and general bug fixes.
