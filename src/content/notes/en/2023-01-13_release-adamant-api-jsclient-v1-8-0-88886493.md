This release of the ADAMANT API JS client introduces a new `checkNodeTimeout` parameter, allowing developers to configure the timeout duration used when checking node availability. This addition provides greater control over how the client verifies connectivity to ADAMANT nodes, helping applications handle slow or unresponsive nodes more gracefully.
