This release focuses on test improvements and dependency maintenance for the ADAMANT codebase. The node test suite has been fixed, and new API tests have been added to broaden coverage. A new `POST /api/transactions` endpoint is introduced as part of this release.

The sodium dependency has been updated to resolve compatibility problems encountered with recent Node.js versions, ensuring smoother operation on current runtime environments.
