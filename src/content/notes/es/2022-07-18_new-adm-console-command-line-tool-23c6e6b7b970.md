La Consola ADAMANT, una herramienta de línea de comandos para interactuar con la blockchain ADAMANT, ha sido completamente rediseñada en la versión 2.0.0. Junto con las API existentes de JS y Node, la Consola permite a los usuarios crear cuentas, transferir tokens, enviar mensajes y registrar delegados.

La herramienta funciona tanto como una interfaz de línea de comandos independiente como una biblioteca NodeJS, lo que la hace adecuada para su integración en scripts y aplicaciones externas. Los desarrolladores pueden aprovechar la Consola para construir aplicaciones descentralizadas completas, como mensajeros o sistemas de autenticación de dos factores basados en blockchain, aprovechando sus nuevas capacidades de descifrado de mensajes.

El lanzamiento de la versión 2.0.0 representa una refactorización completa de la base de código, que ahora utiliza el paquete `adamant-api`. Introduce un modo interactivo, ayuda integrada para comandos, dependencias actualizadas y varias correcciones de errores.
