ADAMANT ha lanzado la versión 4.11.0 de su Messenger PWA, ahora disponible en Web, Tor, Android, Linux, Windows y macOS. La versión se centra en mejoras de confiabilidad, pulido y madurez, en lugar de introducir funciones nuevas importantes.

La última compilación está etiquetada como v4.11.1 en GitHub, lo que indica un parche rápido posterior a la publicación inicial de la versión 4.11.0. Los usuarios que ejecuten el PWA deben actualizar a la etiqueta más reciente para aplicar correcciones posteriores al lanzamiento.

ADAMANT Messenger es una aplicación de mensajería de código abierto basada en blockchain. La versión PWA (Progressive Web App) permite a los usuarios ejecutar el mensajero en un navegador o instalarlo como una aplicación similar a nativa en plataformas de escritorio y móviles compatibles, incluido Tor para usuarios que necesitan acceso mediante enrutamiento onion.
