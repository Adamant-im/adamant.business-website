Esta versión añade soporte para Dash con la funcionalidad InstantSend y compatibilidad con el hardfork London de Ethereum. Los usuarios ya no necesitan especificar la moneda en los comentarios.

La base de código ha sido completamente refactorizada, incluyendo linting y varias optimizaciones. La aplicación ahora utiliza ADAMANT API 1.1.0, y todas las dependencias han sido actualizadas.
