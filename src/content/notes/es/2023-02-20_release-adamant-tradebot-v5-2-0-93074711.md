Esta versión se centra en mejoras de mantenimiento y estabilidad para ADAMANT Tradebot. Las dependencias se han actualizado a sus últimas versiones, y se ha realizado una refactorización interna para mejorar la calidad del código. Esta versión incluye varias correcciones de errores.

El conector de la API de P2PB2B se ha reescrito para mejorar la fiabilidad y la mantenibilidad.
