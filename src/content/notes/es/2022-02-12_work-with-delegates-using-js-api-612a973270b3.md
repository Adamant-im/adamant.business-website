La biblioteca ADAMANT JavaScript API v1.4.0 introduce dos nuevos métodos para la gestión de delegados: `delegateNew()` para registrar un nuevo delegado y `voteForDelegate()` para emitir votos por delegados existentes. Esta versión también incluye actualizaciones rutinarias de dependencias. Consulte la [API specification](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification) para obtener detalles sobre el uso.
