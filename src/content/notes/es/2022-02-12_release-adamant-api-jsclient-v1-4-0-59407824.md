Esta versión introduce dos nuevos métodos al cliente JS de la API ADAMANT: `delegateNew()` y `voteForDelegate()`. Estas adiciones amplían la funcionalidad del cliente relacionada con delegados. Además, las dependencias del proyecto se han actualizado a sus últimas versiones.
