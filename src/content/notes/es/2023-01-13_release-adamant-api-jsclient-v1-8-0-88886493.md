Esta versión del cliente JS de ADAMANT API introduce un nuevo parámetro `checkNodeTimeout`, que permite a los desarrolladores configurar la duración del tiempo de espera utilizada al verificar la disponibilidad del nodo. Esta adición proporciona un mayor control sobre cómo el cliente verifica la conectividad con los nodos de ADAMANT, ayudando a las aplicaciones a manejar nodos lentos o no responsivos de manera más eficiente.
