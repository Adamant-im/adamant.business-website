Esta versión de ADAMANT Messenger introduce soporte para tratar un salto de línea simple como un salto de línea en los mensajes, de acuerdo con GitHub Flavored Markdown. Anteriormente, los saltos de línea simples no se renderizaban como saltos de línea; este cambio alinea el formato de los mensajes con las convenciones comunes de markdown.

Se ha aplicado una corrección al contador de mensajes no leídos, resolviendo un problema por el cual el conteo podría mostrarse incorrectamente. Además, se ha implementado una solución temporal para abordar problemas de compatibilidad en iOS 11, mejorando la estabilidad para los usuarios de esa plataforma. También se han realizado pequeños ajustes en la lógica de almacenamiento de estado para mejorar aún más la fiabilidad.

### Cambios importantes

No hay cambios importantes en esta versión.
