La biblioteca ADAMANT JavaScript API ahora soporta claves criptográficas Lisk junto con una refactorización completa del código base. La versión v1.5.0 reemplaza el cliente HTTP anterior con Axios, mejorando el manejo de solicitudes y la mantenibilidad en toda la biblioteca. Las directrices de contribución y el estilo de código también se han actualizado para agilizar la colaboración.

La documentación completa de la API está disponible en la [ADAMANT API wiki](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification), y los detalles de la versión se encuentran en la [página de lanzamiento v1.5.0](https://github.com/Adamant-im/adamant-api-jsclient/releases/tag/v1.5.0).
