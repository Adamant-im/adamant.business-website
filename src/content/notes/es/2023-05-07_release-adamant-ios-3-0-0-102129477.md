Esta versión introduce soporte para una amplia gama de tokens ERC20, incluyendo Binance Coin (BNB), Binance USD (BUSD), Dai (DAI), Ethereum Name Service (ENS), Holo (HOT), Injective (INJ), Chainlink (LINK), Decentraland (MANA), Polygon (MATIC), PAX Gold (PAXG), Quant (QNT), Ren (REN), SKALE (SKL), Status (SNT), Synthetix Network (SNX), TrueUSD (TUSD), Uniswap (UNI), USD Coin (USDC), PAX Dollar (USDP), Stably USD (USDS), Tether (USDT), WOO Network (WOO) y Onyxcoin (XCN). Los usuarios ahora pueden gestionar las monedas visibles en una billetera, y las transferencias de ETH y ERC20 se han optimizado para reducir tarifas y mejorar el funcionamiento.

La actualización también trae una mejor gestión de transacciones dentro del chat, procesamiento más rápido de transacciones y una interfaz de mensajes más limpia y rápida. Las direcciones ADM ahora se muestran como enlaces markdown. El soporte para macOS se ha mejorado con notificaciones push y optimización del uso de CPU. Además, la aplicación incluye una conexión más estable con comprobaciones de salud, una actualización mejorada del chat al regresar desde segundo plano y mejoras generales de rendimiento junto con varios ajustes menores de errores.

### Cambios importantes

La aplicación ahora requiere iOS 15 o posterior para funcionar.
