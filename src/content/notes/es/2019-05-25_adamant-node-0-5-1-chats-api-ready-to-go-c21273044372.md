ADAMANT Node versión 0.5.1 ya está disponible en GitHub, completando el trabajo de la API de Chats anunciado por primera vez en enero. La función principal es el punto final `/api/chatrooms`, que permite recuperar contactos y chats significativamente más rápido que la API anterior. Los desarrolladores deben consultar la [especificación AIP-14](https://aips.adamant.im/AIPS/aip-14) para obtener detalles de implementación.

Esta actualización no es obligatoria para todos los propietarios de nodos. Sin embargo, los nodos que necesiten conectarse con aplicaciones de mensajería deben actualizar a la última versión para aprovechar la nueva API. El equipo de ADAMANT para Android ya ha demostrado mejoras medibles de velocidad utilizando el nuevo punto final, y se esperan ganancias similares en otras aplicaciones ADAMANT a medida que la adopten.
