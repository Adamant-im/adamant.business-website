ADAMANT InfoServices se ha actualizado a la v3.5.0, ofreciendo una API combinada de tasas de divisas y criptomonedas. El servicio agrega datos de varias fuentes principales, incluyendo MOEX, Currency-Api, ExchangeRates.host, CoinMarketCap, CryptoCompare y CoinGecko, luego cruza los resultados para detectar discrepancias y mejorar la fiabilidad.

Dado que se ejecuta en su propio servidor, el servicio mantiene los datos históricos de tasas bajo su control y permanece libre para cualquier uso. El proyecto es de código abierto y está alojado bajo la organización ADAMANT en GitHub.

La documentación de la API está disponible en el [InfoServices API wiki](https://github.com/Adamant-im/adamant-currencyinfo-services/wiki/InfoServices-API-documentation).
