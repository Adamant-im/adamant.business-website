Esta versión introduce una función de verificación de salud de la lista de nodos y reescribe los componentes internos del cliente ADM API para mejorar la confiabilidad.

Varias correcciones se incluyen en esta compilación. La barra inferior ya no desaparece durante el uso, y el error de mensajes vacíos reportado en el problema #139 ha sido resuelto. Los enlaces Markdown ahora se renderizan correctamente según se describe en el problema #141. Se han abordado los problemas relacionados con las votaciones y se ha corregido el script de compilación de Travis.
