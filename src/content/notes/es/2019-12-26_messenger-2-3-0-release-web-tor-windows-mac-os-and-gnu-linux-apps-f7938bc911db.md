ADAMANT Messenger v2.3.0 ya está disponible con versiones para Web, Tor, Windows, macOS y GNU/Linux. Esta versión introduce soporte para Bitcoin junto con varias mejoras en el manejo de estados de transacción y correcciones de errores.

El procesamiento de transacciones se ha perfeccionado en múltiples cadenas. El contador de confirmaciones en las pantallas de detalles de transacción ahora se actualiza dinámicamente, y un nuevo estado Registrado para transacciones permite gestionar las actualizaciones de estado de forma más eficaz. Se han corregido los estados de transacciones de Ethereum, que anteriormente eran incorrectos. Además, se ha corregido el procesamiento de enlaces a chats de msg.adamant.im.

La base de código actual se encuentra en la rama master del [repositorio de ADAMANT Messenger en GitHub](https://github.com/Adamant-im/adamant-im). Los desarrolladores interesados en contribuir al proyecto de código abierto ADAMANT pueden encontrar allí el código fuente más reciente.
