Esta versión se centra en mejoras de pruebas y mantenimiento de dependencias para la base de código ADAMANT. Se corrigió la suite de pruebas del nodo y se añadieron nuevas pruebas de API para ampliar la cobertura. Se introduce un nuevo endpoint `POST /api/transactions` como parte de esta versión.

La dependencia sodium se ha actualizado para resolver problemas de compatibilidad encontrados con versiones recientes de Node.js, asegurando un funcionamiento más fluido en entornos de ejecución actuales.
