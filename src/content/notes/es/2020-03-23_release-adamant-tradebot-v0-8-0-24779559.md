Esta versión del ADAMANT Tradebot introduce soporte para el intercambio Bit-Z, ampliando la gama de plataformas disponibles para el comercio automatizado. Los usuarios ahora pueden establecer la precisión de salida para los pares comerciales mediante las nuevas opciones de configuración coin1Decimals y coin2Decimals. La base de datos de órdenes se ha actualizado para incluir el campo exchange, mejorando el registro de operaciones en las plataformas compatibles. Un nuevo comando /params muestra la configuración comercial actual, facilitando la revisión de la configuración activa. Esta versión también incluye varias correcciones de errores.
