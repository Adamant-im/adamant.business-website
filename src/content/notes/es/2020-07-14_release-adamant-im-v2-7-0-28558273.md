ADAMANT Messenger v2.7.0 introduce la capacidad de exportar claves privadas para billeteras internas, dando a los usuarios mayor control sobre sus fondos. Esta versión también añade una función de propinas que ayuda a nuevas cuentas a obtener tokens gratuitos para comenzar a usar el mensajero.
