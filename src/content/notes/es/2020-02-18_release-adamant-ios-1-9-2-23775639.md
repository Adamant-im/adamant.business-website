Esta versión introduce chats estáticos para el bot Exchanger y Bet, junto con un nuevo popup para recibir tokens ADM gratuitos. Las billeteras ahora incluyen una función de actualización, y las pantallas de historial de billeteras muestran un mensaje de "No hay transacciones aún" cuando no hay transacciones, además de mostrar el año para transacciones antiguas. La aplicación ahora también verifica si la cuenta de un socio tiene una clave pública.

Esta versión incluye varias correcciones de errores. Se ha resuelto un problema de desplazamiento al enviar un mensaje, así como un mal funcionamiento cuando una cuenta tiene un nombre vacío. Se ha corregido un fallo que ocurría al emitir un segundo voto para un delegado, y se ha solucionado un bloqueo al volver a iniciar sesión.
