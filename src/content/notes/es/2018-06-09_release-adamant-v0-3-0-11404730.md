Esta versión añade soporte para Mensajes de Contenido Enriquecido según lo definido en [AIP 5](https://aips.adamant.im/AIPS/aip-5), permitiendo un formato de mensaje más rico dentro de la plataforma de mensajería ADAMANT. También se ha introducido soporte para Mensajes Señal según [AIP 6](https://aips.adamant.im/AIPS/aip-6). Además, esta versión corrige los filtros en la API de transacciones de estado (KVS).
