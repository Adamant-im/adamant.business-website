Esta versión de ETH Transactions Storage introduce índices opcionales adicionales que pueden usarse para optimizar el rendimiento de las consultas en casos de uso específicos. Estos índices no están habilitados de forma predeterminada, lo que permite a los usuarios aplicarlos selectivamente según sus requisitos individuales de carga de trabajo.

El README también se ha actualizado para reflejar las nuevas opciones de indexación y proporcionar una guía más clara sobre la configuración y el uso.
