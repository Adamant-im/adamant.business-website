### Mayor velocidad de API en la última versión para Android

Tras los anuncios anteriores sobre mejoras de velocidad de la API para el nodo ADAMANT, el equipo de desarrollo para Android ha lanzado una actualización que incluye una nueva API de chats.

Una comparación entre la API antigua y la actualizada muestra que tanto las operaciones de chat como el procedimiento de inicio de sesión se completan significativamente más rápido que antes. Se planean más optimizaciones.

El último archivo APK está disponible en la [página de lanzamientos de GitHub para Android](https://github.com/Adamant-im/adamant-android/releases/). Los usuarios existentes deben asegurarse de ejecutar la versión más reciente; si la aplicación no se actualiza automáticamente, desinstalen la versión anterior e instalen manualmente el nuevo APK.

Los comentarios y reportes de errores pueden enviarse a través del repositorio de GitHub. El equipo también está preparando un lanzamiento en Google Play.
