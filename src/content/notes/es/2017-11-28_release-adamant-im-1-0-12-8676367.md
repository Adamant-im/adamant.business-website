Esta versión incluye un nuevo estilo visual en la aplicación ADAMANT Messenger, junto con correcciones específicas para dispositivos Android. El flujo de cierre de sesión ahora incluye una redirección para mejorar la experiencia del usuario al finalizar una sesión.

El estado de las transacciones ahora se actualiza automáticamente, permitiendo a los usuarios ver el estado más reciente de sus transferencias sin necesidad de actualizar manualmente. Además, se ha mejorado el orden de los chats para ofrecer una organización más intuitiva de las conversaciones.
