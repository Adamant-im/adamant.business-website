ADAMANT Messenger v4.1.2 es una versión menor de mantenimiento centrada en correcciones de errores y mejoras generales de estabilidad. La actualización no introduce funciones nuevas, sino que resuelve problemas que afectan la fiabilidad general del mensajero en todas las plataformas compatibles.

La aplicación es una Progressive Web Application disponible para Web, Tor, Android, Windows, Linux y macOS. El código fuente se publica en GitHub bajo el repositorio del proyecto ADAMANT, lo que permite a los usuarios compilar la aplicación desde el código para verificar de forma independiente sus propiedades de privacidad y seguridad.

Las notas de la versión y las instrucciones de compilación están disponibles en la etiqueta de lanzamiento [v4.1.2](https://github.com/Adamant-im/adamant-im/releases/tag/v4.1.2) en GitHub.
