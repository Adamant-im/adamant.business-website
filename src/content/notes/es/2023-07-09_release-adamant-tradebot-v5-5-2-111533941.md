Esta versión aborda dos problemas específicos de exchanges y realiza mantenimiento rutinario. Se ha corregido la cancelación de órdenes para StakeCube, resolviendo un problema que podía impedir que las operaciones se cancelaran correctamente en esa plataforma. También se ha corregido la función getRates para Azbit, asegurando una recuperación confiable de las tasas para ese exchange.

Además, se han actualizado las dependencias a sus últimas versiones compatibles y se ha actualizado el archivo readme del proyecto para reflejar la información actual.
