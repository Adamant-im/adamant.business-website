ADAMANT v2.5.0 introduce varias mejoras y correcciones de errores en sus aplicaciones web, Tor y de escritorio. Como mensajero basado en blockchain, ADAMANT se beneficia de un mayor número de nodos de red para una mejor descentralización. La aplicación web ahora incluye nueve nodos, compuestos por tres tipos HTTP y seis HTTPS. Al usar una conexión HTTPS, solo están disponibles los nodos HTTPS, mientras que las aplicaciones de escritorio para Windows, macOS y Linux pueden acceder a los nueve nodos.

La actualización también perfecciona la lista de intercambios para comprar y vender ADM, eliminando IDCM y agregando CoinDeal, junto con enlaces actualizados para Resfinex y Bit-Z. El soporte de Markdown en ADAMANT se ha mejorado para mostrar correctamente fuentes monoespaciadas en bloques `code`.

![Mejoras en ADAMANT v2.5.0 para Web, Tor, Windows, Mac OS y Linux](/images/engineering-notes/medium/161563fe990b/002-0-sqpbhli5aps7yqjd.webp)

En la versión Tor del mensajero, se han corregido las conexiones WebSocket para garantizar una entrega más rápida de mensajes.

![Mejoras en ADAMANT v2.5.0 para Web, Tor, Windows, Mac OS y Linux](/images/engineering-notes/medium/161563fe990b/003-0-fblaod14ec32orwh.webp)

Mantenimientos adicionales en esta versión incluyen dependencias actualizadas, verificaciones del protocolo de nodos y correcciones para estados de nodos y conexiones de socket en hosts HTTP. La actualización también aborda nombres estáticos de chats, validación de direcciones pegadas en el flujo de "Iniciar un chat nuevo" y generación de códigos QR para direcciones ADM en Windows y otras aplicaciones. Se realizaron mejoras en la localización de los mensajes de error "no hay clave pública" y "no hay hash".
