Esta versión añade registros para el script de migración de la base de datos. El nodo ADAMANT ahora registrará mensajes antes y después de ejecutar el script de migración:

```
[inf] 2023-12-22 04:00:25 | Found 5 pending migrations. Start executing, this may take a while…
[inf] 2023-12-22 04:17:13 | Migrations have been successfully completed.
```

El valor por defecto de la propiedad `cors.origin` ha cambiado de reflejar el origen de la solicitud (`true`) a `*`. Consulte la documentación para obtener más información sobre la configuración de CORS en el nodo ADAMANT.

```json5
{
  "cors": {
    "origin": "*" // default
  }
}
```

El archivo `config.json` se ha añadido a `.gitignore`. Ahora los usuarios deben crear el archivo `config.json` manualmente o copiar `config.default.json` para iniciar el nodo ADAMANT.

```
cp config.default.json config.json
```

### Cambios importantes

El valor predeterminado del encabezado CORS para `Access-Control-Allow-Origin` ha cambiado de reflejar el origen de la solicitud a `*`. Los usuarios que dependan del comportamiento anterior de reflejo del origen deben actualizar su archivo `config.json` en consecuencia. Además, `config.json` ahora es ignorado por git, por lo que los usuarios deben crear o copiar este archivo manualmente antes de iniciar el nodo.
