ADAMANT iOS versión 1.3 introduce soporte para el monedero Lisk junto con avatares de usuario para una experiencia de mensajería más personalizada. Los usuarios ahora pueden enviar transferencias ADM con comentarios y disfrutar de soporte completo para iPad, ampliando la compatibilidad con dispositivos.

Esta versión también añade renderizado de Markdown para un formato de mensajes más rico. Se han mejorado el envío de mensajes para mayor fiabilidad y los códigos QR para un mejor rendimiento de escaneo. Mejoras generales de estabilidad y correcciones de errores completan esta actualización.
