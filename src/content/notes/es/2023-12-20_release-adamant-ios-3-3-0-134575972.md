Esta versión de la aplicación ADAMANT iOS introduce varias mejoras en el procesamiento de transacciones y la experiencia de chat.

El mensajero ahora realiza una verificación de salud de las monedas, seleccionando solo nodos operativos para conectarse. Los usuarios pueden elegir entre conectarse a nodos aleatorios o preferir el nodo más rápido. La lista de transacciones también se ha mejorado: la aplicación almacena las transacciones localmente y ya no recarga toda la lista cada vez que se abre.

En la pantalla de chat, un toque corto sobre el nombre de un destinatario abre el código QR de su dirección, mientras que un toque prolongado permite cambiarle el nombre.

Se incluyen correcciones adicionales para mejorar la experiencia general.
