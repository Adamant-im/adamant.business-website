Esta versión introduce una nueva ventana de lista de transacciones junto con múltiples cambios de estilo y mejoras de refactorización interna en todo el código base.

Varios errores se han resuelto en esta versión. Se ha corregido el error que ocurría al enviar ETH con una dirección de contacto con nombre modificado. Abrir un chat sin mensajes desde la lista de transacciones ahora funciona correctamente. Se ha solucionado el comportamiento de desplazamiento automático hacia abajo al abrir un chat anterior. Además, el bloque de tokens ADM gratuitos en la pestaña Cartera ahora está oculto cuando el saldo es mayor que cero.
