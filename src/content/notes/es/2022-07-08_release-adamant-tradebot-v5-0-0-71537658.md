Esta versión se centra en la refactorización interna y las actualizaciones de dependencias para ADAMANT Tradebot. Como parte del trabajo de mantenimiento, se ha añadido un archivo `config.default.json` para proporcionar una configuración básica más clara para los operadores.

Se introduce una nueva opción para el creador de mercado, `mm_orderBookMaxOrderPercent`, que permite un control más preciso sobre la participación en el libro de órdenes.

### Cambios importantes
No se introducen cambios importantes en esta versión.
