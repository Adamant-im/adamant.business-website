Esta versión añade soporte para Resfinex Token (RES) y actualiza el nombre y logo de Stably Dollar (USDS). Se ajustó la altura del menú Enviar Token para una mejor usabilidad, y se actualizó el diálogo Comprar y Vender tokens.

Ahora está disponible una conexión socket cliente-nodo, junto con un nuevo estado "En blockchain" para los mensajes. Se añadió validación de direcciones BTC y una corrección para la validación de direcciones pegadas desde el portapapeles. Una corrección adicional resuelve un problema con montos negativos al enviar.

### Cambios importantes

No se introducen cambios importantes en esta versión.
