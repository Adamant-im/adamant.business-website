Esta versión del ADAMANT Tradebot introduce soporte para archivos de configuración `.jsonc`, lo que permite comentarios y comas finales en la configuración del bot para facilitar el mantenimiento. El sistema de notificaciones se ha ampliado para enviar mensajes a varios canales simultáneamente, y un nuevo mecanismo de notificación con prioridad garantiza que las alertas críticas lleguen rápidamente a los operadores.

Ahora están disponibles los alias de comandos, lo que permite a los usuarios definir nombres abreviados para comandos utilizados con frecuencia. Se han añadido confirmaciones interactivas para los comandos `/buy`, `/sell` y `/fill` para ayudar a prevenir operaciones accidentales. Además, el comercio basado en el libro de órdenes se ha mejorado para una ejecución más confiable al operar directamente contra el libro de órdenes.
