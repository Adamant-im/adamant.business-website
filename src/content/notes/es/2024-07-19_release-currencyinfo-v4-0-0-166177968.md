Esta versión introduce un nuevo punto de conexión `GET /status` para monitorear el estado del servicio. Los puntos de conexión `/get` y `/getHistory` ahora incluyen los campos `version` y `last_update` en sus respuestas. Se pueden enviar notificaciones a ADAMANT Messenger y Discord cuando se detectan cambios significativos. Se ha añadido un Dockerfile para simplificar el despliegue.

### Cambios importantes

Se ha implementado un nuevo Sistema de Detección de Cambios Significativos para manejar las diferencias de tasas provenientes de múltiples fuentes. Además, ExchangeRateHost ahora requiere una clave API para funcionar, lo que puede requerir actualizaciones de configuración en despliegues existentes.
