Esta versión añade soporte para Bitcoin dentro de la aplicación. La pantalla de detalles de transacción ahora actualiza dinámicamente el conteo de confirmaciones. Para mejorar la gestión de estados, se ha añadido un estado Registrado para transacciones, lo que permite actualizar los estados de forma más efectiva. Además, esta versión corrige la notificación de estados para transacciones Ethereum y resuelve un problema con el procesamiento de enlaces a chats en msg.adamant.im.
