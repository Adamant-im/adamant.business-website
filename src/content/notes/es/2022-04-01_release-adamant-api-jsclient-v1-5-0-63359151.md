Esta versión del cliente ADAMANT API JS introduce soporte para la blockchain Lisk junto con la funcionalidad existente de ADAMANT. La capa HTTP ha sido migrada a un cliente basado en Axios, mejorando el manejo de solicitudes y la mantenibilidad en todas las redes soportadas.

Un refactorizado más amplio acompaña a estos cambios, limpiando la estructura interna y reduciendo duplicaciones. El proyecto también recibe actualizaciones en sus pautas de estilo y documentación de contribución para ayudar a nuevos colaboradores a comenzar más fácilmente.
