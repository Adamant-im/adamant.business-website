Esta versión del cliente ADAMANT API JS introduce soporte para la generación de claves BTC. La actualización amplía las capacidades criptográficas del cliente para incluir la creación de pares de claves de Bitcoin junto con la funcionalidad de claves ADAMANT existente.

No se introducen cambios que rompan compatibilidad en esta versión.
