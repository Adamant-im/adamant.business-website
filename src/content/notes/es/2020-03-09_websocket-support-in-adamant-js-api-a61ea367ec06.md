ADAMANT es un proyecto de código abierto con una API de blockchain pública, una herramienta de consola y un cliente API de JavaScript. El cliente API de JavaScript se ha actualizado a la versión 0.5.0, siendo la característica más destacada el soporte para WebSocket.

Con conexiones WebSocket, las aplicaciones reciben notificaciones instantáneas sobre nuevas transacciones y mensajes. Esto reduce la latencia en comparación con el uso de llamadas repetidas a la API REST, permitiendo que los bots y servicios respondan a las solicitudes de los usuarios más rápidamente.

La versión está disponible en GitHub, y la documentación de la API de JavaScript se mantiene en la wiki del proyecto.
