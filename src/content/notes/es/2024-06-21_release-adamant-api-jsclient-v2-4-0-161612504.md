Esta versión corrige la validación de claves públicas y nombres de delegados para el método `api.voteForDelegate()`. También añade una comprobación para verificar si una dirección es realmente un delegado al llamar a `api.voteForDelegate()`, junto con mensajes de error de validación más detallados para el mismo método.
