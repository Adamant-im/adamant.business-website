Esta versión de ADAMANT iOS introduce varias mejoras en autenticación de usuario e interfaz. Los usuarios ahora pueden mantener la sesión iniciada entre sesiones, con autorización por código PIN que proporciona una capa adicional de seguridad. La autenticación biométrica es compatible mediante Touch ID y Face ID.

Las barras de navegación se han actualizado para usar títulos grandes en el estilo de iOS 11, ofreciendo una interfaz más moderna y legible. También se realizaron limpiezas generales del código para mejorar la mantenibilidad.

### Cambios importantes

No se introdujeron cambios importantes en esta versión.
