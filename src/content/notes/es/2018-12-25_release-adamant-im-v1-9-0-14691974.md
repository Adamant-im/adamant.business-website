Esta versión reescribe y actualiza cómo se almacenan los datos en el dispositivo del usuario. Los usuarios que prefieran no cerrar sesión automáticamente deben establecer ahora una contraseña adicional, que se utiliza para cifrar los datos locales. Todos los datos almacenados en el dispositivo se cifran mediante cifrado local, y se utiliza Argon2 para el hash de contraseñas. Como resultado, almacenar datos en dispositivos personales para un acceso más rápido es ahora seguro.

Los datos protegidos también se han movido de LocalStorage a IndexedDb, lo que proporciona más espacio para mensajes.
