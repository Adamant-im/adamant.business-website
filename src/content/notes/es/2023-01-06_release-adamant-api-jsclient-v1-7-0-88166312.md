Esta versión introduce el método `api.setStartupCallback()`, que permite a los desarrolladores registrar una devolución de llamada que se ejecuta durante el proceso de inicio del cliente. Además, el constructor de la API ahora acepta una devolución de llamada como tercer parámetro, proporcionando una forma conveniente de suministrar lógica de inicio directamente en el momento de la instanciación.
