Esta versión presenta una refactorización completa del ADAMANT Console, ahora basado en adamant-api. La consola obtiene la capacidad de descifrar mensajes y proporciona ayuda integrada para los comandos disponibles. Se ha introducido un modo interactivo para facilitar su uso.

El paquete ahora puede incluirse como una biblioteca de JavaScript, ampliando las opciones de integración más allá de la línea de comandos. Se han actualizado las dependencias y se incluyen varias correcciones de errores.

### Cambios importantes

La refactorización completa implica que la estructura interna y las API han cambiado significativamente. Los scripts o integraciones existentes que dependan de la arquitectura anterior deberán actualizarse para funcionar con la nueva base basada en adamant-api.
