ADAMANT Messenger v4.5.1 introduce persistencia de mensajes borrador, localización en chino simplificado, soporte para el token STORJ en billeteras y chats, y varias actualizaciones de dependencias y correcciones de errores.

### Mensajes borrador

Los usuarios ahora pueden redactar un mensaje y guardarlo como borrador para enviarlo más tarde. Esto evita la pérdida de texto parcialmente escrito cuando un mensaje no puede completarse en una sola sesión, mejorando el flujo de mensajería para un uso intermitente.

![Conozca la actualización de ADAMANT Messenger v4.5.1](/images/engineering-notes/medium/29ce6e2d6a60/003-0-k0dduujxur7pnr9.webp)

### Soporte para el idioma chino simplificado

La aplicación ahora admite chino simplificado, ampliando la cobertura de localización para usuarios de habla china. Esta actualización también incluye correcciones de errores de pluralización en el idioma ruso para cadenas relacionadas con transacciones.

![Conozca la actualización de ADAMANT Messenger v4.5.1](/images/engineering-notes/medium/29ce6e2d6a60/002-0-l2314o6c9trohor6.webp)

### Integración del token STORJ

STORJ ya está disponible como activo compatible en la billetera, permitiendo a los usuarios transferir STORJ directamente dentro de los chats junto con las criptomonedas previamente soportadas.

![Conozca la actualización de ADAMANT Messenger v4.5.1](/images/engineering-notes/medium/29ce6e2d6a60/004-0-n9t5myjiuilzj6-d.webp)

### Correcciones de errores y actualizaciones técnicas

Se ha resuelto el problema de navegación con la flecha izquierda en la billetera. Se actualizaron bibliotecas subyacentes como Vue, Vuetify y Electron, y se mejoraron el archivo README y las configuraciones de internacionalización.

La versión está disponible en GitHub, donde se puede compilar la aplicación desde el código fuente. ADAMANT Messenger es una Progressive Web Application que soporta Web, Tor, Android, Windows, Linux y macOS.
