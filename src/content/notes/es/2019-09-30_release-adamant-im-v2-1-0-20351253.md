Esta versión de ADAMANT Messenger introduce soporte para AIP-8, AIP-9 y AIP-15. Exchangebot y Betbot oficiales se han agregado a la libreta de direcciones para facilitar el acceso.

Las direcciones cripto ahora incluyen un botón Compartir que permite a los usuarios copiar rápidamente una dirección o generar un código QR. Todos los campos de entrada de direcciones ahora pueden completarse usando un lector de códigos QR, agilizando el proceso de transferencia. Los campos de entrada en dispositivos de escritorio muestran una sugerencia sobre el uso de emojis. Además, los estados de transferencia se vuelven a verificar al iniciar sesión mediante LoginByPassword.
