Esta versión del ADAMANT Tradebot introduce soporte para StakeCube, ampliando el rango de exchanges compatibles. La dependencia de MongoDB ha sido actualizada a una versión más reciente, y se han aplicado varias correcciones de errores para mejorar la estabilidad general. También se incluyen actualizaciones adicionales de dependencias para mantener el proyecto actualizado con las bibliotecas superiores.
