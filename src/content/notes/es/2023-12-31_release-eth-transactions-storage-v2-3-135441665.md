Esta versión actualiza los nombres de las funciones en todo el código base de ETH Transactions Storage para alinearse con los cambios introducidos en la última versión del paquete web3.py. La biblioteca web3.py cambia periódicamente los nombres de funciones y métodos como parte de su propia evolución, y esta actualización garantiza la compatibilidad con la interfaz de API actual.

No se introducen nuevas funciones en esta versión. Los cambios se limitan a ajustes en los nombres de las funciones, lo que permite que el proyecto siga funcionando correctamente con instalaciones recientes de web3.py.

### Cambios importantes

Los nombres de las funciones se han actualizado para coincidir con las convenciones del nuevo paquete web3.py. Cualquier código que llame directamente o haga referencia a los nombres anteriores de las funciones deberá actualizarse en consecuencia.
