Esta es la primera versión estable del cliente JavaScript de ADAMANT API. La biblioteca ofrece acceso altamente confiable a la blockchain ADAMANT, incluyendo solicitudes GET para leer datos en cadena y métodos para enviar tokens y mensajes.

El cliente gestiona operaciones criptográficas como el almacenamiento en caché de claves públicas, el cifrado y descifrado de mensajes, la creación y firma de transacciones, y el trabajo con pares de claves ADM. También admite la generación de una dirección criptográfica de Ethereum y claves vinculadas a una cuenta ADM, así como el manejo del tiempo de época de ADAMANT.

Entre sus capacidades adicionales se incluye el soporte para conexiones WebSocket y un sistema integrado de registro de eventos para advertencias, errores y mensajes informativos.
