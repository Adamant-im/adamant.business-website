ADAMANT Messenger para iOS, versión 2.0.0, ahora permite almacenar y transferir Binance Coin (BNB), Bit-Z token (BZ), Stably USD (USDS), Resfinex token (RES) y KuCoin Shares (KCS) directamente en las conversaciones. Esta actualización alinea las funcionalidades de la aplicación iOS con los clientes existentes para Android, Web y Tor.

La billetera ADAMANT es completamente de código abierto, lo que permite a los usuarios compilar la aplicación desde el código fuente y verificar su seguridad, a diferencia de muchas alternativas de código cerrado. Las claves privadas se almacenan localmente y nunca salen del dispositivo del usuario. Una única frase de acceso proporciona acceso a todas las billeteras cripto integradas, y los usuarios pueden exportar sus claves privadas para realizar copias de seguridad.

La transferencia de criptomonedas directamente dentro de las conversaciones sigue siendo una característica central de la plataforma ADAMANT, ofreciendo un método simplificado para enviar fondos entre usuarios.

Además del nuevo soporte para tokens, esta versión incluye una lista de nodos actualizada y correcciones generales de errores.
