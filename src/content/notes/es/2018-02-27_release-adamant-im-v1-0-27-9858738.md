Esta versión incluye una implementación inicial de escaneo de códigos QR, que aún podría contener errores, y agrega una opción para enviar mensajes al presionar Enter.

Se incluyen varias correcciones. Se ha corregido el envío de mensajes con Cmd+Enter. Anteriormente, el monto máximo disponible para transferencia podía ser incorrecto; este problema ya está resuelto. También se ha corregido un error de multiplicación con punto flotante.

Nuevas traducciones disponibles en japonés, ucraniano e italiano.
