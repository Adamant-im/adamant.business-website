El equipo de ADAMANT realizó un encuentro presencial cerca de Vladimir a finales de agosto para discutir el futuro de la plataforma de mensajería y priorizar las próximas tareas del roadmap en varias áreas de ingeniería.

En el ámbito de la blockchain, las prioridades incluyen ampliar la cobertura de pruebas para los nodos de ADAMANT y desarrollar una API refinada para chats, pares, bootstrap, libreta de direcciones y otros registros KVS, acompañada de propuestas AIP para su estandarización. El equipo también planea investigar chats grupales, chats cerrables, almacenamiento de archivos en nodos y evaluar qué actualizaciones de Lisk deben retrotraerse.

El trabajo de infraestructura se centra en crear un servicio de grupo de forjado de ADM, implementar monitoreo y notificaciones de fallos, realizar pruebas de seguridad, revisar el código en busca de vulnerabilidades y lanzar el primer concurso de seguridad.

Para la aplicación web progresiva, el equipo está migrando a Vue 3.0 con mejoras en la velocidad de la experiencia de usuario. Se planean integraciones de billetera para Binance Coin y tokens ERC20, estos últimos respaldados por un repositorio dedicado en GitHub para listas de tokens. Otras mejoras en la PWA incluyen listas de transacciones para Ethereum y Binance Coin, un nuevo diseño para la lista de tokens, inicio de sesión con contraseña opcional con almacenamiento seguro en el dispositivo y una versión de escritorio Electron dirigida tanto a la Windows Store como a la Mac App Store.

La aplicación para iOS incorporará integración con Ethereum y Lisk para almacenar y transferir esos activos, junto con ajustes de diseño para iPad, soporte para múltiples cuentas, un flujo de bienvenida para nuevos usuarios, migración a la nueva API de bootstrap y KVS, búsqueda dentro de mensajes, soporte para tema oscuro y generación de imágenes de avatar. En Android, se está desarrollando la primera versión con Material Design 2.0, con una versión de prueba pública planeada en Google Play.

Las tareas organizativas incluyen listar ADM en más exchanges, solicitar autorización para procesamiento en fiat, buscar una asociación con una empresa de seguridad informática, identificar conferencias internacionales, contactar a autores de artículos comparativos de mensajería y reclutar representantes regionales. Los esfuerzos de diseño abarcan directrices de diseño, diseño de folleto informativo, mercancía y traducciones actualizadas de idiomas.

Muchas de estas tareas ya se han implementado, mientras que el resto se encuentran en progreso activo.
