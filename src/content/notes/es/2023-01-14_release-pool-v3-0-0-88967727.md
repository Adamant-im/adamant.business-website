Esta versión actualiza las dependencias del proyecto y reescribe el panel usando Svelte. Se rediseñó y optimizó toda la base de código y se corrigieron varios errores.

### Cambios importantes

Se eliminó el soporte para versiones de Node.js anteriores a la 18.12.1, la versión LTS actual. El pool ahora utiliza lowdb como base de datos. Consulte la sección de migración en el archivo README para obtener más detalles.
