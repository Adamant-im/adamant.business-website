Esta versión refactoriza el ApiService para mejorar la estructura interna. Ahora se muestran avatares para contactos de ADAMANT, aunque la red ADAMANT aún no soporta avatares de usuario.

La escena de Chat se ha actualizado con un teclado numérico para el campo de texto, y se ha añadido un botón para copiar al portapapeles las direcciones del socio. La estructura de comisiones también se ha actualizado.

Se han corregido los problemas de mensajes duplicados y desaparecidos mediante una API actualizada.
