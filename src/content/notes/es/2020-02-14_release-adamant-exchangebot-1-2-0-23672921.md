Esta versión introduce soporte para el token RES dentro de ADAMANT Exchangebot. Se ha añadido un nuevo paso de validación en el proceso de intercambio que verifica si hay tasas cripto disponibles para un token determinado antes de continuar. Además, se han aplicado varias correcciones de errores para mejorar la estabilidad general.
