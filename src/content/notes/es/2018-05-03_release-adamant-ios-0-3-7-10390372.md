Esta versión de ADAMANT para iOS resuelve un fallo que ocurría al usar Face ID en el iPhone X. Las nuevas transferencias ahora se marcan automáticamente como no leídas para ayudar a los usuarios a llevar un seguimiento de la actividad reciente. Además, se han corregido la recuperación en segundo plano y las notificaciones push para garantizar una entrega confiable de mensajes. La aplicación ahora también admite notificaciones dentro de la app tanto para mensajes entrantes como para transferencias.
