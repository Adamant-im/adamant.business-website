Esta versión de ADAMANT iOS introduce varias mejoras de seguridad y correcciones de errores. Los usuarios ahora pueden identificar transferencias duplicadas dentro de la interfaz de chat, lo que ayuda a prevenir transacciones repetidas accidentales o fraudulentas. La aplicación también incorpora soporte para el token GT y se conecta a nuevos nodos de Ethereum y Lisk.

Las mejoras adicionales incluyen un formato modificado de la altura del nodo y un requisito de que solo se utilicen nodos que ejecuten una versión superior a la versión mínima compatible. Esta versión corrige problemas relacionados con el teclado y resuelve varios errores de interfaz de usuario. También se han introducido causas adicionales para indicadores de estado inconsistentes con el fin de mejorar la claridad del diagnóstico.
