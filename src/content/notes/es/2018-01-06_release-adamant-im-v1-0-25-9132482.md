Esta versión añade descripciones emergentes a los botones de la interfaz de escritorio. Se incluyen varias correcciones: los títulos de las pestañas ahora se actualizan correctamente, el chat se desplaza hasta el último mensaje según lo esperado (resolviendo el problema #8), el título de la pantalla de inicio de sesión está localizado para ruso y el selector de idioma en la pantalla de Opciones se muestra correctamente. Nuevas traducciones disponibles para turco, filipino, hindi y chino tradicional.
