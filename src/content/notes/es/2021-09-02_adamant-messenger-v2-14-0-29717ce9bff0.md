ADAMANT Messenger versión 2.14.0 incluye varias mejoras de estabilidad y usabilidad para transferencias de criptomonedas, junto con enlaces Tor actualizados en la Aplicación Web Progresiva (PWA). Los detalles de las transacciones ahora permiten copiar directamente el ID de la transacción, así como las direcciones del remitente y del destinatario. Para Dash, se han ajustado los límites máximo y mínimo de envío, y se ha corregido un error que impedía transferir pequeñas cantidades de Lisk. La aplicación también incluye actualizaciones de compatibilidad para el hard fork London de Ethereum. Los mensajes de error que aparecen durante el envío de transacciones ahora tienen un formato más claro, y las notificaciones tipo snackbar se han refinado. El botón de actualización dentro de la vista de detalles de transacción ahora tiene una animación para mejorar la experiencia del usuario. Finalmente, se ha actualizado el enlace del sitio web Tor y se han renovado las dependencias subyacentes.
