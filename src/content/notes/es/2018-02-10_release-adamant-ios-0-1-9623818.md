Este es el lanzamiento inicial de la aplicación ADAMANT para iOS. La aplicación permite enviar y recibir mensajes, ofreciendo la funcionalidad básica de mensajería para la plataforma ADAMANT en dispositivos iOS.

Existe un problema conocido con la pantalla de inicio de sesión, que podría comportarse de manera inconsistente durante este lanzamiento temprano.
