Esta versión actualiza las dependencias del proyecto a Vue3-cli, acompañado de una significativa refactorización en toda la base de código. La aplicación de mensajería ahora soporta tokens ERC20 genéricos y añade funcionalidad de monedero y transferencias de BNB (Binance Coin). Instascan, el escáner QR, ahora se carga de forma diferida, lo que resulta en tiempos de carga inicial más rápidos para los usuarios.

Esta versión incluye varias correcciones. El prefijo "U" ahora se añade correctamente a las direcciones numéricas en el cuadro de diálogo para iniciar un nuevo chat. Se han resuelto errores en la lista de transacciones y los nombres mostrados ya no se sobrescriben al escanear códigos QR. Las comisiones para transferencias no ADM ahora se calculan correctamente, y se ha corregido la inicialización de un nuevo chat desde la lista de transacciones. También se ha actualizado el README.

### Cambios importantes

Las dependencias y comandos CLI ahora se gestionan con yarn en lugar de npm. Los desarrolladores que contribuyan al proyecto deben usar yarn para la gestión de paquetes a partir de ahora, ya que los flujos de trabajo basados en npm podrían no ser compatibles con la nueva cadena de herramientas.
