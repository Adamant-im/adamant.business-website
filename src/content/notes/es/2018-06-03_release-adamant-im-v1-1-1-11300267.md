Esta versión introduce cambios de estilo para hacer que los nuevos campos sean más utilizables en dispositivos móviles. Ahora se admite el inicio de sesión con una frase de acceso almacenada como código QR, gracias a @chovy. El flujo de desarrollo se ha mejorado para que `npm run dev` ahora admita recarga automática.

Se incluyen varias correcciones y mejoras. El almacenamiento de datos en KVS ahora se activa solo cuando hay suficientes tokens ADM disponibles, eliminando ventanas molestas. Se ha corregido un error y se han añadido nuevas constantes de idioma junto con actualizaciones generales de traducciones.

Los idiomas ucraniano, filipino, sueco, hindi, turco, vietnamita, rumano y árabe han sido desactivados temporalmente hasta que alguien actualice los archivos de idioma correspondientes.
