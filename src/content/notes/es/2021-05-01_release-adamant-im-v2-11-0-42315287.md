Esta versión optimiza los tiempos de actualización de transacciones para diferentes monedas y corrige el manejo de UTXO para Doge. Un nuevo módulo txVerify verifica la consistencia de transacciones, y ahora se puede actualizar el detalle de transacciones desde la pantalla de chat y desde la pantalla de detalles de transacción.

Se ha añadido validación de direcciones cripto para cuentas de usuario y socios, y se verifica la consistencia de transacciones en la vista de detalles de transacción. La vista de Nodos ahora muestra la versión del nodo. Se ha corregido un error que provocaba una lista de transacciones vacía, junto con otras correcciones menores.
