Esta versión introduce soporte para Bit-Z Token (BZ), ampliando el rango de activos compatibles. Se han agregado conexiones WebSocket para mejorar el manejo de datos en tiempo real y la comunicación con las API de intercambio. Las dependencias se han actualizado a sus últimas versiones, y se han aplicado varias correcciones para resolver problemas identificados en versiones anteriores.
