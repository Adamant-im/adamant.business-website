### ADAMANT 年度回顾：技术进步与平台发展

又一年的 ADAMANT 开发工作已经结束，整个平台实现了重大的技术进步。ADAMANT TECH LABS 首席执行官 Alexei Lebedev 指出：构建一个前所未有的信使应用需要应对诸多技术挑战，即使是经典信使中常见的功能，也需要深入研究并采用新的方法。所有工作都沿着通向去中心化互联网的清晰路径推进。

**基础设施和 API** 取得了重大进展。区块浏览器（Block Explorer）完成了重新设计，ADAMANT 的核心基础设施也得到了增强。随着节点版本 0.5.1 的发布，Chatrooms API 正式上线，并推出了 Android 演示版本。同时发布了 Node API 规范和 JavaScript API 规范，为开发者提供了在平台上构建应用所需的文档。锻造池（forging pool）软件也进行了更新。

**聊天内加密货币转账** 成为核心功能，现已支持在聊天中直接存储和转账 Lisk、DOGE、Dash、KuCoin Shares 和 BTC。iOS 版本也新增了对 Dash 转账的支持。此外，还引入了汇率服务以完善这些功能。

**安全与隐私** 方面的改进包括推出了 ADAMANT 2FA、iOS 上支持导出私钥，以及支持通过 Tor 访问 ADAMANT。

**Web 应用** 经历了显著演进，完成了全面重新设计，发布了 2.0 版本，随后又推出了 2.3.0 版本。新增了通知功能，引入了暗色主题，并实现了消息搜索功能。用户还可以通过控制台（Console）读取和发送消息。

**桌面和移动端扩展** 带来了适用于 Mac、Windows 和 GNU/Linux 的原生应用。在 iOS 上，推送通知功能得到增强，可包含消息内容。

**机器人和交易所** 成为新的发展方向。ADAMANT 区块链上推出了机器人，包括一个基于加密货币汇率的投注机器人（Bet Bot）。基于 ADAMANT 平台构建了匿名加密货币交易所，用户还可以创建自己的 Exchanger 实例。

**合作与上币** 方面包括与 Stably 和 Stably Dollar 的集成、与 Resfinex 的合作（促成 ADM/USDT 交易对上线），以及在其他代币整合资源平台上的上币。

这些广泛的发展成果体现了向功能更强大的去中心化消息平台稳步迈进的过程。未来一年的计划将聚焦于让 ADAMANT 更快、更便捷、功能更完善。
