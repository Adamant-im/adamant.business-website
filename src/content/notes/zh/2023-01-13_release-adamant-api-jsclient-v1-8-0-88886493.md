此版本的 ADAMANT API JS 客户端引入了一个新的 `checkNodeTimeout` 参数，允许开发者配置检查节点可用性时的超时持续时间。这一新增功能提供了更精细的控制，使客户端能够更好地验证与 ADAMANT 节点的连接，帮助应用程序更优雅地处理响应缓慢或无响应的节点。
