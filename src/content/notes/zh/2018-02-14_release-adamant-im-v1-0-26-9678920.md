## 变更

本次发布将消息发送费用从 0.005 降至 0.001 AMD，降低了网络通信成本。应用程序现在更频繁地检查传入消息，提升了响应速度。新增了一个链接至 ICO 页面的按钮，用户将接收 ICO 消息，取代此前的 Pre-ICO 消息。同时更新了网站图标（favicons），并进行了一些小的内部调整。

## 修复

已解决导致消息消失的问题。针对本地存储（Local Storage）应用了临时修复方案，预计在后续版本中将完全移除。修复了链接渲染问题（#16），该问题此前可能被用于欺骗用户。

## 翻译

新增了阿拉伯语、德语、葡萄牙语（由 @DataDrug 通过 PR #17 贡献）、罗马尼亚语、瑞典语（由 @pdahlberg 通过 PR #29 贡献）和越南语的翻译。
