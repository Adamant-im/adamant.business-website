此版本的 ADAMANT API JS Client 在原有 ADAMANT 功能基础上新增对 Lisk 区块链的支持。HTTP 层已迁移至基于 Axios 的客户端，提升了跨网络请求处理能力和可维护性。

伴随这些变更，项目还进行了更广泛的重构，清理了内部结构并减少了代码重复。项目同时更新了代码风格指南和贡献文档，以帮助新贡献者更轻松地开始参与开发。
