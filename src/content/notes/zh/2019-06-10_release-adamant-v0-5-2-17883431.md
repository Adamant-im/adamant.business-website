此版本为 `/api/transactions` 和 `/api/transactions/get` 端点增加了对 `returnAsset` 参数的支持。提供该参数时，此参数会强制 API 在响应负载中包含交易资产。此项更改为此后版本中整合 API 端点奠定了基础，减少了获取完整交易详情所需的调用次数。
