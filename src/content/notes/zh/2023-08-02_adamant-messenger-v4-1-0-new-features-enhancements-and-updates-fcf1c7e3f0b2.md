ADAMANT Messenger v4.1.0 引入了根据 AIP-16 的引用和回复消息功能、用于清理通知的“标记全部为已读”按钮，以及多行消息输入的十行上限，以保持聊天界面紧凑。这些改进使对话更易于跟进，并减少视觉杂乱。

在钱包方面，来自 adamant-wallets 仓库的生成脚本简化了对新代币的支持。现在通过下拉手势可刷新余额，加载时会显示省略号指示器。支持的币种列表已扩展，新增了 Tether (USDT)、USD Coin (USDC) 和 Dai，同时继续支持 ADM、BTC、ETH、Doge、Dash 和 Lisk。

![ADAMANT Messenger v4.1.0：新功能、改进与更新](/images/engineering-notes/medium/fcf1c7e3f0b2/002-0-rgxdqzzrdjk3m7qj.webp)

界面优化包括生成二维码时的加载动画，以及在 BuyTokensDialog 中新增 StakeCube 交易所。代表投票图标也已重新设计，以提供更清晰的视觉反馈。

![ADAMANT Messenger v4.1.0：新功能、改进与更新](/images/engineering-notes/medium/fcf1c7e3f0b2/003-0-xor1ka7rgr2ppf0e.webp)

本次发布还包含旨在提升稳定性和兼容性的错误修复和库更新。ADAMANT Messenger 是一款渐进式网页应用，支持 Web、Tor、Android、Windows、Linux 和 macOS。完整发布详情请见 GitHub 发布页面：https://github.com/Adamant-im/adamant-im/releases/tag/v4.1.0
