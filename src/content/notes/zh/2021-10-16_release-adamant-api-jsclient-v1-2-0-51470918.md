此版本的 ADAMANT API JS 客户端新增了对 BTC 密钥生成的支持。此次更新扩展了客户端的加密功能，使其在原有 ADAMANT 密钥功能的基础上，增加了比特币密钥对的创建能力。

此版本未引入任何破坏性变更。
