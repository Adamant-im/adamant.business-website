ADAMANT 正在扩大团队，招聘开发者和业务发展经理以推动其去中心化消息生态系统。所有职位均需高度认同隐私价值观并对项目充满热情。

### 业务发展经理
该全职职位专注于通过与大使、合作伙伴和客户合作来扩大 ADM 社区。候选人必须精通英语，并具备在 Medium 和 X/Twitter 上进行内容创作的能力，包括图像和视频编辑。必须具备扎实的区块链技术知识、社区管理经验以及针对区块链产品的营销策略理解。

### iOS 开发者
iOS 开发者将使用 Swift 维护和开发原生 ADAMANT iOS 客户端。技术栈包括 UIKit、SwiftUI、Combine、Swinject、Core Data、GCD、MVVM、VIPER、SnapKit、Git、Socket.io、REST、CocoaPods 和 SPM。该职位为全职或兼职，每周至少需工作 25 小时。

### 前端开发者（Vue/TypeScript）
该职位涉及使用 Vue 开发 ADAMANT 渐进式网页应用（PWA）。候选人需精通 JavaScript、TypeScript、HTML5、CSS3、SCSS 和 Vue3，并具备构建可扩展的 Web 应用、RESTful API 和微服务的经验。需熟悉 Git 以及 Jest 和 Mocha 等测试框架。该职位为全职或兼职，每周至少需工作 25 小时。

### 后端 JavaScript 开发者（Node.js）
后端开发者将负责 ADAMANT 区块链核心节点及其周边服务的开发。候选人必须理解区块链技术、dPoS 共识机制、交易、加密签名和加密算法。需具备扎实的 JavaScript、TypeScript、数据库系统（PostgreSQL、MongoDB、Redis）、Git 和测试框架知识。该职位为全职或兼职，每周至少需工作 25 小时。

### 全栈 JavaScript 开发者（Vue/Node.js）
全栈开发者将参与 ADAMANT 生态系统的多种工具开发，包括 Botfactory、Exchanger、Blockchain Explorer、Forging pool、CoinOptimus 和 TradeBot。候选人需精通 JavaScript、TypeScript、Node.js、Vue3（或 React/AngularJS）、数据库（PostgreSQL、MongoDB、Redis）以及测试框架。该职位为全职或兼职，每周至少需工作 25 小时。

### 申请流程
申请请将简历发送至 msg # adamant.im，或通过 adamant.im/contact 上的联系页面提交。ADAMANT 不接受由 ChatGPT 生成的申请，并将通过面试核实技能。候选人应认同 cryptopunk 和隐私价值观，富有热情并愿意努力工作。薪资可协商，但整体较为适中。
