此版本将项目依赖更新至 Vue3-cli，同时对整个代码库进行了大规模重构。该消息应用现在支持通用 ERC20 代币，并新增了 BNB（Binance Coin）钱包和转账功能。Instascan（二维码扫描器）现在采用懒加载，使用户首次加载速度更快。

此版本包含多项修复。在“新建聊天”弹窗中，数值地址现在会正确添加 "U" 前缀。修复了交易列表的若干 bug，扫描二维码时不再覆盖显示名称。非 ADM 转账的手续费现在可正确计算，且已修复从交易列表初始化新聊天的问题。README 也已同步更新。

### 重大变更

依赖项和 CLI 命令现在使用 yarn 而非 npm 提供。今后参与项目开发的开发者应使用 yarn 进行包管理，因为基于 npm 的工作流可能不再与更新后的工具链兼容。
