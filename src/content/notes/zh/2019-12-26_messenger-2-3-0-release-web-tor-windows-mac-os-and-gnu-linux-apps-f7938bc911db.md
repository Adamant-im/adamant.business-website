ADAMANT Messenger v2.3.0 现已发布，支持 Web、Tor、Windows、macOS 和 GNU/Linux 平台。本次更新引入了对比特币的支持，并对交易状态处理进行了多项改进，同时修复了若干漏洞。

多条区块链的交易处理已得到优化。交易详情页面中的确认次数现在可动态更新，新增的交易 Registered 状态使得状态更新能够更有效地处理。此前存在错误的以太坊交易状态问题也已修复。此外，对指向 msg.adamant.im 聊天页面的链接处理也已修正。

当前代码库位于 [ADAMANT Messenger GitHub 仓库](https://github.com/Adamant-im/adamant-im) 的 master 分支。有意参与开源 ADAMANT 项目开发的开发者可在此获取最新源码。
