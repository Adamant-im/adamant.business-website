本次发布对 ADAMANT Console 进行了全面重构，现基于 adamant-api 构建。控制台新增了消息解密功能，并为可用命令提供了内置帮助。引入了交互模式以简化使用流程。

该包现在可作为 JavaScript 库引入，扩展了除命令行之外的集成选项。同时更新了依赖项，并包含了多项错误修复。

### Breaking changes

由于完全重构，内部结构和 API 发生了重大变化。依赖旧版架构的现有脚本或集成需进行更新，方可与基于 adamant-api 的新架构兼容。
