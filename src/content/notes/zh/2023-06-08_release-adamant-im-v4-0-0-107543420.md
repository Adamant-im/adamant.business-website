本次发布重点是对 ADAMANT Messenger 代码库进行重大现代化升级。应用已升级至 Vue 3 和 Vuetify 3，样式系统从 Stylus 迁移至 SASS 并完成重写。构建系统已从 Webpack 迁移到 Vite，测试运行器从 Jest 切换为 Vitest。新增了 TypeScript 支持，并配置了 Prettier 和 EditorConfig。包管理器从 Yarn 切换为 NPM。axios、socket.io-client、@mdi/font 等依赖项已升级至最新主版本，所有依赖包均已更新至最新的次版本和补丁版本。已修复失败的测试用例和过时的快照，Browserslist 和 ESLint 配置已从 package.json 移至独立文件，多个组件已重写以使用组合式 API（Composition API）。

在功能方面，聊天界面中已移除 Bounty Wallet 机器人（#482），但此前与该机器人有过对话的用户仍可访问其聊天历史记录。WebAppManifest 中已添加更多信息以支持 PWA 安装（#481）。在 BuyTokensDialog 中，Coindeal 已被 Azbit 替代（#476）。已新增用于 Electron 和 PWA 构建的 GitHub Actions 工作流（#458, #461）。

本次发布修复了多个问题。修复了新账户在 DelegatesTable 中加载代表时出现的错误（#475）。修复了部分设备在 Transactions 视图中未触发滚动事件的问题（#474）。现在在返回导航时聊天界面可保持滚动位置（#470），当收到新消息时聊天列表会自动滚动到底部（#455）。此外还应用了大量细微的样式修复（#454, #465, #466, #467, #468, #469, #471, #473, #479, #480）。
