此版本为 ADAMANT 节点软件引入了多项 API 改进和修复。`/api/delegates/get` 端点现在除了之前支持的 `username` 和 `publicKey` 参数外，还接受 `address` 参数。使用方式非常简单：

```bash
/api/delegates/get?address=U1234567890
```

WebSocket 支持已增强，允许订阅多种交易类型，而不仅限于一种。对于消息交易，新增的 `assetChatTypes` 事件允许客户端订阅特定的消息类型。

`/api/node/status` 端点现在通过合并 `/api/loader/status` 和 `/api/loader/status/sync` 的数据返回更多属性，减少了获取完整区块链状态信息所需的请求数量：

```json5
{
  // ...
  "loader": {
    "loaded": true,
    "now": 45345521,
    "syncing": false,
    "consensus": 100,
    "blocks": 0,
    "blocksCount": 0
  }
}
```

已修复多个漏洞。现已解决版本名称中包含预发布或构建元数据（如 `+alpha` 或 `-dev.0`）的节点在连接时出现的问题，完全符合 SemVer 2.0 规范。修复了一次性连接错误后错误移除节点的问题，并修正了配置文件验证功能。

`/delegates/get` 端点已针对单个交易搜索进行了优化。依赖项已更新，日志功能有小幅改进，并新增了有关测试的贡献者指南。
