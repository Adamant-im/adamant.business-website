ADAMANT Node v0.6.5 现已发布。本次更新聚焦代码质量和工具改进，而非协议变更，因此更新为可选，且不影响网络共识。

默认广播设置已更改，安装用 bash 脚本也已更新。开发依赖已刷新，代码库已完成 lint 检查，测试套件已修复。

两个值得注意的新增功能是 `-genesis` 命令行参数，以及测试网专用的 `portWS` 36665。这些改进使得运行创世节点和测试网配置更加便捷，无需手动覆盖配置。

Ubuntu、macOS 和 Windows 的安装说明详见 [ADAMANT Node v0.6.5 发行说明](https://github.com/Adamant-im/adamant/releases/tag/v0.6.5)。
