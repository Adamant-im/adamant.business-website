ADAMANT InfoServices 已更新至 v3.5.0，提供统一的法币和加密货币汇率 API。该服务聚合了多个上游数据源，包括 MOEX、Currency-Api、ExchangeRates.host、CoinMarketCap、CryptoCompare 和 CoinGecko，并对结果进行交叉验证，以标记异常并提高可靠性。

由于服务运行在您自己的服务器上，历史汇率数据由您完全掌控，且可免费用于任何用途。该项目为开源项目，托管在 GitHub 的 ADAMANT 组织下。

API 文档请参见 [InfoServices API wiki](https://github.com/Adamant-im/adamant-currencyinfo-services/wiki/InfoServices-API-documentation)。
