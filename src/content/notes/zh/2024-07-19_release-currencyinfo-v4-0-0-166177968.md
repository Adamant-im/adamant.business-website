此版本引入了一个新的 `GET /status` 接口，用于监控服务健康状况。`/get` 和 `/getHistory` 接口的响应中现在包含 `version` 和 `last_update` 字段。当检测到重大变化时，可向 ADAMANT Messenger 和 Discord 发送通知。新增 Dockerfile 以简化部署。

### 重大变更

已实现新的重大变化检测系统，用于处理来自多个数据源的汇率差异。此外，ExchangeRateHost 现在需要 API 密钥才能运行，现有部署可能需要更新配置。
