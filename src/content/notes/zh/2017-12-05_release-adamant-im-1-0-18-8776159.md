此版本的 ADAMANT Messenger 引入了对消息中单个换行符的支持，使其行为与 GitHub 风格 Markdown 保持一致。此前，单个换行符不会被渲染为实际的换行；此项更改使消息格式符合常见的 Markdown 规范。

已修复未读消息计数器的问题，解决了计数可能显示错误的情况。此外，已应用临时修复以解决 iOS 11 上的兼容性问题，提升了该平台用户的稳定性。同时还对状态存储逻辑进行了小幅调整，以进一步提高可靠性。

### 重大变更

此版本中没有重大变更。
