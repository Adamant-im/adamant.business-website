ADAMANT Messenger 版本 4.0.0 引入了全面的技术栈更新，旨在提升整体性能并降低内存消耗。此次发布主要作为基础性升级，以促进未来功能的开发。

由于性能优化和多项错误修复提升了稳定性，用户将体验到更快速、更流畅的消息服务。ADAMANT Messenger 继续以渐进式网页应用（PWA）形式运行，保持在 Web、Tor、Android、Windows、Linux 和 macOS 环境中的兼容性。

更多技术细节请参阅 ADAMANT GitHub 仓库中的官方发布说明。
