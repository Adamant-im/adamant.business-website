此版本新增对 Bit-Z Token (BZ) 的支持，扩展了支持的资产范围。已添加 WebSocket 连接，以改善实时数据处理及与交易所 API 的通信。依赖项已更新至最新版本，并应用了多项修复，以解决先前版本中发现的问题。
