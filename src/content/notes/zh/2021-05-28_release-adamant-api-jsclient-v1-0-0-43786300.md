这是 ADAMANT API JavaScript 客户端的首个稳定版本。该库提供对 ADAMANT 区块链的高可靠性访问，包括用于读取链上数据的 GET 请求，以及发送代币和消息的方法。

客户端处理加密操作，例如缓存公钥、加密和解密消息、构建和签名交易，以及管理 ADM 密钥对。它还支持为 ADM 账户生成绑定的以太坊加密地址和密钥，并支持使用 ADAMANT 纪元时间（epoch time）。

其他功能包括支持 WebSocket 连接，以及针对警告、错误和信息性消息的内置日志记录。
