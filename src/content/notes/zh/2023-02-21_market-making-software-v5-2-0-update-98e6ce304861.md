ADAMANT 交易与做市机器人是一款免费的开源软件，可用于在加密交易所执行交易、生成交易量、维持价差和流动性、设置价格范围以及构建动态订单簿。v5.2.0 版本是一次以稳定性和连接器可靠性为重点的维护性发布。

本次发布最显著的变化是对 P2PB2B API 连接器的完全重写，这将改善与交易所的通信，并减少与连接器相关的错误。此次更新还包括全代码库的通用重构、依赖项升级以及错误修复。

源代码可在 [GitHub](https://github.com/Adamant-im/adamant-tradebot) 上获取。
