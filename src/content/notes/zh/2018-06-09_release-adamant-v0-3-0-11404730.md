此版本根据 [AIP 5](https://aips.adamant.im/AIPS/aip-5) 增加了对富内容消息（Rich Content Messages）的支持，使 ADAMANT 消息平台内的消息格式更加丰富。同时，根据 [AIP 6](https://aips.adamant.im/AIPS/aip-6) 引入了对信号消息（Signal Messages）的支持。此外，此版本还修复了状态(KVS)交易 API 中的过滤器问题。
