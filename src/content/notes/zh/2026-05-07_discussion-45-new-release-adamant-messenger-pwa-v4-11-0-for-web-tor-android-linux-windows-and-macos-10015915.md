ADAMANT 已发布其 Messenger PWA 的 v4.11.0 版本，现已适用于 Web、Tor、Android、Linux、Windows 和 macOS 平台。本次发布主要聚焦于可靠性、完善性和成熟度的改进，而非引入重大新功能。

最新的构建版本在 GitHub 上标记为 v4.11.1，表明在初始版本 4.11.0 发布后进行了快速的后续补丁更新。运行 PWA 的用户应升级至最新版本标签，以获取发布后的修复内容。

ADAMANT Messenger 是一款开源的、基于区块链的 messaging 应用程序。PWA（渐进式网页应用）版本允许用户在浏览器中运行该 messenger，或将其作为类原生应用安装在支持的桌面和移动平台上，包括为需要洋葱路由访问的用户提供的 Tor 支持。
