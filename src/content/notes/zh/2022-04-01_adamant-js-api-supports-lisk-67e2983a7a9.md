ADAMANT JavaScript API 库现支持 Lisk 加密密钥，同时进行了全面的代码库重构。v1.5.0 版本将之前的 HTTP 客户端替换为 Axios，改善了整个库的请求处理和可维护性。贡献指南和代码风格也已更新，以简化协作。

完整的 API 文档可在 [ADAMANT API wiki](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification) 上查阅，版本详情请见 [v1.5.0 发布页面](https://github.com/Adamant-im/adamant-api-jsclient/releases/tag/v1.5.0)。
