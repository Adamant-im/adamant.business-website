### 最新 Android 版本中的 API 速度更快

继此前宣布对 ADAMANT 节点的 API 速度进行优化后，Android 开发团队发布了一项更新，其中包含新的聊天 API。

新旧 API 的对比显示，聊天操作和登录流程的完成速度均显著快于以往。未来还计划进行更多优化。

最新 APK 可从 [Android 的 GitHub 发布页面](https://github.com/Adamant-im/adamant-android/releases/) 获取。现有用户应确保已运行最新版本；如果应用未自动更新，请卸载旧版本并手动安装新的 APK。

反馈和错误报告可通过 GitHub 仓库提交。团队也正在准备 Google Play 的发布。
