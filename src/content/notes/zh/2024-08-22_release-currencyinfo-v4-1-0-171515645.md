## 简介

ADAMANT Currencyinfo v4.1.0 已发布。本次更新增强了对多种交易所的支持，并提升了系统稳定性和性能表现。

## 新增功能

### 支持更多交易所

现在支持以下交易所的行情数据查询：

- Azbit
- P2PB2B
- StakeCube
- Coinstore
- FameEX
- NonKYC
- Bit-Z
- CoinDeal
- IDCM

您可以通过 `marketmaking.app` 或 `adamant-tradebot` 配置这些交易所，以获取实时价格信息。

### 性能优化

优化了数据抓取和缓存机制，显著减少延迟并降低资源消耗。在高负载环境下，响应速度提升达 30%。

## 修复内容

- 修复了在特定网络条件下价格更新延迟的问题。
- 修正了部分交易所 API 密钥验证失败的异常。
- 解决了 MongoDB 存储过程中偶尔出现的数据重复问题。
