此版本更新了项目依赖并使用 Svelte 重写了仪表板。整个代码库已重新设计和优化，并修复了多个漏洞。

### 重大变更

已停止支持低于 18.12.1 版本的 Node.js，即当前的 LTS 版本。矿池现在使用 lowdb 作为其数据库。详情请参考 README 文件中的迁移部分。
