此版本为 ETH、BTC、DASH、DOGE 和 LSK 节点引入了健康检查功能，同时用 TypeScript 重写了节点 API 客户端。登录页面已添加指向节点页面的链接，节点复选框的状态现在会持久化存储在 LocalStorage 中。

包含多项聊天功能改进：滚动到底部按钮新增了消息计数器，返回按钮下方现在显示未读消息数量，内联和多行 Markdown 代码块可正确格式化显示。过长的聊天名称现以省略号显示。发送消息前会检查 ADM 余额，并在清空输入框前完成检查，重复转账的不一致交易状态也已正确显示。

在技术债务方面，已升级 `bitcoinjs-lib`，使用 `Psbt` 替代 `TransactionBuilder`，签名机制更改为需先获取未花费输出（unspents）。`getAccount()` 函数现使用 `ecpair` 和 `tiny-secp256k1` 生成比特币地址，并新增 `vite-plugin-wasm` 和 `vite-plugin-top-level-await` 作为依赖项。`web-eth` 库已从 v1 升级至 v4，移除了 `abi-decoder` 依赖，改用自定义实现，将 `INCREASE_FEE_MULTIPLIER` 从 2 修改为 1.5，并新增了测试。所有依赖项的次要版本和补丁版本均已更新。来自 `adamant-wallets` 的钱包更新在配置中添加了 Tor 节点，引入了 `minNodeVersion`，并重构了 `serviceNodes`。
