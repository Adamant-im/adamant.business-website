此版本新增支持多种 ERC20 代币，包括 Binance Coin (BNB)、Binance USD (BUSD)、Dai (DAI)、Ethereum Name Service (ENS)、Holo (HOT)、Injective (INJ)、Chainlink (LINK)、Decentraland (MANA)、Polygon (MATIC)、PAX Gold (PAXG)、Quant (QNT)、Ren (REN)、SKALE (SKL)、Status (SNT)、Synthetix Network (SNX)、TrueUSD (TUSD)、Uniswap (UNI)、USD Coin (USDC)、PAX Dollar (USDP)、Stably USD (USDS)、Tether (USDT)、WOO Network (WOO) 以及 Onyxcoin (XCN)。用户现在可以管理钱包中显示的代币，且 ETH 和 ERC20 转账已优化，手续费更低，操作更流畅。

此次更新还带来了改进的聊天内交易处理、更快的交易处理速度，以及更简洁快速的消息界面。ADM 地址现在以 Markdown 超链接形式呈现。macOS 支持也得到增强，包括推送通知和 CPU 使用率优化。此外，应用现具备更稳定的连接和健康检查机制，从后台返回时聊天更新更顺畅，并实现了整体性能提升及多项小错误修复。

### 破坏性变更

该应用现要求 iOS 15 或更高版本才能运行。
