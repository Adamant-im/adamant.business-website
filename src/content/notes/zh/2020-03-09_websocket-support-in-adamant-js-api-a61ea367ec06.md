ADAMANT 是一个开源项目，拥有公共区块链 API、控制台工具和 JavaScript API 客户端。JavaScript API 客户端已更新至 0.5.0 版本，其中最重要的新增功能是 WebSocket 支持。

通过 WebSocket 连接，应用程序可以即时收到关于新交易和消息的通知。相比轮询 REST API，这显著降低了延迟，使机器人和服务能够更快地响应用户请求。

该版本发布在 GitHub 上，JavaScript API 文档由项目 Wiki 维护。
