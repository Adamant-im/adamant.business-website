ADAMANT JavaScript API 库 v1.4.0 引入了两个新的代表节点管理方法：`delegateNew()` 用于注册新代表节点，`voteForDelegate()` 用于对现有代表节点投票。此版本还包括常规的依赖项更新。有关使用详情，请参阅 [API 规范](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification)。
