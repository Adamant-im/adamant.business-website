此版本引入了 `api.setStartupCallback()` 方法，允许开发者注册在客户端启动过程中执行的回调函数。此外，API 构造函数现在将回调函数作为其第三个参数，提供了一种在实例化时直接注入启动逻辑的便捷方式。
