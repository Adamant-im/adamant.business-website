此版本的 ADAMANT Messenger 扩展了可连接节点的列表，并引入了节点协议检查功能。已修正 HTTP 主机上的节点状态和套接字连接问题，并更新了底层依赖。

交易所集成已进行调整：新增了 CoinDeal 的链接，移除了 IDCM 的链接，并更新了 Resfinex 和 Bit-Z 的链接。多个与聊天相关的问题已修复，包括聊天中内联 `code` 的样式、静态聊天名称、发起新聊天时粘贴地址的验证、"无公钥"和"无哈希"消息的本地化，以及在 Windows 和其他应用中为 ADM 地址生成二维码的功能。
