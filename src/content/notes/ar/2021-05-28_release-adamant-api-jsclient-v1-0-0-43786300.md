هذا هو الإصدار المستقر الأول من عميل ADAMANT API بلغة JavaScript. توفر المكتبة وصولاً عالي الموثوقية إلى بلوكشين ADAMANT، بما في ذلك طلبات GET لقراءة البيانات الموجودة على البلوكشين، وأساليب لإرسال الرموز والرسائل.

يتعامل العميل مع العمليات التشفيرية مثل تخزين المفاتيح العامة مؤقتًا، وتشفير وفك تشفير الرسائل، وتكوين وتوقيع المعاملات، والعمل مع أزواج المفاتيح ADM. كما يدعم توليد عنوان عملات رقمية على شبكة إيثريوم ومفاتيح مرتبطة بحساب ADM، بالإضافة إلى العمل مع وقت العصر (epoch time) في ADAMANT.

تشمل القدرات الإضافية دعم اتصالات WebSocket، وتسجيل الأحداث الداخلي للتحذيرات والأخطاء والرسائل المعلوماتية.
