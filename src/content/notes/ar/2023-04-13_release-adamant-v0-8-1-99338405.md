تضيف هذه الإصدارة تسجيل السجلات (logging) لنص هجرة قاعدة البيانات. سيقوم عقدة ADAMANT الآن بتسجيل رسائل قبل وبعد تنفيذ نص الهجرة:

```
[inf] 2023-12-22 04:00:25 | Found 5 pending migrations. Start executing, this may take a while…
[inf] 2023-12-22 04:17:13 | Migrations have been successfully completed.
```

تم تغيير القيمة الافتراضية لخاصية `cors.origin` من عكس مصدر الطلب (`true`) إلى `*`. راجع الوثائق لمعرفة المزيد حول تهيئة CORS لعقدة ADAMANT.

```json5
{
  "cors": {
    "origin": "*" // default
  }
}
```

تمت إضافة ملف `config.json` إلى ملف `.gitignore`. يجب على المستخدمين الآن إنشاء ملف `config.json` بأنفسهم أو نسخ `config.default.json` لبدء تشغيل عقدة ADAMANT.

```
cp config.default.json config.json
```

### التغييرات المُدخلة

تم تغيير القيمة الافتراضية لرأس CORS `Access-Control-Allow-Origin` من عكس مصدر الطلب إلى `*`. يجب على المستخدمين الذين يعتمدون على سلوك العكس السابق تحديث ملف `config.json` الخاص بهم وفقًا لذلك. بالإضافة إلى ذلك، أصبح ملف `config.json` متجاهَلًا الآن بواسطة git، وبالتالي يجب على المستخدمين إنشاء هذا الملف أو نسخه يدويًا قبل بدء تشغيل العقدة.
