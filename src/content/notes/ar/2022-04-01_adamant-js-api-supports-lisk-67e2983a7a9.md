يدعم مكتبة ADAMANT JavaScript API الآن مفاتيح Lisk التشفيرية إلى جانب إعادة هيكلة شاملة للشفرة المصدرية. إصدار v1.5.0 يستبدل عميل HTTP السابق بـ Axios، مما يحسن معالجة الطلبات وسهولة الصيانة عبر المكتبة. كما تم تحديث إرشادات المساهمة وأسلوب الشفرة لتسهيل التعاون.

تتوفر وثائق واجهة برمجة التطبيقات الكاملة على [Wiki ADAMANT API](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification)، وتفاصيل الإصدار متوفرة على [صفحة إصدار v1.5.0](https://github.com/Adamant-im/adamant-api-jsclient/releases/tag/v1.5.0).
