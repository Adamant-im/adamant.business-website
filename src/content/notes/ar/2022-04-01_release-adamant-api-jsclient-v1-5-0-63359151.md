يُقدِّم هذا الإصدار من عميل ADAMANT API JS دعمًا لسلسلة Lisk إلى جانب الوظائف الحالية لـ ADAMANT. تم نقل طبقة HTTP إلى عميل يعتمد على Axios، مما يحسّن التعامل مع الطلبات وسهولة الصيانة عبر الشبكات المدعومة.

ترافق هذه التغييرات عملية إعادة هيكلة أوسع، تشمل تنظيف البنية الداخلية وتقليل التكرار. كما تلقى المشروع تحديثات على إرشادات الأسلوب ووثائق المساهمة لمساعدة المساهمين الجدد على البدء بسهولة أكبر.
