تُقدِّم هذه الإصدارة طريقة `api.setStartupCallback()`، التي تتيح للمطورين تسجيل دالة تُنفَّذ أثناء عملية بدء تشغيل العميل. بالإضافة إلى ذلك، أصبح بإمكان بُنية API الآن قبول دالة ردّ كمعامل ثالث، مما يوفّر وسيلة مريحة لتوفير منطق بدء التشغيل مباشرةً عند الإنشاء.
