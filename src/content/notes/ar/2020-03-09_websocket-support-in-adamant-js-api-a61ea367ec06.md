ADAMANT هو مشروع مفتوح المصدر يحتوي على واجهة برمجة تطبيقات كتلة عامة، وأداة وحدة تحكم، وعميل واجهة برمجة تطبيقات JavaScript. تم تحديث عميل واجهة برمجة تطبيقات JavaScript إلى الإصدار 0.5.0، مع أهم إضافة وهي دعم WebSocket.

مع اتصالات WebSocket، تتلقى التطبيقات إشعارات فورية حول المعاملات والرسائل الجديدة. وهذا يقلل من زمن الوصول مقارنةً بالاستعلام الدوري لواجهة REST API، مما يسمح للروبوتات والخدمات بالاستجابة لطلبات المستخدمين بشكل أسرع.

يتوفر هذا الإصدار على GitHub، ويتم الحفاظ على وثائق واجهة برمجة تطبيقات JavaScript في ويكي المشروع.
