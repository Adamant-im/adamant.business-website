بعد الإدراج في بورصة مركزية (CEX)، غالبًا ما تواجه شركات إصدار الرموز سجل أوامر ضعيفًا، وفروقات سعر واسعة، ورسمًا بيانيًا يُعاقب الصفقات الصغيرة. الحل التقليدي هو إقراض الرموز ومشاركة مفاتيح API مع جهة خارجية تقوم بعملية صنع السوق على بنية تحتية غير شفافة. توفر برمجية ADAMANT لصنع السوق الإصدار v9.0.0 بديلاً: مجموعة برمجية للتحكم الذاتي في صنع السوق تقوم بتثبيتها كأي برنامج عادي — بدون الحاجة إلى `git clone`، ولا تسليم للوصاية على الأصول.

### النموذج: تجارة ذاتية، تحكم ذاتي

غالبًا ما يتضمن صنع السوق التقليدي إرسال الرموز إلى طرف ثالث، ومشاركة مفاتيح API مع نظام مغلق، وآملًا أن يبدو سجل الأوامر صحيًا — وأن تتمكن من استرداد أصولك. ADAMANT تقلب هذا النموذج رأسًا على عقب، بحيث تقوم أنت بتشغيل برمجية صنع السوق على خادمك الخاص، باستخدام حسابك الخاص في البورصة ومفاتيحك الخاصة.

مقارنة بين نموذج الوصاية التقليدي وبرمجية ADAMANT لصنع السوق

### ما الذي يفعله الإصدار v9 لرسمك البياني

النسخة المجانية والمفتوحة المصدر تركز على المشكلات التي تظهر مباشرة بعد الإدراج. تقوم ببناء سجل الأوامر عن طريق ملء الفجوات لتجنب مظهر السوق المهجور، وتحافظ على فروقات سعر أضيق لتقليل الانطباع السلبي الأولي، وتوفر عمقًا بحيث لا تؤثر الصفقات الصغيرة بشكل كبير على السعر. تراقب النطاقات السعرية التي تحددها وتطبق سياسات الحجم عبر أوضاع الانتشار، وسجل الأوامر، والعمق، والوضع الأمثل. المراقبة شفافة — الأرصدة، الأوامر، والإحصائيات متاحة عبر أوامر تحكم من خلال ADAMANT Messenger، دون وجود لوحة تحكم عامة معرّضة افتراضيًا.

موصلات البورصات المركزية المدعومة في النسخة المفتوحة المصدر تشمل Azbit، P2PB2B، StakeCube، Coinstore، FameEX، وNonKYC. تتوفر خيارات اتصال متميزة ومخصصة لمزيد من البورصات.

### البدء (الطريقة عبر npm)

تحتاج إلى خادم Linux أو جهاز Mac (أو أي جهاز يدعم npm)، وNode.js 22+، وMongoDB، ومفتاح API من بورصة مركزية لحسابك الخاص.

![Your Token Deserves a Real Market — Not a Loan to a Black-Box Market Maker](/images/engineering-notes/medium/42fcfb71beb3/003-1-duiackkmcteg8he9ccmqdq-png.webp)

مستودع npm الرسمي

قم بتثبيت الحزمة بشكل عام وإنشاء دليل عمل:

```bash
npm install -g adamant-tradebot
mkdir my-mm && cd my-mm
```

أمر واجهة سطر الأوامر (CLI) هو `mm`. قم بتهيئة البوت باستخدام معالج تفاعلي، ثم قم بفحص الصحة:

```bash
mm init    # interactive wizard - exchange, pair, API keys
mm doctor  # checks config, MongoDB, exchange API
```

لا تغادر الرموز حسابك في البورصة لتمويل صانع السوق. أنت فقط تربط بيانات اعتماد API الخاصة بك بالبوت على جهازك الخاص. ابدأ وتحقق من الحالة:

```bash
mm on
mm status
```

أرسل الأمر `/balances` إلى بوتك من حساب ADM الإداري الخاص بك، وتصبح جاهزًا للعمل. أوقف في أي وقت باستخدام `mm off`، واطلع على السجلات باستخدام `mm logs`.

### بديل باستخدام Docker

يمكنك أيضًا استيراد الصورة المنشورة من GitHub Container Registry:

```bash
docker pull ghcr.io/adamant-im/adamant-tradebot:9.0.0
```

يتم تشغيل MongoDB ضمن Compose جنبًا إلى جنب مع التطبيق، مع الاحتفاظ بالتكوين والسجلات في مجلدات محلية تتحكم فيها.

### لماذا الإصدار v9 معلم بارز

قبل v9، كان البدء يعني استنساخ مستودع وربط التبعيات يدويًا — وهو أمر مناسب للمطورين، لكنه يشكل عائقًا أمام المؤسسين الذين يريدون فقط سوقًا أكثر صحة. الإصدار v9.0.0 يقدم توزيعًا احترافيًا عبر npm وGHCR، وأداة `mm` CLI مع أوامر `init`، `on`، `off`، `doctor`، `status`، `logs`، و`config`، ونشر تلقائي عبر GitHub Actions لكل إصدار على npm وDocker، ومحركًا مُعاد هيكلته يغطي المتداول، وبناء سجل الأوامر، ومزود السيولة، وراصد السعر، بالإضافة إلى مجموعات اختبار Jest والتوثيق.

ADAMANT هي مشروع تشفير مفتوح المصدر مع عشر سنوات من التطوير العلني.

### الفئة المستهدفة

تم تصميم هذا البرنامج لشركات إصدار الرموز بعد الإدراج في بورصة مركزية مع سجل أوامر ضعيف أو فارغ، والفرق التي لا تستطيع تحمل رسوم باهظة بالإضافة إلى قروض رمزية، والمؤسسين الذين لا يثقون في تسليم مفاتيحهم لأطراف ثالثة غير شفافة، والمشاريع التي تطلب الشفافية — القدرة على قراءة الكود، مراقبة السجلات، والاحتفاظ بزر الإيقاف. توجد وحدات متميزة لاستراتيجيات متقدمة، وواجهة ويب، وبورصات إضافية، أو مساعدة في الإعداد. النسخة المجانية المفتوحة المصدر مفيدة بشكل مقصود بذاتها.

### ملاحظة حول المسؤولية

يجب أن يتم صنع السوق وفقًا لقواعد البورصة والقوانين المعمول بها. توفر ADAMANT برمجيات، وليس وصاية أو تنفيذ نيابة عنك. أنت من يُهيئها، ويُشغلها، ويبقى مسؤولًا عن استخدامها.
