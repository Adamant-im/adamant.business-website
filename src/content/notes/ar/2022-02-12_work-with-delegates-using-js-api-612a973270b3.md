تقدم مكتبة ADAMANT JavaScript API الإصدار v1.4.0 طريقتين جديدتين لإدارة المندوبين: `delegateNew()` لتسجيل مندوب جديد و `voteForDelegate()` للتصويت للمندوبين الحاليين. يتضمن هذا الإصدار أيضًا تحديثات روتينية للاعتمادات. راجع [مواصفات واجهة برمجة التطبيقات](https://github.com/Adamant-im/adamant-api-jsclient/wiki/API-Specification) للحصول على تفاصيل الاستخدام.
