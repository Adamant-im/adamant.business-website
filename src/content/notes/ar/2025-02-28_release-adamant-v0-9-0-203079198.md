تُقدِّم هذه الإصدارة عدة تحسينات وتصحيحات لواجهة برمجة تطبيق عقدة ADAMANT. أصبحت نقطة النهاية `/api/delegates/get` تقبل الآن معلمة `address` بالإضافة إلى المعلمات المدعومة سابقًا `username` و`publicKey`. استخدامها بسيط:

```bash
/api/delegates/get?address=U1234567890
```

تم تحسين دعم WebSocket للسماح بالاشتراك في أنواع متعددة من المعاملات بدلاً من نوع واحد فقط. بالنسبة لمعاملات الرسائل، يتيح حدث جديد يُسمى `assetChatTypes` للعملاء الاشتراك في أنواع رسائل محددة.

الآن تُرجع نقطة النهاية `/api/node/status` خصائص إضافية من خلال دمج البيانات من `/api/loader/status` و`/api/loader/status/sync`، مما يقلل من عدد الطلبات الضرورية للحصول على معلومات الحالة الكاملة لسلسلة الكتل:

```json5
{
  // ...
  "loader": {
    "loaded": true,
    "now": 45345521,
    "syncing": false,
    "consensus": 100,
    "blocks": 0,
    "blocksCount": 0
  }
}
```

تم إصلاح العديد من الأخطاء. تم حل مشكلة الاتصال مع الأقران الذين تتضمن أسماء إصداراتهم بيانات إصدار تجريبي أو بيانات بناء (مثل `+alpha` أو `-dev.0`)، ما يضمن التوافق الكامل مع مواصفات SemVer 2.0. تم تصحيح إزالة الأقران بعد أخطاء اتصال لمرة واحدة، وتم إصلاح التحقق من صحة ملف الإعدادات.

تم تحسين أداء نقطة النهاية `/delegates/get` للبحث في معاملة واحدة. تم تحديث التبعيات، وتم إجراء تحسينات طفيفة في تسجيل السجلات، بالإضافة إلى إضافة دليل جديد للمشاركين حول الاختبارات.
